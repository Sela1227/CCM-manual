# A3. 醫師團隊名冊

**彰濱秀傳癌症中心｜個管師指導書系列**

> 📌 本檔是各 MDT 團隊**醫師成員聯絡名冊**，供個管師日常工作查詢使用——MDT 會議聯絡、轉診、會診、個案討論等。
>
> 📊 **資料來源**：MDT 會議管理系統匯出主檔（2026/04/30）。共 38 位醫師、14 個科別、8 個 MDT 例會。**分機資訊待 Sela 與主管確認後填入**。

---

## 使用說明

**本檔架構**：依 [A1 第一部分「多專科團隊列表」](A_work-guide.md#多專科團隊列表) 的 MDT 團隊順序，每團隊一段。

**對應癌別檔**（C 系列）：每團隊段落會註明對應的 C 系列檔連結，方便個管師同時查臨床照護與聯絡資訊。

**欄位說明**：

| 欄位 | 內容 |
|------|------|
| 姓名 | 醫師姓名 |
| 科別 | 該醫師的科別 |
| 角色 | 召集人 / 固定成員 |
| 分機 | 院內聯絡分機（**不寫手機**——資安考量；緊急聯絡請洽主管或秘書）|

> 💡 **更新原則**：本檔由（待填：負責人/秘書）依 MDT 系統主檔變動同步更新。人事異動時請通知更新。
>
> 📌 **保密提醒**：本檔僅供院內個管師日常工作參考，**請勿外流**。醫師個人聯絡資訊屬個資範疇。
