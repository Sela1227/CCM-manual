## 功能狀態評估

### ECOG Performance Status（每個病人都要評）

Eastern Cooperative Oncology Group 體能狀態量表，是最常用的功能評估工具。

| 分數 | 定義 | 白話說明 |
|------|------|---------|
| 0 | 完全正常活動，無任何限制 | 跟生病前一樣 |
| 1 | 輕度症狀，可自由行動，可從事輕度工作 | 有點不舒服但可以正常生活 |
| 2 | 可自理生活，但無法工作；白天臥床時間 < 50% | 可以照顧自己，但沒辦法工作 |
| 3 | 僅能部分自理；白天臥床時間 > 50% | 需要別人幫忙照顧 |
| 4 | 完全無法自理，完全臥床 | 完全躺在床上 |

**使用時機：**
- 收案時（必評）
- 每次追蹤
- 治療前評估
- 決定是否適合積極治療

> 💡 ECOG ≥ 3 通常不適合接受積極化療，需要跟醫師討論治療目標。

---

### Karnofsky Performance Status (KPS)

比 ECOG 更細緻的功能評估，有時病歷或文獻會用到。

| KPS | 定義 | 對應 ECOG |
|-----|------|----------|
| 100 | 正常，無症狀 | 0 |
| 90 | 能正常活動，有輕微症狀 | 0 |
| 80 | 勉強可正常活動，有些症狀 | 1 |
| 70 | 可自理，但無法正常工作 | 1 |
| 60 | 偶需協助，但大部分可自理 | 2 |
| 50 | 需要較多協助和醫療照護 | 2 |
| 40 | 失能，需特殊照護 | 3 |
| 30 | 嚴重失能，需住院 | 3 |
| 20 | 病重，需積極支持治療 | 4 |
| 10 | 瀕死 | 4 |

**ECOG 與 KPS 快速對照：**

| ECOG | KPS |
|------|-----|
| 0 | 90-100 |
| 1 | 70-80 |
| 2 | 50-60 |
| 3 | 30-40 |
| 4 | 10-20 |

## 衰弱與共病評估

### Clinical Frailty Scale (CFS) 臨床衰弱量表

評估老年病人的整體衰弱程度，有助於治療決策。

| 分數 | 狀態 | 說明 |
|------|------|------|
| 1 | 非常健康 | 規律運動，同年齡中最健康的一群 |
| 2 | 健康 | 沒有明顯疾病，但不如第 1 級活躍 |
| 3 | 維持健康 | 有慢性病但控制良好，不影響日常活動 |
| 4 | 輕度衰弱 | 日常活動需要協助（如購物、理財） |
| 5 | 中度衰弱 | 需要協助處理日常生活（如準備餐食） |
| 6 | 中重度衰弱 | 所有戶外活動和家務都需要協助 |
| 7 | 嚴重衰弱 | 完全依賴他人照顧，但生命徵象穩定 |
| 8 | 非常嚴重衰弱 | 完全依賴，接近生命末期 |
| 9 | 末期 | 預期存活 < 6 個月 |

**使用時機：**
- 老年癌症病人（≥ 65 歲）
- 評估是否適合積極治療
- CFS ≥ 5 需謹慎考慮治療強度

### Charlson Comorbidity Index (CCI) 共病指數

評估病人的共病負擔，可預測死亡風險。

| 分數 | 疾病 |
|------|------|
| **1 分** | 心肌梗塞、鬱血性心衰竭、周邊血管疾病、腦血管疾病、失智症、慢性肺病、結締組織疾病、消化性潰瘍、輕度肝病、糖尿病（無併發症） |
| **2 分** | 偏癱、中重度腎病、糖尿病（有併發症）、任何腫瘤、白血病、淋巴瘤 |
| **3 分** | 中重度肝病 |
| **6 分** | 轉移性實體腫瘤、AIDS |

**年齡校正：** 每增加 10 歲加 1 分（從 50 歲開始）

| CCI 分數 | 10 年存活率 |
|---------|------------|
| 0 | 93% |
| 1-2 | 90% |
| 3-4 | 77% |
| ≥ 5 | 53% |

## 肝功能評估

### Child-Pugh Score 肝功能分級

評估肝硬化病人的肝功能，**肝癌、肝膽胰癌病人必評**。

| 項目 | 1 分 | 2 分 | 3 分 |
|------|------|------|------|
| **腦病變** | 無 | 輕度 (Grade 1-2) | 中重度 (Grade 3-4) |
| **腹水** | 無 | 輕度（可控制） | 中重度（難控制） |
| **Bilirubin (mg/dL)** | < 2 | 2-3 | > 3 |
| **Albumin (g/dL)** | > 3.5 | 2.8-3.5 | < 2.8 |
| **PT 延長 (秒)** | < 4 | 4-6 | > 6 |
| 或 **INR** | < 1.7 | 1.7-2.3 | > 2.3 |

**分級與預後：**

| 分級 | 分數 | 1 年存活率 | 2 年存活率 | 說明 |
|------|------|-----------|-----------|------|
| **A** | 5-6 | 100% | 85% | 肝功能良好，可接受手術 |
| **B** | 7-9 | 80% | 60% | 肝功能中等，治療需謹慎 |
| **C** | 10-15 | 45% | 35% | 肝功能差，預後不佳 |

> 💡 Child-Pugh C 的肝癌病人通常不適合積極抗癌治療，以支持性照護為主。

---

### ALBI Grade 白蛋白-膽紅素分級

較新的肝功能評估工具，只需要 Albumin 和 Bilirubin 兩個數值。

**計算公式：**
```
ALBI = (log10 Bilirubin × 0.66) + (Albumin × -0.085)

Bilirubin 單位：μmol/L
Albumin 單位：g/L
```

| Grade | ALBI 分數 | 說明 |
|-------|----------|------|
| 1 | ≤ -2.60 | 肝功能良好 |
| 2 | -2.60 ~ -1.39 | 肝功能中等 |
| 3 | > -1.39 | 肝功能差 |

> 💡 實際使用時可用線上計算機，輸入數值即可得到 Grade。

## 腎功能與化療劑量

### Cockcroft-Gault 公式（CCr 計算）

計算肌酸酐清除率，評估腎功能，決定化療劑量是否需要調整。

**公式：**
```
CCr (mL/min) = [(140 - 年齡) × 體重(kg)] / [72 × Cr(mg/dL)]

女性需再 × 0.85
```

**簡易對照：**

| CCr (mL/min) | 腎功能 | 化療劑量調整 |
|--------------|--------|-------------|
| ≥ 60 | 正常或輕度下降 | 通常不需調整 |
| 30-59 | 中度下降 | 部分藥物需減量 |
| 15-29 | 重度下降 | 多數藥物需減量或避免 |
| < 15 | 腎衰竭 | 需謹慎評估 |

> 💡 Cisplatin 通常需要 CCr ≥ 60 才能使用標準劑量。

---

### Calvert 公式（Carboplatin AUC）

計算 Carboplatin 的劑量，依據目標 AUC 和腎功能。

**公式：**
```
Carboplatin 劑量 (mg) = 目標 AUC × (GFR + 25)
```

**常見 AUC 目標：**

| 適應症 | AUC |
|--------|-----|
| 與紫杉醇併用 | 5-6 |
| 單獨使用 | 5-7 |
| 週療 | 2 |

**例如：** 目標 AUC 5，GFR 60
- 劑量 = 5 × (60 + 25) = 425 mg

> 💡 AUC 是「曲線下面積」，代表藥物暴露量。由藥師或醫師計算，個管師了解概念即可。

---

### BSA 體表面積計算

許多化療藥物劑量是以 mg/m² 計算。

**Mosteller 公式：**
```
BSA (m²) = √[(身高(cm) × 體重(kg)) / 3600]
```

**快速查表：**

| 身高/體重 | 50 kg | 60 kg | 70 kg | 80 kg |
|----------|-------|-------|-------|-------|
| 150 cm | 1.44 | 1.58 | 1.71 | 1.83 |
| 160 cm | 1.50 | 1.64 | 1.77 | 1.89 |
| 170 cm | 1.56 | 1.70 | 1.83 | 1.96 |
| 180 cm | 1.61 | 1.76 | 1.89 | 2.02 |

## 營養評估

### PG-SGA（Patient-Generated Subjective Global Assessment）

癌症病人專用的營養評估工具。

**病人自填部分：**

**A. 體重變化（過去 1 個月）**

| 狀況 | 分數 |
|------|------|
| 無變化 | 0 |
| 減少但不確定多少 | 1 |
| 減少 1-2 kg | 1 |
| 減少 2-5 kg | 2 |
| 減少 5-10 kg | 3 |
| 減少 > 10 kg | 4 |

**B. 進食變化**

| 狀況 | 分數 |
|------|------|
| 無變化 | 0 |
| 比平常少 | 1 |
| 只能吃流質 | 2 |
| 只能吃少量流質 | 3 |
| 幾乎無法進食 | 4 |

**C. 症狀（過去 2 週，每個症狀 1 分）**

噁心、嘔吐、便秘、腹瀉、口腔潰瘍、口乾、食物味道改變、聞到味道不舒服、吞嚥困難、容易飽、疼痛、疲倦、其他

**D. 活動功能**

| 狀況 | 分數 |
|------|------|
| 正常 | 0 |
| 比平常差但可下床 | 1 |
| 大部分時間躺或坐 | 2 |
| 臥床 | 3 |

**總分判讀：**

| 總分 | 營養狀況 | 處理 |
|------|---------|------|
| 0-1 | 良好 | 持續監測 |
| 2-3 | 輕度風險 | 營養衛教 |
| 4-8 | 中度風險 | **轉介營養師** |
| ≥ 9 | 重度風險 | **緊急轉介營養師** |

---

### BMI 與體重變化

**BMI 計算：**
```
BMI = 體重(kg) / 身高(m)²
```

| BMI | 判讀 |
|-----|------|
| < 18.5 | 過輕 |
| 18.5-24 | 正常 |
| 24-27 | 過重 |
| ≥ 27 | 肥胖 |

**體重變化警訊：**

| 時間 | 需注意的體重下降 |
|------|-----------------|
| 1 週 | > 2% |
| 1 個月 | > 5% |
| 3 個月 | > 7.5% |
| 6 個月 | > 10% |

## 疼痛評估

### NRS 數字疼痛量表

請病人用 0-10 分表示疼痛程度。

```
0 ─────────────────────────────────────── 10
無痛                                    最痛
```

| 分數 | 程度 | 處理原則 |
|------|------|---------|
| 0 | 無痛 | - |
| 1-3 | 輕度 | 第一階止痛藥（如 Acetaminophen、NSAIDs） |
| 4-6 | 中度 | 弱鴉片類（如 Tramadol）± 第一階 |
| 7-10 | 重度 | 強鴉片類（如 Morphine）± 第一階 |

**疼痛評估要問的 PQRST：**

| 項目 | 問法 |
|------|------|
| **P**rovoke 誘發/緩解 | 什麼情況會更痛/比較不痛？ |
| **Q**uality 性質 | 什麼樣的痛？（刺痛、鈍痛、燒灼感...） |
| **R**egion 部位 | 哪裡痛？有沒有延伸到其他地方？ |
| **S**everity 強度 | 0-10 分幾分？ |
| **T**ime 時間 | 持續多久？什麼時候最痛？ |

**還要問：**
- 有沒有影響睡眠、活動、心情？
- 目前有沒有在吃止痛藥？效果如何？

## 心理困擾評估

### Distress Thermometer (DT) 心理困擾溫度計

| 分數 | 程度 | 處理 |
|------|------|------|
| 0-3 | 輕度 | 一般支持 |
| **4-6** | **中度** | **加強關懷，考慮轉介** |
| **7-10** | **重度** | **優先轉介心理師/精神科** |

**困擾來源清單：**

| 類別 | 項目 |
|------|------|
| 實際問題 | 照顧小孩、照顧家人、經濟、保險、工作/學校、交通、住處 |
| 情緒問題 | 憂鬱、恐懼、緊張、悲傷、擔心、失去興趣、憤怒 |
| 身體問題 | 疼痛、噁心、疲倦、睡眠、食慾、便秘/腹瀉、外觀改變、呼吸困難 |
| 靈性問題 | 宗教信仰、生命意義 |

> 💡 DT ≥ 4 分要考慮轉介，但也要看困擾來源，選擇適當的轉介對象（心理師、社工師、宗教師等）。

### PHQ-9 憂鬱量表

過去 2 週，以下情況出現的頻率：
- 0 = 完全沒有
- 1 = 幾天
- 2 = 超過一半的天數
- 3 = 幾乎每天

| 項目 | 評分 |
|------|------|
| 1. 做事提不起勁或沒興趣 | 0-3 |
| 2. 感到沮喪、憂鬱或絕望 | 0-3 |
| 3. 睡眠困難或睡太多 | 0-3 |
| 4. 感覺疲倦或沒有精力 | 0-3 |
| 5. 食慾不好或吃太多 | 0-3 |
| 6. 覺得自己不好、是失敗者 | 0-3 |
| 7. 無法專注 | 0-3 |
| 8. 動作或說話變慢/坐立不安 | 0-3 |
| 9. 想到死亡或傷害自己 | 0-3 |

**總分判讀：**

| 分數 | 程度 | 建議 |
|------|------|------|
| 0-4 | 極輕微 | 持續監測 |
| 5-9 | 輕度 | 支持性會談 |
| 10-14 | 中度 | 考慮轉介 |
| 15-19 | 中重度 | **轉介精神科/心理師** |
| 20-27 | 重度 | **緊急轉介** |

> ⚠️ **第 9 題有任何分數都要特別注意**，評估自殺風險。

## 副作用分級 CTCAE

### 血液學毒性

**嗜中性球低下（Neutropenia）**

| Grade | ANC 數值 | 說明 |
|-------|---------|------|
| 1 | < LLN - 1,500/μL | |
| 2 | < 1,500 - 1,000/μL | |
| 3 | < 1,000 - 500/μL | 嚴重，感染風險高 |
| 4 | < 500/μL | 非常嚴重 |

> ⚠️ **Febrile Neutropenia**：ANC < 500（或 < 1000 且預期下降）+ 體溫 ≥ 38.3°C（或 ≥ 38°C 持續 1 小時），是**腫瘤急症**！

| Grade | Hb 數值 |
|-------|---------|
| 1 | < LLN - 10 g/dL |
| 2 | < 10 - 8 g/dL |
| 3 | < 8 g/dL；需輸血 |
| 4 | 危及生命 |

**血小板低下（Thrombocytopenia）**

| Grade | Platelet 數值 |
|-------|--------------|
| 1 | < LLN - 75,000/μL |
| 2 | < 75,000 - 50,000/μL |
| 3 | < 50,000 - 25,000/μL |
| 4 | < 25,000/μL |

## 安寧預後評估

### PPI（Palliative Prognostic Index）緩和預後指數

評估末期病人的預後，幫助決定照護目標。

| 項目 | 選項 | 分數 |
|------|------|------|
| **PPS** | 10-20 | 4 |

## 腫瘤治療反應評估

### RECIST 1.1 簡介

Response Evaluation Criteria in Solid Tumors（實體腫瘤療效評估標準）

| 反應 | 縮寫 | 定義 |
|------|------|------|
| 完全緩解 | CR | 所有目標病灶消失 |
| 部分緩解 | PR | 目標病灶縮小 ≥ 30% |
| 疾病穩定 | SD | 縮小 < 30% 且增大 < 20% |
| 疾病惡化 | PD | 增大 ≥ 20% 或出現新病灶 |

**常見說法：**
- 「影像穩定」= SD
- 「影像有進步」= PR 或 CR
- 「影像有惡化」= PD

## D3. 電話追蹤腳本

> 腳本是參考框架，不是逐字稿。依病人狀況靈活調整，重點是讓病人感覺被關心、有問題知道找誰。

---

### 初次收案電話

**時機：** 收案後 3 天內，確認已建立關係

```
「○○先生/小姐您好，我是彰濱秀傳癌症中心的個案管理師○○○，
  您的主治醫師○○醫師請我來關心您。
  請問您現在方便說話嗎？」

（確認可以說話後）

「我的工作是在您接受治療的過程中協助您——
  如果有任何關於檢查、治療或是需要哪些資源的問題，都可以聯絡我。
  
  我的直線電話是 ○○○-○○○○○，您方便的話記錄一下。
  
  想請問您，現在有沒有最困擾您的問題或擔心？」

（傾聽，依回答給予回應）

「我們會定期聯絡您。如果您有任何問題，不要客氣，直接打給我。
  祝您一切順利，再見！」
```

**記得問：**
- [ ] 確認聯絡電話是否正確
- [ ] 有無緊急聯絡人
- [ ] 目前最擔心的事（作為後續追蹤重點）

---

### 定期追蹤電話（治療穩定中）

**時機：** 每月或每兩週依計畫追蹤

```
「○○先生/小姐您好，我是○○○個管師，
  打電話來關心一下您最近的狀況，現在方便說話嗎？」

（確認可以說話）

「最近身體怎麼樣？」

── 依回答深入詢問：
  「有沒有哪裡不舒服？（疼痛、噁心、疲倦...）」
  「吃得好嗎？體重有沒有變化？」
  「睡得好嗎？心情怎麼樣？」
  「藥有按時吃嗎？有沒有什麼副作用？」

「下次回診是○月○日，○○醫師門診，您記得嗎？」

「如果有任何問題，隨時打給我，我的電話是○○○-○○○○○。
  保重，再見！」
```

### 術後追蹤電話（手術後 1–4 週）

**時機：** 手術後第 3–5 天、第 2 週、第 4 週各一次

```
「○○先生/小姐您好，我是○○○個管師，
  您上週剛做了手術，特別打電話來關心。現在方便說話嗎？」

【必問項目】

傷口：
  「傷口這幾天怎麼樣？有沒有紅腫、流膿或特別多的分泌物？」

疼痛：
  「疼痛程度 0 到 10 分怎麼樣？止痛藥夠用嗎？」

飲食：
  「可以吃東西了嗎？吃了什麼？體重有沒有很快下降？」

活動：
  「可以下床走動嗎？有沒有憋氣走路或腿部腫脹？」（警覺肺栓塞）

病理報告：
  「有沒有安排回診和醫師討論報告的時間？」（術後約 2 週回診）

【警示 → 立即回診】
  • 傷口紅腫熱痛加劇或流膿
  • 發燒 > 38°C
  • 突發胸痛或呼吸困難
  • 嚴重腹脹、無法排氣排便（腸阻塞）
```

---

### 化療中追蹤電話

**時機：** 每個 Cycle 治療後第 3–5 天（副作用高峰期）、治療後第 7–14 天（血球低點）

```
「○○先生/小姐您好，我是○○○個管師。
  您上週做了化療，特別打電話問一下您最近的狀況。現在方便嗎？」

【Cycle 後 3–5 天（副作用峰值）】

噁心嘔吐：
  「噁心嘔吐怎麼樣？有沒有吃止吐藥？一天吐幾次？吃得下東西嗎？」
  → 一天吐 3 次以上或完全無法進食超過 24 小時 → 請回診

口腔：
  「口腔裡面有沒有破？吃東西會痛嗎？」

腹瀉：
  「有沒有拉肚子？一天幾次？」
  → 一天超過 4–6 次 → 請回診

疲倦：
  「體力怎麼樣？有沒有很嚴重的疲倦影響日常活動？」

【治療後第 7–14 天（血球低點）】

發燒：
  「這幾天有沒有量體溫？有沒有發燒超過 38 度？」
  → 發燒 > 38°C → 立即就醫（不能等到明天）

感染警示：
  「有沒有喉嚨痛、傷口不舒服、或身上有什麼感染的跡象？」

出血：
  「有沒有不明原因的瘀青擴散、刷牙出血很多、或身上哪裡出血？」

!!! warning "發燒提醒"
    「特別提醒您，化療期間如果發燒超過 38 度，不管是幾點，
    都要打急診電話或直接去急診，不能等到隔天門診，
    這可能是感染，需要馬上處理。」
```

---

### 放療中追蹤電話

**時機：** 每週一次（整個放療療程）；放療結束後 2 週再追蹤一次

```
「○○先生/小姐您好，我是○○○個管師，
  您這週有去做放療，特別打電話關心一下。現在方便嗎？」

【依照射部位調整問題】

頭頸部放療：
  「口腔裡面有沒有破？吃東西會不會痛或吞嚥困難？」
  「皮膚有沒有紅腫、脫皮？有沒有照著皮膚護理的方法？」
  「有沒有口乾的問題？喝水量夠嗎？」
  → 吞嚥困難導致無法進食 → 安排營養師評估

胸部放療：
  「吞嚥有沒有不舒服？（食道炎）」
  「有沒有新出現的咳嗽或喘氣？」（放射性肺炎警示）
  「皮膚有沒有紅腫或脫皮？」

腹部/骨盆放療：
  「有沒有腹瀉？一天幾次？」
  「有沒有頻尿、解尿不適？」
  「有沒有噁心？」

全部都要問：
  「這幾天疲倦程度怎麼樣？影響到日常活動嗎？」

【放療完成後的追蹤】
  「放療結束後副作用可能還會持續 2–4 週，
  不要擔心，這是正常的，不代表病情惡化。
  我們下個月會安排追蹤影像，我會跟您聯繫預約時間。」
```

### 口服藥物追蹤電話（標靶 / 荷爾蒙 / 口服化療）

**時機：** 開始服藥後第 2 週、第 1 個月，之後每 1–2 個月

```
「○○先生/小姐您好，我是○○○個管師，
  您最近開始吃新的藥，打電話確認一下服藥狀況，現在方便嗎？」

服藥遵從性：
  「藥有沒有每天按時吃？有沒有漏吃過？」
  「請問您都是什麼時間吃藥？飯前還是飯後？」（確認服藥方式正確）

依藥物種類問副作用：

──標靶藥物（EGFR、ALK 等）：
  「皮膚有沒有出現紅疹或像青春痘的東西？」
  「有沒有拉肚子？一天幾次？」
  「有沒有吃東西不舒服、噁心或口腔破？」

──荷爾蒙治療（AI / Tamoxifen）：
  「有沒有關節痠痛？會影響走路嗎？」
  「有沒有熱潮紅的問題？」
  「體重有沒有明顯變化？」

──口服化療（Capecitabine / Ufur）：
  「手腳有沒有脫皮、紅腫或疼痛？（手足症候群）」
  「有沒有腹瀉？一天幾次？」

──免疫治療（PD-1/PD-L1 抑制劑）：
  「有沒有新出現的皮疹？」
  「有沒有新出現的喘氣或咳嗽？」（肺炎警示）
  「有沒有腹瀉？」
  「有沒有很嚴重的疲倦或心悸？」

「如果有任何副作用讓您擔心，或覺得影響到生活，
  一定要告訴我，我們可以跟醫師討論是否要調整藥物。」
```

# D3. 電話追蹤腳本

> 電話追蹤是個管師最重要的工作之一。以下依**情境分類**，每個情境都有建議開場、必問項目和收尾。
> 開始前先查看病歷，了解病人目前的治療狀況，再打電話。

---

## 通用開場與結尾

**開場（任何情境通用）**
```
「○○先生/小姐您好，我是彰濱秀傳癌症中心的個管師○○○，
想關心一下您最近的狀況，請問現在方便說話嗎？」
```

若不方便：
```
「沒關係，請問什麼時候比較方便？我到時候再打給您。」
→ 記錄預計回電時間
```

**通用結尾**
```
「謝謝您今天的時間。如果有任何不舒服或問題，
隨時可以打電話給我，我的分機是○○○（或手機 ○○○）。
祝您順利，我們○月○日門診見。」
```

---

## 腳本一：手術後追蹤（出院後 1 週內）

**目的**：確認傷口恢復、飲食活動、是否出現異常。

**必問項目：**

| 項目 | 建議問法 |
|------|---------|
| **傷口狀況** | 「傷口有沒有紅腫、流膿、或滲液比較多？」 |
| **疼痛程度** | 「傷口和身體有多痛？0-10 分幾分？止痛藥有在吃嗎？有沒有效？」 |
| **進食狀況** | 「吃得下東西嗎？有沒有噁心或嘔吐？」 |
| **活動能力** | 「可以下床走動嗎？不舒服嗎？」 |
| **排便/排尿** | 「有沒有正常排便排尿？」（腹部手術格外重要）|
| **回診確認** | 「下次門診是○月○日，記得嗎？傷口拆線安排好了嗎？」 |

**緊急警示（立刻叫病人就醫）：**
- 傷口紅腫化膿、發燒 > 38°C
- 嚴重腹痛或腹脹
- 完全無法進食超過 24 小時
- 排尿困難或完全排不出尿

---

## 腳本二：化療療程中追蹤（每個 Cycle 後第 7–10 天）

**目的**：確認骨髓抑制最低點期間的安全、副作用管理。

**必問項目：**

| 項目 | 建議問法 |
|------|---------|
| **🚨 發燒（最重要）** | 「這幾天有沒有發燒？有量體溫嗎？幾度？」 |
| **噁心嘔吐** | 「化療後噁心嘔吐有幾天？最嚴重時吃得下東西嗎？止吐藥有吃嗎？」 |
| **疲倦程度** | 「體力怎麼樣？可以做一般家事嗎？」 |
| **手腳感覺** | 「手腳有沒有麻麻的、刺刺的感覺？（Oxaliplatin/Paclitaxel 等）」 |
| **口腔** | 「嘴巴裡有沒有潰瘍或破皮、痛到吃不下東西？」 |
| **腹瀉** | 「有沒有腹瀉？一天幾次？（Irinotecan/Capecitabine 等）」 |
| **體重** | 「體重有沒有下降？最近幾天吃得怎麼樣？」 |
| **下次化療** | 「下次化療是○月○日，驗血和看診都安排了嗎？」 |

**緊急警示：**
- 🚨 **發燒 > 38°C** → 立刻就醫，不能等到明天（感染性嗜中性球低下）
- 嚴重嘔吐無法進食超過 24 小時
- 嚴重腹瀉（一天 ≥ 6 次）
- 呼吸困難或胸痛

---

## 腳本三：放療療程中追蹤（每週一次）

**目的**：監測放療相關副作用、確認療程如期進行。

**先確認照射部位再問對應問題。**

**通用問題：**

| 項目 | 建議問法 |
|------|---------|
| **放療如期進行** | 「這週有沒有按時去放療？有沒有中斷？」 |
| **疲倦** | 「體力怎麼樣？疲倦有沒有影響日常生活？」 |
| **皮膚** | 「照射的地方皮膚有沒有紅腫、脫皮、起水泡？」 |

**依部位追加詢問：**

| 照射部位 | 追加問題 |
|---------|---------|
| **頭頸部** | 「吃東西有沒有很痛或很難吞？有沒有口腔潰瘍？體重有沒有下降？」 |
| **胸部/食道** | 「吞嚥有沒有不舒服？有沒有新的喘氣或咳嗽？」 |
| **腹部/骨盆** | 「有沒有腹瀉？一天幾次？有沒有排尿不舒服或頻尿？」 |

**緊急警示：**
- 放療中斷超過 1 天（子宮頸癌品質指標：需在 63 天內完成）
- 新發生的喘氣或高燒（可能是放射性肺炎）
- 完全無法吞嚥（頭頸部放療）

---

## 腳本四：口服藥物追蹤（標靶/荷爾蒙/口服化療）

**目的**：確認服藥遵從性，主動偵測家中不易察覺的副作用。

**必問項目：**

| 項目 | 建議問法 |
|------|---------|
| **服藥遵從性** | 「藥有沒有每天按時吃？有沒有忘記？」 |
| **皮膚/皮疹** | 「皮膚有沒有出現紅疹、痘子、或乾燥脫皮？（標靶治療常見）」 |
| **腹瀉** | 「有沒有腹瀉？一天幾次？」 |
| **疲倦/體力** | 「體力怎麼樣？有沒有越來越沒力氣？」 |
| **關節/骨骼** | 「關節有沒有酸痛感？（荷爾蒙治療常見）」 |
| **血壓** | 「最近有沒有量血壓？（Bevacizumab 等需監測）」 |
| **藥物用完** | 「藥快吃完了嗎？下次門診記得帶藥袋讓醫師開下一批。」 |

**特別注意（依藥物）：**
- **荷爾蒙治療（乳癌 5–10 年）**：「有沒有想過要停藥？有任何讓您不想繼續吃藥的原因嗎？」
- **Capecitabine（截瘤達）**：「手腳有沒有脫皮、紅、腫、很痛？影響走路或拿東西嗎？」
- **標靶 TKI（如 Erlotinib）**：「臉部和背部有沒有出疹子？有沒有腹瀉超過 2 天？」

---

## 腳本五：首次未接電話後續處理

**第一次未接：**
```
留下 SMS/LINE 訊息：
「○○您好，我是秀傳癌症中心個管師○○○，
想關心您的狀況，方便時請回電 ○○○○○○，謝謝。」
```

**第二次未接（換不同時段）：**
```
再傳一次訊息，並嘗試聯絡緊急聯絡人。
```

**第三次未接：**
```
嘗試緊急聯絡人電話，說明：
「我是○○的主治醫師護理個管師，想確認○○目前的狀況，
請問○○最近有沒有回家或出去看診？」
```

**三次未接 + 緊急聯絡人也聯繫不上：**
- 依院內規定發掛號信
- 系統標記「嘗試聯絡中」
- 記錄：每次嘗試的日期、時間、方式

> 詳細失聯定義與後續處理流程：→ **[E｜困難個案處理](E_advanced.md)**
