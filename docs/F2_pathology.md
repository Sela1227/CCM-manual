## 基因與分子檢測

> 個管師不需要解讀基因報告，但要知道「這個檢測在等什麼結果、結果出來影響什麼治療」。

### 常見單一基因/分子檢測

| 檢測 | 相關癌別 | 用途 | 結果影響 |
|------|---------|------|---------|
| **EGFR 突變** | 非小細胞肺癌 | 標靶適用性 | 陽性 → Erlotinib/Osimertinib 等 |
| **ALK 融合** | 肺癌 | 標靶適用性 | 陽性 → Alectinib/Crizotinib 等 |
| **ROS-1 融合** | 肺癌 | 標靶適用性 | 陽性 → Crizotinib/Entrectinib |
| **KRAS / RAS 突變** | 大腸癌 | 排除抗 EGFR 標靶 | 突變 → Cetuximab **無效** |
| **BRAF V600E** | 大腸癌、肺癌、黑色素瘤 | 標靶適用性 | 陽性 → Dabrafenib + Trametinib |
| **HER2 擴增/過度表現** | 乳癌、胃癌 | 標靶適用性 | 陽性 → Trastuzumab、T-DM1 |
| **PD-L1 表現** | 多種癌別 | 免疫治療門檻 | TPS/CPS 高 → Pembrolizumab 給付 |
| **MSI-H / dMMR** | 大腸癌、子宮內膜癌等 | 免疫治療、遺傳評估 | 陽性 → 免疫治療有效；排查 Lynch 症候群 |
| **BRCA1/2 突變** | 乳癌、卵巢癌 | PARP 抑制劑；遺傳諮詢 | 突變 → Olaparib/Niraparib；家屬風險評估 |
| **NTRK 融合** | 任何癌別（癌種無關）| 標靶適用性 | 陽性 → Larotrectinib/Entrectinib |
| **FGFR2 融合** | 膽管癌 | 標靶適用性 | 陽性 → Pemigatinib |
| **IDH1/IDH2 突變** | 膽管癌、腦瘤、AML | 標靶適用性 | 突變 → Enasidenib 等 |
| **TMB（腫瘤突變負擔）** | 多種癌別 | 免疫治療預測 | TMB-High（≥10 mut/Mb）→ Pembrolizumab |
| **HRD（同源重組缺陷）** | 卵巢癌 | PARP 抑制劑 | 陽性 → Niraparib 給付條件之一 |

### NGS 次世代定序（二代定序）

**什麼是 NGS？**

NGS（Next Generation Sequencing）是一次同時偵測數十至數百個基因的技術，比傳統單基因檢測更全面，常用於：
- 確認標靶藥物適用性
- 找出罕見突變（單一基因檢測可能遺漏）
- 評估 TMB、MSI 等整體指標
- 偵測基因融合（fusion）

**組織型（Tissue-based NGS）vs. 液態切片（Liquid Biopsy）**

| 比較項目 | 組織型 NGS | 液態切片（血液 NGS）|
|---------|-----------|------------------|
| 樣本 | 腫瘤組織（手術/切片）| 血液（10 mL）|
| 敏感度 | 較高（腫瘤直接取樣）| 較低（偵測循環腫瘤 DNA）|
| 適用時機 | 初診斷、有組織可取 | 無法取組織；追蹤復發/抗藥突變 |
| 等待時間 | 2–4 週 | 1–2 週 |
| 健保給付 | 部分癌別有給付（如肺癌）| 多數自費 |

**常見 NGS 套組**

| 套組 | 偵測基因數 | 特色 |
|------|-----------|------|
| FoundationOne CDx | 324 個基因 + TMB + MSI | 美國 FDA 核准，廣泛癌別 |
| Oncomine | 依版本不同 | 常用於肺癌 |
| 院內自建 panel | 各院不同 | 成本較低，依院內需求設計 |

**健保給付 NGS 適用癌別（19 種，113 年 5 月起）**

| 類別 | 癌別清單 |
|------|---------|
| **14 大類實體腫瘤** | 非小細胞肺癌、三陰性乳癌、卵巢癌／輸卵管癌／原發性腹膜癌、攝護腺癌、胰臟癌、NTRK 基因融合實體腫瘤、肝內膽管癌、甲狀腺癌、甲狀腺髓質癌、大腸直腸癌 ★、泌尿道上皮癌 ★、黑色素瘤 ★、腸胃道間質瘤 ★、胃癌 ★ |
| **5 大類血液腫瘤** | 急性骨髓性白血病（AML）、高風險骨髓分化不良症狀群（MDS）、急性淋巴芽細胞白血病（B-ALL 及 T-ALL）、B 細胞淋巴癌（BCL）★、T 或 NK 細胞血癌與淋巴癌（NKTL）★ |

> **註 1**｜NGS 給付以檢測結果有對應「**具藥證的標靶藥物**」且「**效果明確之癌別及檢測基因**」為優先。
>
> **註 2**｜上表標 ★ 的 7 大類癌症，**專家共識建議採單基因檢測**（不一定用 NGS panel），個管師需留意主治醫師選擇的檢測方式。

**醫院執行資格（三條件，缺一不可）**

1. 限**區域級以上醫院**或**癌症診療品質認證醫院**
2. 為主管機關核定之**實驗室開發檢測（LDTs）施行計畫表列醫療機構**
3. 醫院須自行設立或跨院聯合組成**分子腫瘤委員會（Molecular Tumor Board, MTB）**

> 醫院須先向健保署提出申辦並通過審核，才能對病人收取 NGS 健保給付。

**檢測結果上傳義務**

健保給付的 NGS 檢測**結果必須上傳**至健保署系統，目的：

- **精準治療**：彙整資料協助臨床決策
- **免重複受檢**：跨院查得到同病人既往報告，避免重做
- **確保檢測品質**：主管機關後續監測各家實驗室準確度
- **評估治療效益**：累積實證資料支持後續健保政策調整

!!! tip "個管師追蹤重點"
    NGS 報告等待時間 **2–4 週**，若病人詢問為何遲遲未開始治療，需說明是在等基因報告以確定最適合的治療方向。報告出來後協助安排與主治醫師的討論。

!!! note "液態切片提醒"
    液態切片目前**多數自費**（部分癌別如肺癌有健保給付）。若主治醫師開立液態切片，需協助病人了解費用和流程，並追蹤結果是否已回到門診。

!!! warning "個管師對話要點"
    - 病人聽到「NGS」常以為等於「全部癌都能做」——實際是 19 種癌別、且優先給付有對應藥證的基因
    - 病人問「為什麼別的醫院能做我們不能」——可能是該院未通過 LDTs 表列或未設 MTB，並非歧視
    - 病人擔心「結果上傳會不會洩露隱私」——可說明這是健保管理義務（去識別化），且自己往後跨院就醫不用重做檢測

## 病理染色與免疫組織化學（IHC）

> 病理報告中常出現這些染色結果，個管師需要知道它們在說什麼、與治療決策的關係。

### IHC 判讀基礎

IHC（免疫組織化學染色）是在病理切片上用抗體偵測特定蛋白質是否表現。結果通常用「陽性（+）」或「陰性（-）」表示，有些用 **0 / 1+ / 2+ / 3+** 評分。

### 常見 IHC 染色

**荷爾蒙與 HER2（乳癌、婦癌）**

| 染色 | 意義 | 個管師重點 |
|------|------|-----------|
| **ER（雌激素受體）** | 乳癌荷爾蒙依賴性 | 陽性 → 荷爾蒙治療（Tamoxifen、AI）；治療期 5–10 年 |
| **PR（黃體素受體）** | 乳癌荷爾蒙依賴性 | 通常與 ER 同步判讀 |
| **HER2（IHC）** | HER2 蛋白表現量 | 0=陰性；1+=陰性；2+=不確定需 FISH 確認；**3+=陽性** → 標靶治療 |
| **Ki-67** | 腫瘤增殖指數（%）| 高（>20–30%）= 腫瘤生長快，預後較差；常影響是否加化療的決策 |

!!! note "HER2 2+ 的處理"
    HER2 IHC 2+ 是「不確定」，需要進一步做 **FISH（螢光原位雜交）** 確認是否有基因擴增。個管師追蹤時需確認 FISH 結果是否已完成。

**免疫治療相關（多癌別）**

| 染色 | 評分系統 | 意義 |
|------|---------|------|
| **PD-L1** | TPS（腫瘤比例分數）：腫瘤細胞陽性% | 肺癌：TPS≥50% 可用 Pembrolizumab 一線單藥 |
| **PD-L1** | CPS（綜合陽性分數）：(陽性腫瘤+免疫細胞)/總腫瘤×100 | 胃癌/頭頸癌：CPS≥10 或 ≥1 依藥物不同 |
| **PD-L1** | IC（免疫細胞分數）：免疫細胞陽性% | 膀胱癌 Atezolizumab 使用 |

**錯配修復蛋白（MMR）（大腸癌、子宮內膜癌等）**

| 染色 | 正常結果 | 異常（dMMR）| 意義 |
|------|---------|------------|------|
| **MLH1** | 陽性 | 陰性 | dMMR → MSI-H → 免疫治療有效；排查 Lynch 症候群 |
| **MSH2** | 陽性 | 陰性 | 同上 |
| **MSH6** | 陽性 | 陰性 | 同上 |
| **PMS2** | 陽性 | 陰性 | 同上 |

!!! tip "個管師重點"
    MMR 4 個蛋白**任一陰性** = dMMR（錯配修復缺陷）= MSI-H。大腸癌和子宮內膜癌病人若為 dMMR，需評估 Lynch 症候群遺傳風險，建議轉介遺傳諮詢。

**肺癌病理定性**

| 染色 | 意義 |
|------|------|
| **TTF-1** | 陽性 → 肺腺癌或甲狀腺癌；陰性 + p40/p63 陽性 → 肺鱗癌 |
| **p40 / p63** | 陽性 → 鱗狀細胞癌 |
| **Napsin A** | 陽性 → 肺腺癌（補強 TTF-1）|
| **Synaptophysin / Chromogranin A** | 陽性 → 神經內分泌腫瘤（類癌、小細胞肺癌）|

**腫瘤來源鑑別（轉移癌找原發灶）**

| 染色 | 陽性提示來源 |
|------|------------|
| **CK7** | 肺、乳房、卵巢、子宮內膜、膽管 |
| **CK20** | 大腸直腸、胃、胰臟、膀胱 |
| **CK7(+) / CK20(-)** | 肺腺癌、乳癌、卵巢癌 |
| **CK7(-) / CK20(+)** | 大腸直腸癌 |
| **PSA** | 攝護腺癌 |
| **AFP / Glypican-3** | 肝細胞癌 |
| **PAX8** | 腎臟、甲狀腺、女性生殖系統 |

**淋巴瘤分型相關**

| 染色 | 意義 |
|------|------|
| **CD20** | B 細胞淋巴瘤（Rituximab 標靶的目標）|
| **CD3** | T 細胞標記 |
| **CD10 / BCL-2** | 濾泡性淋巴瘤、DLBCL 分型 |
| **CD30** | 霍奇金淋巴瘤、ALCL |
| **Ki-67（淋巴瘤）** | >90% → 高度侵襲性（Burkitt 淋巴瘤接近 100%）|
