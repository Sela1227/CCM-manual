# C3. 大腸直腸癌照護指引

**彰濱秀傳癌症中心｜個管師指導書系列**

> 📌 本文件為大腸直腸癌專科訓練教材。內容架構與 C1 肺癌、C2 乳癌一致。
> ⚠️ **準則引用**：本檔治療策略依據 NCCN Colon Cancer 2025 v1、台灣大腸直腸外科醫學會建議。**待醫師團隊確認版本**。

## 一、疾病概述

### 流行病學

- 台灣癌症發生率：**第 2 位**（依國健署 108 年資料，標準化發生率 42.9/10 萬人）
- 台灣癌症死亡率：**第 3 位**
- 平均診斷年齡：**65–70 歲**
- 性別差異：男性略高於女性（直腸癌男性比女性多約 30%）
- 本院每年新案數：約（待補充）例

### 危險因子

| 危險因子 | 說明 |
|---------|------|
| **年齡** | ≥ 50 歲為高風險族群 |
| **大腸瘜肉** | 腺瘤性息肉是癌前病變 |
| **家族史** | 一等親有大腸癌史風險增加 |
| **遺傳症候群** | FAP（家族性腺瘤瘜肉症）、Lynch 症候群 |
| **發炎性腸病** | 潰瘍性結腸炎、克隆氏症長期發炎 |
| **生活習慣** | 紅肉/加工肉品、低纖維、肥胖、缺乏運動、抽菸、飲酒 |

### 常見症狀

| 部位 | 常見症狀 |
|------|---------|
| **遠端結腸 / 直腸**（約 60%）| 直腸出血、排便習慣改變、新發生便秘或腹瀉、大便變細、腹痛 |
| **近端右側結腸**（約 40%）| 慢性貧血、疲倦、腹部腫塊、體重減輕 |
| **直腸**（特殊症狀）| 排便急迫感、裡急後重感、排便帶血或黏液、排便不完全感 |
| **晚期/轉移** | 黃疸（肝轉移）、腹脹（腹水）、咳嗽肋膜積液（肺轉移）|

> ⚠️ 早期通常無症狀，**糞便潛血篩檢（FIT）**是早期診斷關鍵。

## 二、診斷與分期

### 診斷流程

```
糞便潛血陽性 / 症狀懷疑
        ↓
    大腸鏡檢查 + 切片
        ↓
    病理確診
        ↓
    分期檢查（CT 胸/腹/骨盆、MRI 直腸癌）
        ↓
    基因檢測（MSI/MMR、RAS、BRAF）
        ↓
    MDT 討論治療計畫
```

### 必要檢查項目

| 檢查 | 目的 | 備註 |
|------|------|------|
| **大腸鏡 + 切片** | 確診、評估腫瘤位置 | 必要 |
| **CT 胸腹骨盆** | 分期、評估遠端轉移 | 必要 |
| **直腸 MRI** | 直腸癌局部分期（T 期、N 期、CRM）| 直腸癌必要 |
| **CEA** | 腫瘤指數，治療反應與復發追蹤 | 必要 |
| **MSI/MMR 檢測** | 評估是否適合免疫治療、判斷 Lynch 症候群 | 全部需做 |
| **RAS / BRAF 基因** | 決定標靶藥物選擇（轉移性必做）| 轉移性必做 |

### TNM 分期（AJCC 第 8 版）

| 期別 | 大致說明 | 5 年存活率（參考）|
|------|---------|----------------|
| **0 期**（Tis）| 原位癌 | > 95% |
| **I 期** | T1-2, N0 | ~90% |
| **II 期** | T3-4, N0 | ~70-85% |
| **III 期** | 任何 T, N1-2 | ~50-70% |
| **IV 期** | M1（遠端轉移）| ~15% |

## 三、治療指引

> ⚠️ **依據準則**：NCCN Colon/Rectal Cancer 2025、台灣大腸直腸外科醫學會建議。**待醫師團隊確認版本與本院實作差異**。

### 早期（I-II 期）

- **手術切除**為主（部分結腸切除 + 局部淋巴結切除，N ≥ 12 顆）
- **II 期高風險者**（T4、淋巴結取樣 < 12、低分化、淋巴/血管/神經侵犯、術前阻塞或穿孔）→ 考慮輔助化療（FOLFOX 或 CapeOx）

### III 期（淋巴結轉移、無遠端轉移）

- **手術 + 術後輔助化療**（FOLFOX 或 CapeOx，6 個月）
- 直腸癌 III 期：**術前 CCRT 或全程新輔助療法（TNT）**+ 手術 + 術後輔助化療

### IV 期（轉移性 mCRC）

| 治療線別 | 治療策略 |
|---------|---------|
| **第一線** | 化療 ± 標靶（依 RAS/BRAF/MSI 分流）|
| **RAS wild-type** | 加 EGFR 抑制劑（Cetuximab / Panitumumab）|
| **RAS 突變** | 加 VEGF 抑制劑（Bevacizumab）|
| **BRAF V600E 突變** | Encorafenib + Cetuximab |
| **MSI-H / dMMR** | **免疫治療**（Pembrolizumab，健保 2025/6 起給付第一線）|
| **可切除肝/肺轉移** | 手術切除 + 圍術期化療（部分病人可治癒）|

### 直腸癌特殊：術前放化療

- **T3-4 或 N+** 病人需術前 CCRT 或 TNT（**全程新輔助療法**）
- 目的：縮小腫瘤、降低局部復發率、提高保肛機會

## 四、本癌別特殊注意事項

| 重點 | 說明 |
|------|------|
| **基因檢測** | MSI/MMR（全部）、RAS（轉移必做）、BRAF（轉移必做）|
| **造口照護** | 直腸癌部分需永久或暫時性造口（迴腸造口/結腸造口），需造口護理師協助 |
| **CEA 追蹤** | 治療前基準值、術後每 3-6 個月、急遽上升需查復發 |
| **直腸癌術前放化療** | T3/T4 或 N+ 必做，5-FU 或 Capecitabine 為基底 |
| **遺傳諮詢** | < 50 歲確診、家族史強烈、Lynch 症候群可能 → 轉介遺傳諮詢 |

## 六、個管照護重點

### 收案時

- 完成 CEA 基準值記錄
- 確認基因檢測（MSI/MMR）已開單
- 評估營養狀態（PG-SGA）—— 大腸癌常見體重減輕
- 評估 ECOG / Karnofsky—— 影響治療強度決定
- 心理困擾溫度計（DT）—— 確診衝擊大、需介入

### 治療期間

- **手術期**：傷口照護、造口照護衛教（如適用）、飲食重新適應
- **化療期**：FOLFOX 注意周邊神經病變（Oxaliplatin）；Capecitabine 注意手足症候群
- **CCRT 期**（直腸癌）：注意腹瀉、皮膚反應、頻尿
- **轉移性個案**：副作用管理 + 心理支持 + 安寧轉介時機評估

### 追蹤期

- 術後第 1-2 年：每 3 個月 CEA + 每 6 個月 CT
- 第 3-5 年：每 6 個月 CEA + 每年 CT
- 大腸鏡：術後 1 年、3 年、5 年（依風險調整）

## 七、常見問題與處理

### Q: 病人問「我為什麼要做造口？是不是一輩子？」

A: 直腸癌靠近肛門時可能需要造口。**永久性造口**（如低位直腸癌切除後）約佔少數；**暫時性造口**多為保護吻合處、術後 3-6 個月可關回。實際情況請主治醫師說明。

### Q: 病人問「我做完手術，為什麼還要化療？」

A: 病理顯示淋巴結有轉移（III 期）或 II 期高風險，**輔助化療可降低 30-40% 復發率**。這是「保險」，不是治療失敗。

### Q: 病人是 RAS 突變/BRAF 突變，治療選擇有差嗎？

A: 是的。**RAS 突變**不能用 EGFR 抑制劑（Cetuximab/Panitumumab），改用 VEGF 抑制劑（Bevacizumab）；**BRAF V600E 突變**有專屬組合療法（Encorafenib + Cetuximab）。基因型決定藥物選擇。

### Q: 病人問「我是 MSI-H，聽說有免疫治療？」

A: 是的。**MSI-H / dMMR 的轉移性大腸癌可用 Pembrolizumab**（健保 2025/6 起第一線給付）。MSI-H 約佔轉移性大腸癌 5-7%，反應率比一般化療高。
