# C5. 頭頸癌照護指引

**彰濱秀傳癌症中心｜個管師指導書系列**

> 📌 本文件為頭頸癌專科訓練教材，含口腔癌、口咽癌、下咽癌、喉癌、鼻咽癌等。內容架構與 C1 肺癌、C2 乳癌一致。
> ⚠️ **準則引用**：NCCN Head and Neck Cancers 2025、台灣頭頸部腫瘤醫學會。**待醫師團隊確認版本**。

---

## 一、疾病概述

### 流行病學

- 台灣男性癌症發生率：**第 4 位**（口腔癌為主）
- 25-44 歲男性癌症發生率：**第 1-2 位**（口腔癌）
- 男女比：男性遠高於女性（與檳榔/菸/酒高度相關）
- 主要部位：口腔（最多）、口咽、下咽、喉、鼻咽

### 危險因子

| 部位 | 主要危險因子 |
|------|------------|
| **口腔癌 / 口咽癌 / 下咽癌 / 喉癌** | 檳榔、抽菸、飲酒（**檳菸酒三聯**最高風險）|
| **鼻咽癌** | EBV 病毒感染、高鹽醃漬食品、家族史、廣東/福建族群 |
| **HPV 相關口咽癌** | HPV 16 型感染（年輕、非吸菸者比例上升）|

### 常見症狀

| 部位 | 症狀 |
|------|------|
| **口腔** | 潰瘍 > 2 週不癒、白斑/紅斑、咀嚼疼痛、牙齒鬆動、頸部腫塊 |
| **鼻咽** | 鼻塞、流鼻血、聽力下降（單側中耳積液）、頸部腫塊（最常見的就診原因）|
| **下咽 / 喉** | 聲音沙啞 > 2 週、吞嚥困難、異物感、咳血 |

> ⚠️ **頸部不明腫塊 + 上呼吸消化道症狀**永遠要懷疑頭頸癌。

## 二、診斷與分期

### 診斷流程

```
症狀懷疑 / 健檢發現
        ↓
    耳鼻喉科內視鏡 + 切片
        ↓
    病理確診（多為鱗狀細胞癌）
        ↓
    分期檢查（CT/MRI、PET-CT）
    + 鼻咽癌：EBV DNA 定量
        ↓
    HPV 檢測（口咽癌）
        ↓
    多學科 MDT（含牙科、營養師、語言治療師）
```

### 必要檢查項目

| 檢查 | 目的 | 備註 |
|------|------|------|
| **內視鏡 + 切片** | 確診 | 必要 |
| **頸部 CT / MRI** | T、N 分期 | 必要 |
| **PET-CT** | 評估遠端轉移、第二原發癌 | III-IV 期建議 |
| **鼻咽癌 EBV DNA** | 確診、追蹤、預後因子 | 鼻咽癌必查 |
| **HPV / p16 檢測** | 口咽癌預後分組 | 口咽癌必查 |
| **牙科評估** | 放療前牙齒處理（避免放射性骨壞死）| 放療前必做 |
| **吞嚥/語言評估** | 治療前基線、治療中追蹤 | 必要 |
| **營養評估（PG-SGA）** | 治療前 50% 已有營養不良 | 必要 |

### TNM 分期（AJCC 第 8 版）

> 頭頸癌各部位有不同的 TNM 規則，以下為通則。**口咽癌依 HPV/p16 狀態分為兩套分期系統**（HPV+ 預後較好）。

| 期別 | 大致說明 | 5 年存活率 |
|------|---------|----------|
| **I 期** | T1, N0 | 80–90% |
| **II 期** | T2, N0 | 60–80% |
| **III 期** | T3 或 N1 | ~50% |
| **IV 期** | T4 / N2-3 / M1 | ~30% |

## 三、治療指引

> ⚠️ **依據準則**：NCCN Head and Neck Cancers 2025、台灣頭頸部腫瘤醫學會。**待醫師團隊確認版本與本院實作差異**。

### 早期（I-II 期）

- **單一治療**：手術 **或** 放療
- 部位選擇：口腔癌偏手術、喉癌可選放療保留發聲

### 局部晚期（III-IVA-B 期）

| 部位 | 治療策略 |
|------|---------|
| **口腔癌** | 手術 + 術後放療（高風險再加化療 → 同步放化療 CCRT）|
| **口咽 / 下咽 / 喉癌** | **CCRT（同步放化療）** 為標準（器官保留療法） |
| **鼻咽癌** | **CCRT 為主治療**（鼻咽癌不適合手術，對放療反應好）|
| **N+ 病人** | 加頸部淋巴結廓清術或頸部放療 |

### 復發 / 轉移性

- 第一線：**Pembrolizumab**（PD-L1+，免疫治療）± 化療
- 第二線：Cetuximab + 化療、其他免疫
- 鼻咽癌特殊：對化療反應較好（PF: Cisplatin + 5-FU）

### 鼻咽癌特殊提醒

- **不適合手術**（位置鄰近顱底、血管），主治療為**放療**
- 早期單獨放療、中晚期 CCRT
- 治療中 EBV DNA 定量追蹤

## 四、本癌別特殊注意事項

| 重點 | 說明 |
|------|------|
| **放療前牙科評估** | 放射線可能導致放射性骨壞死，治療前要拔不良牙齒、處理蛀牙 |
| **吞嚥功能** | 治療前評估、治療中監測、治療後語言治療師復健 |
| **說話功能** | 喉癌切除後可能用人工發聲、食道發聲 |
| **外觀改變** | 手術切除大範圍時影響顏面，需心理支持 + 整形重建 |
| **戒菸戒檳** | 治療開始前、治療中、治療後**都要持續衛教** |
| **營養支持** | 治療中體重會下降 5-10%，常需鼻胃管 / 胃造口 |
| **口腔照護** | 放療中口腔黏膜炎、口乾——專屬口腔護理衛教 |
| **頸部放療皮膚照護** | 放療區皮膚會變色、脫皮——保持乾燥、避免日曬與摩擦 |

## 六、個管照護重點

### 收案時

- 完成 PG-SGA、體重基準
- 牙科評估狀態（放療前重要）
- 戒菸戒檳衛教（持續、反覆）
- 心理支持——頭頸癌外觀+功能影響大、心理衝擊強
- 評估社會支持（家屬照顧能力）

### 治療期間

- **手術期**：傷口照護、氣管造口照護、語言治療師早期介入
- **CCRT 期**：
  - 口腔黏膜炎管理（最常見、最痛苦）
  - 體重下降監測（每週量、降 10% 警示）
  - 鼻胃管 / 胃造口時機評估
  - 放療皮膚照護
  - Cisplatin 注意腎毒性、聽力
- **放療結束後**：口乾、味覺改變、頸部僵硬可能持續半年至一年

### 追蹤期

- 第 1 年：每 1-3 個月
- 第 2 年：每 2-4 個月
- 第 3-5 年：每 4-6 個月
- 5 年後：每年
- 鼻咽癌追蹤 EBV DNA、影像

## 七、常見問題與處理

### Q: 病人說「我以後不能說話了嗎？」

A: 看部位與治療方式。**喉癌全切除**會失去自然發聲，但有電子喉、食道發聲、人工發聲鈕等替代方案。**保留性治療**（CCRT）多數病人能保留聲音。語言治療師會在治療前後介入訓練。

### Q: 病人說「我已經戒檳榔/菸/酒了，為什麼還會復發風險高？」

A: 已戒是好事，**繼續戒**才能降低**第二原發癌**風險（口腔癌病人有較高機率得到食道癌、肺癌）。完全戒除後 5-10 年風險才接近一般族群。

### Q: 病人放療中嘴巴痛到吃不下，怎麼辦？

A: **口腔黏膜炎**是 CCRT 最常見的副作用。處理：(1) 鹽水/小蘇打水漱口（不用酒精性漱口水）(2) 局部止痛凝膠 (3) 軟質、室溫食物 (4) 嚴重時改鼻胃管 / 胃造口。**不要中斷治療**——中斷會降低治癒率。

### Q: 病人問「為什麼我要先看牙科？」

A: 放療範圍含口腔時，**放射線會影響牙齒血流、唾液腺**，未來拔牙容易引起放射性骨壞死。治療前先處理蛀牙、拔不良牙齒，避免治療後悔不及。

### Q: 鼻咽癌病人問「為什麼別人都開刀我不用？」

A: 鼻咽位置鄰近顱底、大血管、神經，**手術完全切除幾乎不可能**。但鼻咽癌對放射線反應非常好，所以**放療為主治療**，治癒率不亞於手術。
