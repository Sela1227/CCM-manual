## 一、癌症分期系統

### TNM 分期（最廣泛使用）

TNM 是**解剖學分期**，由三個維度組合決定期別：

| 維度 | 意義 | 常見數值 |
|------|------|---------|
| **T（Tumor）** | 原發腫瘤大小與局部侵犯程度 | T1–T4（越大/越侵犯越高）；Tis = 原位癌；T0 = 無原發腫瘤 |
| **N（Node）** | 區域淋巴結轉移情形 | N0 = 無轉移；N1–N3（依解剖位置）|
| **M（Metastasis）** | 遠端轉移 | M0 = 無遠端轉移；M1 = 有遠端轉移 |

**Stage 分組（大多數實體癌）**

| Stage | 概念 | 大方向 |
|-------|------|--------|
| **Stage 0** | 原位癌（Carcinoma in situ, CIS）| 癌細胞尚未侵犯基底膜 |
| **Stage I** | 局限性（小、無淋巴轉移）| 手術或局部治療通常可根治 |
| **Stage II** | 局部侵犯或少量淋巴轉移 | 通常可切除，多需術後輔助治療 |
| **Stage III** | 較大局部侵犯或較多淋巴轉移 | 部分可切除，常需 CCRT 或多模式治療 |
| **Stage IV** | 遠端轉移（M1）| 通常已不可根治，以控制疾病/維持品質為目標 |

!!! tip "個管師記住這一點"
    Stage I–III = 「局部區域性疾病」= 仍有根治可能；Stage IV = 「轉移性疾病」= 以控制為主。這個概念決定了治療目標的整個方向。

### 臨床分期 vs. 病理分期

| 類型 | 英文 | 時機 | 說明 |
|------|------|------|------|
| **臨床分期** | cTNM | 治療前 | 依影像、體檢、切片結果 |
| **病理分期** | pTNM | 手術後 | 依手術標本的病理報告，更精確 |
| **前導治療後分期** | ypTNM | 術前化療/放療後手術 | 評估治療反應 |

> 品質指標的分母條件常區分「臨床分期」vs.「病理分期」，個管師填報時需注意。

### 各癌別特殊分期系統

部分癌別有自己的分期系統（與 TNM 並行使用）：

| 分期系統 | 適用癌別 | 特點 |
|---------|---------|------|
| **FIGO** | 婦癌（子宮頸、子宮體、卵巢）| 以羅馬數字（I–IV）+ 英文字母細分 |
| **BCLC** | 肝細胞癌（HCC）| 整合腫瘤大小、肝功能（Child-Pugh）、體能狀態，決定治療選擇 |
| **Ann Arbor** | 淋巴瘤 | Stage I–IV 依受侵犯的淋巴結區域數目和橫膈位置 |
| **Gleason / Grade Group** | 攝護腺癌 | 病理分化程度，Gleason score 6–10；Grade Group 1–5 |
| **Durie-Salmon / ISS** | 多發性骨髓瘤 | 整合血液/骨髓指標 |

## 二、病理報告解讀

> 個管師不需要解讀所有病理細節，但要知道哪些欄位影響治療決策、哪些是品質指標的關鍵項目。

### 病理報告必看項目

**① 組織形態（Histology / Morphology）**

說明腫瘤是哪種細胞來源：

| 類型 | 說明 | 常見癌別 |
|------|------|---------|
| **腺癌（Adenocarcinoma）** | 腺體細胞 | 肺腺癌、大腸癌、胃癌、乳癌 |
| **鱗狀細胞癌（Squamous cell carcinoma）** | 鱗狀上皮 | 頭頸癌、肺鱗癌、子宮頸癌、食道癌 |
| **小細胞癌（Small cell carcinoma）** | 神經內分泌 | 小細胞肺癌（SCLC），惡性度高 |
| **移行上皮癌（Urothelial / TCC）** | 泌尿道上皮 | 膀胱癌、上泌尿道癌 |

**② 腫瘤分級（Grade）**

反映腫瘤的分化程度，越高分化越差、惡性度越高：

| Grade | 說明 |
|-------|------|
| **G1 / Well-differentiated** | 高分化，接近正常細胞，通常預後較好 |
| **G2 / Moderately-differentiated** | 中分化 |
| **G3 / Poorly-differentiated** | 低分化，細胞異型性大，惡性度高 |
| **GX** | 無法評估 |

**③ 手術邊緣（Surgical Margin）**

| 結果 | 意義 |
|------|------|
| **R0（切緣陰性）** | 邊緣無殘存癌細胞——手術目標 |
| **R1（顯微鏡下陽性）** | 顯微鏡下見殘存癌細胞，可能需追加放療 |
| **R2（肉眼陽性）** | 肉眼可見殘存腫瘤 |

> R0 切除率是胃癌、食道癌、肝癌、胰臟癌的品質指標。

**④ 淋巴結（Lymph Node）**

病理報告格式：`檢查 X 顆，其中 Y 顆陽性`

- **淋巴結數目**：多個癌別有「最低需檢查顆數」的品質指標（大腸癌 ≥12 顆、口腔癌 ≥15 顆等）
- **陽性比率**：影響是否需要輔助放療（如乳癌 ≥4 顆陽性）

**⑤ 淋巴管/血管侵犯（Lymphovascular Invasion, LVI）**

- 陽性（Positive）= 癌細胞已進入淋巴管或血管，轉移風險較高
- 常影響是否需要輔助治療的決策

**⑥ 神經周圍侵犯（Perineural Invasion, PNI）**

- 陽性表示癌細胞沿神經蔓延，與局部復發率相關
- 口腔癌切緣評估的重要指標之一

## 三、功能狀態評估（ECOG）

功能狀態（Performance Status, PS）反映病人的體能和日常活動能力，直接影響「這個病人能否耐受積極治療」。

### ECOG 量表

| ECOG 分數 | 定義 | 臨床意義 |
|----------|------|---------|
| **0** | 完全正常活動，無任何限制 | 可接受所有標準治療 |
| **1** | 有症狀但可自理日常，輕度限制體力活動，可正常行走工作 | 可接受大多數積極治療 |
| **2** | 可自理日常，但無法從事工作或體力活動；臥床時間 < 白天 50% | 化療劑量可能需調整；部分臨床試驗排除 |
| **3** | 日常自理有限，臥床時間 > 白天 50% | 通常只接受支持性或姑息治療；積極化療風險高 |
| **4** | 完全臥床，無法自理 | 以安寧舒適為主 |

!!! warning "PS 對治療的影響"
    ECOG ≥ 2 通常是積極化療的相對禁忌或需謹慎評估。免疫治療通常要求 ECOG ≤ 1。個管師評估病人體能時，可用這個量表作為共通語言。

## 四、確診到治療的標準流程

### 通用流程（各癌別共同）

```
懷疑 → 切片確診 → 分期影像 → MDT 討論 → 治療計畫 → 治療開始
```

| 節點 | 說明 | 個管師任務 |
|------|------|-----------|
| **懷疑病灶** | 影像發現腫塊或症狀異常 | ── |
| **切片確診** | 病理確認惡性與組織型態 | 確認病理報告已取得並與醫師討論 |
| **分期影像** | CT / MRI / PET-CT 完成全身評估 | 協調影像預約；確認影像已完成 |
| **MDT 討論** | 多科會診共同決定治療策略 | 準備個案資料；確認納入 MDT 討論 |
| **治療計畫書** | 明確記錄治療目標、方式、時程 | 與病人說明治療計畫內容 |
| **治療開始** | 依計畫執行手術 / 化療 / 放療等 | 追蹤治療是否如期開始 |
| **追蹤影像** | 定期評估治療反應 | 確認追蹤影像預約；結果回診討論 |

### 治療開始的時效性（品質指標相關）

| 癌別 | 時效要求 | 品質指標 |
|------|---------|---------|
| 大腸癌（直腸癌）| 確診後 ≤ 6 週開始治療 | 強制指標 |
| 口腔癌 | 手術後 ≤ 6 週開始輔助放療 | 強制指標 |
| 子宮頸癌 | 放療 ≤ 63 天完成 | 強制指標 |
| 肝癌 BCLC 0–B | 確診後 ≤ 45 天開始治療 | 強制指標 |
| 胰臟癌 | 確診後 ≤ 30 天開始治療 | 強制指標 |


> 完整 60 項指標規格 → 詳見 **[G｜品質指標速查](G_quality-index.md)**。

## 五、跨癌別個管照護原則

### 初次收案評估（所有癌別共同）

新收案的個案，無論癌別，個管師應在初次評估時確認以下項目：

**疾病資訊確認**
- 確診日期和病理確認日期
- 癌別、分期（臨床分期已完成）
- 主治醫師和計畫治療方式

**病人評估**
- ECOG 功能狀態（0–4）
- 主要症狀與不適
- 營養狀態（PG-SGA 評估）
- 心理困擾篩檢（Distress Thermometer）

**資源評估**
- 主要照護者與社會支持狀況
- 經濟狀況（是否需轉介社工）
- 交通與就醫障礙

### 追蹤電話通則

每次追蹤電話的結構建議：

1. **開場**：「這個禮拜怎麼樣？有沒有不舒服？」
2. **治療相關**：依目前治療方式（手術後/化療中/放療中）問副作用
3. **緊急警示確認**：「有沒有發燒超過 38 度？」
4. **下一步確認**：「下次回診是什麼時候？有沒有預約影像？」
5. **記錄**：追蹤系統登錄、更新狀態

### 需要立即升級處理的情況

!!! warning "以下情況需立即請病人就醫或通知主治醫師"
    - 發燒 > 38°C（化療中或術後病人）
    - 突發胸痛或呼吸困難
    - 嚴重嘔吐超過 24 小時無法進食
    - 意識改變、肢體無力
    - 傷口紅腫熱痛或滲液

### 失聯個案處理原則

電話聯絡多次未接 → 簡訊留言 → 緊急聯絡人 → 掛號信 → 標記失聯並記錄。
**詳細處理流程與預防策略** → 詳見 **[E｜困難個案處理](E_advanced.md)**。

> 失聯率定義：應追蹤個案中，失聯達 **≥ 12 個月**者。（詳見 G｜品質指標）
