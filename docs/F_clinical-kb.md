## F1. 常見檢驗項目

> 日常查表用。個管師不需要記住所有數值，但要知道「什麼時候這個數字代表需要關注」。

---

### 血液常規（CBC）

| 項目 | 正常值（參考）| 偏低時 | 個管師重點 |
|------|------------|------|-----------|
| **WBC 白血球** | 4.5–11.0 ×10³/µL | 感染風險高 | 化療後 7–14 天最低，發燒 > 38°C 立即就醫 |
| **ANC 嗜中性球** | > 1.5 ×10³/µL | < 0.5 = 嚴重風險 | ANC < 1.0 要小心；< 0.5 + 發燒 = 緊急 |
| **Hb 血紅素** | 男 13.5–17.5 / 女 12–16 g/dL | 貧血、疲倦、喘 | 追蹤體力；< 8 可能需輸血 |
| **Platelet 血小板** | 150–400 ×10³/µL | 出血風險 | < 50 避免碰撞；< 20 有自發出血風險 |

### 生化檢查

| 項目 | 正常值（參考）| 臨床意義 | 個管師注意 |
|------|------------|---------|-----------|
| **AST / ALT** | < 40 U/L | 肝功能 | 化療前必查；異常可能需調整劑量 |
| **BUN / Creatinine** | BUN 7–20 / Cr 0.6–1.2 mg/dL | 腎功能 | Cisplatin 腎毒性追蹤 |
| **Albumin** | 3.5–5.0 g/dL | 營養狀態 | < 3.0 = 嚴重營養不良，需轉介營養師 |
| **電解質（Na / K / Mg）** | 各有範圍 | 嘔吐腹瀉後易流失 | 嚴重嘔吐或腹瀉後需追蹤 |

### 腫瘤指數

> ⚠️ 腫瘤指數**不能單獨用來診斷癌症**，正常也不代表無癌，主要用於追蹤治療反應和復發偵測。

| 指數 | 主要相關癌別 | 臨床用途 |
|------|-----------|---------|
| **CEA** | 大腸癌（首選）、胃癌、肺癌 | 治療反應追蹤、術後復發偵測 |
| **AFP** | 肝癌、生殖細胞瘤 | 肝癌診斷輔助、治療追蹤 |
| **CA-125** | 卵巢癌 | 治療反應、復發追蹤 |
| **CA19-9** | 胰臟癌、膽道癌、胃癌 | 治療反應追蹤 |
| **CA15-3** | 乳癌 | 轉移性乳癌追蹤 |
| **PSA** | 攝護腺癌 | 診斷輔助、治療追蹤 |
| **SCC Ag** | 頭頸癌（鱗狀）、子宮頸癌 | 治療反應追蹤 |
| **NSE** | 小細胞肺癌、神經內分泌瘤 | 治療追蹤 |


### 基因與分子檢測

> 個管師不需要解讀基因報告，但要知道「這個檢測在等什麼結果、結果出來影響什麼治療」。

#### 常見單一基因/分子檢測

| 檢測 | 相關癌別 | 用途 | 結果影響 |
|------|---------|------|---------|
| **EGFR 突變** | 非小細胞肺癌 | 標靶適用性 | 陽性 → Erlotinib/Osimertinib 等 |
| **ALK 融合** | 肺癌 | 標靶適用性 | 陽性 → Alectinib/Crizotinib 等 |
| **ROS-1 融合** | 肺癌 | 標靶適用性 | 陽性 → Crizotinib/Entrectinib |
| **KRAS / RAS 突變** | 大腸癌 | 排除抗 EGFR 標靶 | 突變 → Cetuximab **無效** |
| **BRAF V600E** | 大腸癌、肺癌、黑色素瘤 | 標靶適用性 | 陽性 → Dabrafenib + Trametinib |
| **HER2 擴增/過度表現** | 乳癌、胃癌 | 標靶適用性 | 陽性 → Trastuzumab、T-DM1 |
| **PD-L1 表現** | 多種癌別 | 免疫治療門檻 | TPS/CPS 高 → Pembrolizumab 給付 |
| **MSI-H / dMMR** | 大腸癌、子宮內膜癌等 | 免疫治療、遺傳評估 | 陽性 → 免疫治療有效；排查 Lynch 症候群 |
| **BRCA1/2 突變** | 乳癌、卵巢癌 | PARP 抑制劑；遺傳諮詢 | 突變 → Olaparib/Niraparib；家屬風險評估 |
| **NTRK 融合** | 任何癌別（癌種無關）| 標靶適用性 | 陽性 → Larotrectinib/Entrectinib |
| **FGFR2 融合** | 膽管癌 | 標靶適用性 | 陽性 → Pemigatinib |
| **IDH1/IDH2 突變** | 膽管癌、腦瘤、AML | 標靶適用性 | 突變 → Enasidenib 等 |
| **TMB（腫瘤突變負擔）** | 多種癌別 | 免疫治療預測 | TMB-High（≥10 mut/Mb）→ Pembrolizumab |
| **HRD（同源重組缺陷）** | 卵巢癌 | PARP 抑制劑 | 陽性 → Niraparib 給付條件之一 |

#### NGS 次世代定序（二代定序）

**什麼是 NGS？**

NGS（Next Generation Sequencing）是一次同時偵測數十至數百個基因的技術，比傳統單基因檢測更全面，常用於：
- 確認標靶藥物適用性
- 找出罕見突變（單一基因檢測可能遺漏）
- 評估 TMB、MSI 等整體指標
- 偵測基因融合（fusion）

**組織型（Tissue-based NGS）vs. 液態切片（Liquid Biopsy）**

| 比較項目 | 組織型 NGS | 液態切片（血液 NGS）|
|---------|-----------|------------------|
| 樣本 | 腫瘤組織（手術/切片）| 血液（10 mL）|
| 敏感度 | 較高（腫瘤直接取樣）| 較低（偵測循環腫瘤 DNA）|
| 適用時機 | 初診斷、有組織可取 | 無法取組織；追蹤復發/抗藥突變 |
| 等待時間 | 2–4 週 | 1–2 週 |
| 健保給付 | 部分癌別有給付（如肺癌）| 多數自費 |

**常見 NGS 套組**

| 套組 | 偵測基因數 | 特色 |
|------|-----------|------|
| FoundationOne CDx | 324 個基因 + TMB + MSI | 美國 FDA 核准，廣泛癌別 |
| Oncomine | 依版本不同 | 常用於肺癌 |
| 院內自建 panel | 各院不同 | 成本較低，依院內需求設計 |

!!! tip "個管師追蹤重點"
    NGS 報告等待時間 **2–4 週**，若病人詢問為何遲遲未開始治療，需說明是在等基因報告以確定最適合的治療方向。報告出來後協助安排與主治醫師的討論。

!!! note "液態切片提醒"
    液態切片目前**多數自費**（部分癌別如肺癌有健保給付）。若主治醫師開立液態切片，需協助病人了解費用和流程，並追蹤結果是否已回到門診。

### 病理染色與免疫組織化學（IHC）

> 病理報告中常出現這些染色結果，個管師需要知道它們在說什麼、與治療決策的關係。

#### IHC 判讀基礎

IHC（免疫組織化學染色）是在病理切片上用抗體偵測特定蛋白質是否表現。結果通常用「陽性（+）」或「陰性（-）」表示，有些用 **0 / 1+ / 2+ / 3+** 評分。

#### 常見 IHC 染色

**荷爾蒙與 HER2（乳癌、婦癌）**

| 染色 | 意義 | 個管師重點 |
|------|------|-----------|
| **ER（雌激素受體）** | 乳癌荷爾蒙依賴性 | 陽性 → 荷爾蒙治療（Tamoxifen、AI）；治療期 5–10 年 |
| **PR（黃體素受體）** | 乳癌荷爾蒙依賴性 | 通常與 ER 同步判讀 |
| **HER2（IHC）** | HER2 蛋白表現量 | 0=陰性；1+=陰性；2+=不確定需 FISH 確認；**3+=陽性** → 標靶治療 |
| **Ki-67** | 腫瘤增殖指數（%）| 高（>20–30%）= 腫瘤生長快，預後較差；常影響是否加化療的決策 |

!!! note "HER2 2+ 的處理"
    HER2 IHC 2+ 是「不確定」，需要進一步做 **FISH（螢光原位雜交）** 確認是否有基因擴增。個管師追蹤時需確認 FISH 結果是否已完成。

**免疫治療相關（多癌別）**

| 染色 | 評分系統 | 意義 |
|------|---------|------|
| **PD-L1** | TPS（腫瘤比例分數）：腫瘤細胞陽性% | 肺癌：TPS≥50% 可用 Pembrolizumab 一線單藥 |
| **PD-L1** | CPS（綜合陽性分數）：(陽性腫瘤+免疫細胞)/總腫瘤×100 | 胃癌/頭頸癌：CPS≥10 或 ≥1 依藥物不同 |
| **PD-L1** | IC（免疫細胞分數）：免疫細胞陽性% | 膀胱癌 Atezolizumab 使用 |

**錯配修復蛋白（MMR）（大腸癌、子宮內膜癌等）**

| 染色 | 正常結果 | 異常（dMMR）| 意義 |
|------|---------|------------|------|
| **MLH1** | 陽性 | 陰性 | dMMR → MSI-H → 免疫治療有效；排查 Lynch 症候群 |
| **MSH2** | 陽性 | 陰性 | 同上 |
| **MSH6** | 陽性 | 陰性 | 同上 |
| **PMS2** | 陽性 | 陰性 | 同上 |

!!! tip "個管師重點"
    MMR 4 個蛋白**任一陰性** = dMMR（錯配修復缺陷）= MSI-H。大腸癌和子宮內膜癌病人若為 dMMR，需評估 Lynch 症候群遺傳風險，建議轉介遺傳諮詢。

**肺癌病理定性**

| 染色 | 意義 |
|------|------|
| **TTF-1** | 陽性 → 肺腺癌或甲狀腺癌；陰性 + p40/p63 陽性 → 肺鱗癌 |
| **p40 / p63** | 陽性 → 鱗狀細胞癌 |
| **Napsin A** | 陽性 → 肺腺癌（補強 TTF-1）|
| **Synaptophysin / Chromogranin A** | 陽性 → 神經內分泌腫瘤（類癌、小細胞肺癌）|

**腫瘤來源鑑別（轉移癌找原發灶）**

| 染色 | 陽性提示來源 |
|------|------------|
| **CK7** | 肺、乳房、卵巢、子宮內膜、膽管 |
| **CK20** | 大腸直腸、胃、胰臟、膀胱 |
| **CK7(+) / CK20(-)** | 肺腺癌、乳癌、卵巢癌 |
| **CK7(-) / CK20(+)** | 大腸直腸癌 |
| **PSA** | 攝護腺癌 |
| **AFP / Glypican-3** | 肝細胞癌 |
| **PAX8** | 腎臟、甲狀腺、女性生殖系統 |

**淋巴瘤分型相關**

| 染色 | 意義 |
|------|------|
| **CD20** | B 細胞淋巴瘤（Rituximab 標靶的目標）|
| **CD3** | T 細胞標記 |
| **CD10 / BCL-2** | 濾泡性淋巴瘤、DLBCL 分型 |
| **CD30** | 霍奇金淋巴瘤、ALCL |
| **Ki-67（淋巴瘤）** | >90% → 高度侵襲性（Burkitt 淋巴瘤接近 100%）|

### 常見影像檢查

> 各種影像檢查有不同的目的和限制，個管師需要知道「什麼情況開什麼影像」以及「結果出來後要做什麼」。

#### 影像檢查總覽

| 檢查 | 原理 | 優點 | 限制 |
|------|------|------|------|
| **X 光** | 放射線 | 快速、便宜、輻射低 | 解析度低，早期病灶易遺漏 |
| **CT（電腦斷層）** | 放射線（較高劑量）| 快速、精細、全身可做 | 輻射、顯影劑過敏風險 |
| **MRI（核磁共振）** | 磁場、無輻射 | 軟組織對比最好（腦、肝、骨髓）| 時間長（30–90分鐘）、幽閉、金屬禁忌 |
| **PET-CT（正子掃描）** | 葡萄糖代謝 + CT | 全身功能性評估，找轉移 | 費用高（多自費）、輻射、腦和腎臟不易判讀 |
| **超音波** | 聲波 | 無輻射、即時、便宜 | 深部結構受限、操作者依賴性高 |
| **骨骼掃描** | 放射性同位素 | 全身骨骼轉移篩查 | 假陽性率高（骨折、退化也會亮）|
| **PSMA PET-CT** | 放射性核素標記（靶向 PSMA 蛋白）| 攝護腺癌高敏感度分期/復發偵測 | 費用高（自費）；非攝護腺癌無意義 |
| **乳房攝影（乳攝）** | 低劑量 X 光 | 早期乳癌偵測（微鈣化）| 緻密乳房敏感度低；需正面/斜面兩個角度 |
| **乳房超音波（乳超）** | 聲波 | 無輻射；緻密乳房補強；引導切片 | 操作者依賴；對微鈣化不如乳攝敏感 |

#### 各癌別常用影像

**肺癌**

| 時機 | 常用影像 | 目的 |
|------|---------|------|
| 初診斷/分期 | 胸部 CT + 腹部 CT + 頭部 MRI | TNM 分期、腦轉移評估 |
| PET-CT | 縱膈腔淋巴結、遠端轉移 | 手術前必要（部分）|
| 治療追蹤 | 胸部 CT（含腹部）每 2–3 個月 | 評估治療反應（RECIST 標準）|
| 腦轉移追蹤 | 頭部 MRI with 顯影 | 最敏感的腦轉移評估方式 |

**乳癌**

| 時機 | 常用影像 | 目的 |
|------|---------|------|
| 篩檢（一般族群）| 乳房攝影（乳攝）每 1–2 年 | 早期偵測微鈣化、腫塊 |
| 篩檢（緻密乳房）| 乳攝 + 乳房超音波（乳超）| 乳攝對緻密乳房敏感度低，乳超補強 |
| 初診斷/術前評估 | 乳房 MRI | 範圍、多發性病灶、對側乳房評估 |
| 確診後分期 | 胸腹骨盆 CT + 骨骼掃描 | 局部晚期或轉移性乳癌 |
| 腋下淋巴結評估 | 腋下超音波 ± 切片 | 判斷淋巴結是否轉移 |
| 術後追蹤 | 年度乳攝（手術側 + 對側）| 局部復發、對側新發癌 |

!!! note "乳攝 vs. 乳超——個管師常見問題"
    - **乳攝（乳房攝影）**：低劑量 X 光，最佳的微鈣化偵測工具，是乳癌篩檢黃金標準。需正面（CC 位）+ 斜面（MLO 位）兩個角度。BI-RADS 0–6 分類（4 以上建議切片）。
    - **乳超（乳房超音波）**：無輻射，對「緻密乳房」的實質性腫塊敏感度更高，也用於引導切片、腋下淋巴結評估。但對微鈣化不如乳攝敏感。
    - **兩者互補**：緻密乳房（腺體豐富）的年輕女性建議乳攝加乳超；乳房脂肪化程度高的停經後女性，單做乳攝即可。
    - **個管師追蹤**：乳癌術後每年一次乳攝是標準。確認病人有無按時回診；手術側（保乳手術後）和對側都要攝影。

**大腸直腸癌**

| 時機 | 常用影像 | 目的 |
|------|---------|------|
| 初診斷 | 胸腹骨盆 CT | 分期、肝轉移評估 |
| 直腸癌術前 | 骨盆 MRI | T/N 分期，決定是否術前 CCRT |
| 肝轉移評估 | 肝臟 MRI（多相位）| 確認轉移灶數目和可切除性 |
| 術後追蹤 | CT 每 6–12 個月 × 3–5 年 | 復發偵測 |

**肝癌**

| 時機 | 常用影像 | 目的 |
|------|---------|------|
| 篩檢 | 腹部超音波（每 6 個月）| 高危族群（慢性 B/C 肝）早期偵測 |
| 確診/分期 | 腹部 CT 多相位 / 肝臟 MRI | 動脈相快速強化 + 靜脈相廓清 = 診斷特徵 |
| 治療追蹤 | CT 或 MRI 每 2–3 個月 | 肝癌用 mRECIST 評估（不同於一般 RECIST）|

**頭頸癌**

| 時機 | 常用影像 | 目的 |
|------|---------|------|
| 初診斷/分期 | 頭頸部 MRI + 胸部 CT | 局部侵犯、頸部淋巴結、肺轉移 |
| PET-CT | 不明原發灶、放療後復發評估 | 全身評估 |
| 放療後追蹤 | MRI 每 3–6 個月 | 復發偵測（CT 因放療後纖維化判讀較難）|

**攝護腺癌**

| 時機 | 常用影像 | 目的 |
|------|---------|------|
| 初診斷/分期 | 骨盆 MRI（multiparametric MRI, mpMRI）| 局部範圍評估（T 分期）、PI-RADS 評分 |
| 高風險/局部晚期 | **PSMA PET-CT** | 高敏感度分期（淋巴結轉移、骨轉移）|
| 骨轉移評估 | 骨骼掃描 + 骨盆 CT | 傳統標準；PSMA PET-CT 敏感度更高 |
| 生化復發（PSA 升高）| **PSMA PET-CT** | 找出復發位置（骨骼掃描常在此時無法偵測）|
| 治療追蹤 | PSA 追蹤（主要）+ 必要時 CT/MRI | PSA 上升才考慮影像追蹤 |

!!! note "PSMA PET-CT 詳細說明"
    **PSMA（Prostate-Specific Membrane Antigen）**是攝護腺癌細胞高度表現的蛋白質，PSMA PET-CT 以放射性核素（如 ⁶⁸Ga-PSMA 或 ¹⁸F-PSMA）標記後全身掃描，比傳統骨骼掃描 + CT 更早偵測到轉移病灶。

    **主要優勢：**
    - 骨轉移偵測敏感度顯著優於傳統骨骼掃描
    - PSA ≥ 0.2 ng/mL 的生化復發即可偵測到復發位置
    - 一次全身掃描同時評估軟組織、淋巴結、骨骼

    **個管師追蹤重點：**
    - PSMA PET-CT 目前多為**自費**（部分情況有健保申請），費用約 3–5 萬元
    - 需在核醫科預約，通常需提前 1–2 週安排
    - 不適用於非攝護腺癌病人（PSMA 在其他癌別不特異）
    - 結果出來後協助安排泌尿科/腫瘤科門診討論後續治療

#### 影像報告關鍵詞個管師速查

| 報告用語 | 意思 | 個管師行動 |
|---------|------|-----------|
| **穩定（Stable / SD）** | 病灶大小無明顯變化（±20%）| 告知病人治療有效控制中 |
| **部分緩解（PR）** | 病灶縮小 ≥30% | 治療有效，繼續療程 |
| **完全緩解（CR）** | 影像上看不到病灶 | 討論是否維持治療或停藥觀察 |
| **疾病進展（PD）** | 病灶增大 ≥20% 或新病灶出現 | 需討論換線治療，協助安排多科討論 |
| **不確定（Equivocal）** | 無法確定是否為轉移 | 通常需 MRI 確認或短期追蹤 |
| **強化（Enhancement）** | 顯影劑注射後病灶變亮 | 可能是腫瘤血管豐富或炎症 |
| **淋巴結腫大（LAP）** | 短軸 > 1 cm 或不規則形 | 需評估是否為轉移 |

!!! tip "個管師重點"
    影像追蹤時間表是個管師追蹤的重要工作。新診斷個案需確認：① 分期影像已完成 ② 結果已和主治醫師討論 ③ 下一次追蹤影像的時間已預約。

!!! warning "顯影劑注意事項"
    CT 和 MRI 通常需要顯影劑。腎功能不佳（Cr > 1.5）或有顯影劑過敏史的病人，需在影像前告知放射科。個管師協調影像預約時應確認這兩點。

## F2. 癌症治療方式與副作用

> 這一節是給新人建立「整體概念地圖」用的。
> 讀完的目標：聽到「術後輔助 CCRT」知道在說什麼；打電話追蹤化療病人知道要問什麼。

---

### 一、六種治療方式總覽

| 治療方式 | 主要目的 | 作用原理 |
|---------|---------|---------|
| **手術** | 直接切除腫瘤 | 物理移除 |
| **化學治療** | 全身性殺死快速分裂的細胞 | 細胞毒殺 |
| **放射線治療** | 局部照射殺死癌細胞 | 破壞 DNA |
| **標靶治療** | 針對特定基因突變 | 阻斷特定訊號路徑 |
| **免疫治療** | 活化自身免疫系統 | 解除免疫抑制 |
| **荷爾蒙治療** | 阻斷荷爾蒙依賴性腫瘤生長 | 降低荷爾蒙刺激 |

---

### 二、手術治療

#### 手術目的

| 目的 | 說明 | 舉例 |
|------|------|------|
| **根治性手術** | 切除腫瘤，目標是治癒 | 早期肺癌肺葉切除、乳癌保乳手術 |
| **姑息性手術** | 減輕症狀，非以治癒為目標 | 腸阻塞造口術、骨轉移固定手術 |
| **診斷性手術** | 取組織做病理確診 | 淋巴結切片、開胸取樣 |

!!! tip "個管師要知道的"
    「姑息性手術」不等於放棄，而是改善生活品質。病人常誤以為手術就代表治癒，衛教時需主動說明手術目的。

#### 常見術式

| 術式 | 說明 |
|------|------|
| **開放性手術** | 傳統大切口，視野清楚，適合複雜病灶 |
| **腹腔鏡 / 胸腔鏡（微創）** | 小切口，恢復快，傷口疼痛輕 |
| **達文西機器手臂** | 微創進階版，精準度更高 |
| **內視鏡切除（EMR / ESD）** | 無切口，透過消化道切除早期黏膜癌 |

#### 手術邊緣（Margin）

| 名稱 | 意義 | 後續 |
|------|------|------|
| **R0** | 邊緣無殘存癌細胞（完全切除）| 最佳結果 |
| **R1** | 顯微鏡下見殘存癌細胞 | 可能需追加放療 |
| **R2** | 肉眼可見殘存腫瘤 | 手術不完整 |

!!! note "品質指標連結"
    R0 切除率是胃癌、食道癌、肝癌、胰臟癌的強制申報品質指標。

#### 個管師任務

| 時機 | 任務 |
|------|------|
| **術前** | 確認病人了解手術目的；確認術前評估（ECOG、心肺）完成；確認 MDT 討論過 |
| **術後 1–4 週** | 追蹤傷口、進食、活動；確認病理報告已與醫師討論；確認下一步計畫 |
| **長期** | 提醒回診和影像追蹤時程 |

---

### 三、化學治療

#### 化療目的

| 目的 | 名稱 | 說明 |
|------|------|------|
| 術後消滅微小殘存 | **輔助性化療（Adjuvant）** | 手術後給，降低復發率 |
| 術前縮小腫瘤 | **前導性化療（Neoadjuvant）** | 先化療再手術 |
| 搭配放療增效 | **同步化放療（CCRT）** | 化療讓腫瘤對放療更敏感 |
| 控制但無法治癒 | **姑息性化療** | 延緩進展、維持生活品質 |

#### 給藥概念

**Cycle（療程週期）**：化療是「打幾天、休息幾天」為一個 cycle。

| 週期 | 常見方案 |
|------|---------|
| **Q3W**（每 3 週）| Carboplatin + Paclitaxel |
| **Q2W**（每 2 週）| FOLFOX（大腸癌）|
| **Weekly**（每週）| 低劑量，副作用分散 |

**給藥途徑**

| 途徑 | 個管師注意 |
|------|-----------|
| **周邊靜脈** | 多次後血管易損傷，評估是否需置管 |
| **Port-A（人工血管）** | 確認功能正常、定期沖洗衛教 |
| **PICC** | 注意感染和移位，日常照護衛教 |
| **口服** | 副作用不易主動回報，追蹤電話需主動問 |

#### 常見化療藥物

| 分類 | 代表藥物 | 主要用於 | 特殊副作用 |
|------|---------|---------|-----------|
| **鉑金類** | Cisplatin、Carboplatin、Oxaliplatin | 肺、頭頸、大腸、卵巢 | 腎毒性（Cisplatin）、神經病變（Oxaliplatin）|
| **紫杉醇類** | Paclitaxel、Docetaxel | 乳癌、肺癌、卵巢 | 過敏反應、神經病變 |
| **氟嘧啶類** | 5-FU、Capecitabine、Ufur | 大腸癌、乳癌、胃癌 | 手足症候群（口服）、口腔黏膜炎 |
| **抗代謝劑** | Pemetrexed、Gemcitabine | 肺癌、胰臟癌 | 骨髓抑制 |
| **拓樸酶抑制劑** | Irinotecan、Etoposide | 大腸癌、肺癌 | 嚴重腹瀉（Irinotecan）|
| **烷化劑** | Cyclophosphamide | 乳癌、淋巴瘤 | 出血性膀胱炎（需多喝水）|

**常用口服化療**

| 藥物 | 常用癌別 | 注意事項 |
|------|---------|---------|
| Capecitabine（截瘤達）| 大腸癌、乳癌 | 飯後服用；注意手足症候群 |
| Ufur（愛福錠）| 大腸癌 | 服滿半年為完治標準之一 |
| Erlotinib（得舒緩）| 肺癌（EGFR+）| 空腹服用；注意皮疹 |

#### 化療副作用

!!! tip "不同副作用出現的時間不同，提前告知病人可減少焦慮和延誤就醫。"

| 副作用 | 出現時間 | 個管師追蹤重點 |
|--------|---------|--------------|
| **噁心嘔吐** | 化療當天至 3 天 | 止吐藥有沒有規律服？飲食狀況？ |
| **骨髓抑制** | 化療後 7–14 天（最低點）| 有沒有發燒？（> 38°C = 緊急）|
| **掉髮** | 化療後 2–4 週 | 提前衛教，心理支持 |
| **口腔黏膜炎** | 化療後 3–10 天 | 口腔清潔，嚴重影響進食時需介入 |
| **手足症候群** | 口服化療累積後 | 皮膚脫皮紅腫，影響日常活動 |
| **周邊神經病變** | 累積劑量後 | 手腳麻木，需告知醫師評估 |
| **腎毒性** | Cisplatin 相關 | 確認水分補充足夠；追蹤 Creatinine |
| **嚴重腹瀉** | Irinotecan 相關 | > 24 小時或 > 6 次/天需就醫 |

!!! warning "需立刻就醫的警示症狀"
    - 🚨 **發燒 > 38°C**（感染性嗜中性球低下，有生命危險，不能等到明天）
    - 嚴重嘔吐無法進食超過 24 小時
    - 嚴重腹瀉一天超過 6 次
    - 胸痛或突然喘不過氣

### 四、放射線治療

#### 放療目的

| 目的 | 說明 |
|------|------|
| **根治性放療** | 單獨以放療治癒，如早期鼻咽癌、攝護腺癌 |
| **術後輔助放療** | 手術後追加照射，降低局部復發 |
| **CCRT** | 化療讓腫瘤對放療更敏感，常用於頭頸、子宮頸 |
| **姑息性放療** | 緩解骨轉移疼痛、腦轉移壓迫症狀 |

#### 放療單位與分次

**Gy（戈瑞）**：放射劑量單位。100 cGy = 1 Gy。

| 分次方式 | 說明 | 典型例子 |
|---------|------|---------|
| **常規分次** | 每天 1 次，週一至週五，共 5–7 週 | 頭頸癌：70 Gy / 35 次 |
| **低分次** | 每次劑量較高，總次數少 | 攝護腺癌：60 Gy / 20 次 |
| **SBRT / SABR** | 極少次、極高劑量，精準照射 | 早期肺癌：54 Gy / 3 次 |
| **Brachytherapy** | 放射源放入體內近距離照射 | 子宮頸癌（品質指標要求）|

!!! note "品質指標連結"
    子宮頸癌放療需在 **63 天內完成**，是強制申報指標。個管師需追蹤放療開始和完成日期。

#### 放療技術

| 縮寫 | 說明 |
|------|------|
| **3D-CRT** | 三維適形放療，傳統高精準 |
| **IMRT** | 強度調控放療，各方向劑量可調，保護正常組織更好 |
| **IGRT** | 影像導引放療，每次治療前拍影像確認位置 |
| **SBRT / SABR** | 立體定位放療，少次高劑量，精準打小腫瘤 |

#### 放療副作用

**急性副作用（治療中至治療後 4–6 週）**

| 照射部位 | 常見副作用 | 個管師衛教重點 |
|---------|-----------|--------------|
| **頭頸部** | 口腔黏膜炎、吞嚥困難、皮膚紅腫、口乾 | 口腔清潔；軟食流質；皮膚保護 |
| **胸部** | 食道炎（吞嚥痛）、皮膚炎；放射性肺炎（數週後）| 飲食調整；出現新喘氣要回診 |
| **腹部/骨盆** | 腹瀉、噁心、頻尿、尿道不適 | 飲食清淡；保持水分；腹瀉 > 4 次/天要告知 |
| **全身** | 疲倦（幾乎所有病人）| 不需停治療；輕度規律活動反而有助減輕疲倦 |

**慢性副作用（治療結束數月至數年後）**

| 部位 | 慢性副作用 | 追蹤時主動問 |
|------|-----------|------------|
| **頭頸部** | 永久性口乾、蛀牙、顎骨壞死（ORN）| 「口腔有一直很乾嗎？最近有看牙科嗎？」|
| **胸部** | 肺纖維化、心臟毒性 | 有沒有新的喘氣或心悸？ |
| **骨盆** | 慢性放射性直腸炎、膀胱炎、淋巴水腫 | 有沒有血便、解尿困難、腿部腫脹？ |

#### 個管師任務

| 時機 | 任務 |
|------|------|
| **放療前** | 說明療程長度和副作用；飲食和皮膚照護衛教 |
| **放療中（每週追蹤）** | 確認副作用程度；是否影響進食體重；需不需要轉介營養師 |
| **放療完成後** | 告知「副作用還會持續 2–4 週才漸退，不是復發」；安排後續追蹤影像 |

### 五、標靶治療

#### 什麼是標靶治療

針對癌細胞**特定基因突變或蛋白質**設計的藥物，使用前需先做基因/分子檢測確認適用性。

#### 常見標靶藥物

| 藥物 | 靶點 | 主要癌別 | 常見副作用 |
|------|------|---------|-----------|
| **Trastuzumab（賀癌平）** | HER2 | 乳癌、胃癌 | 心臟毒性（追蹤心臟功能）|
| **Bevacizumab（癌思停）** | VEGF | 大腸癌、肺癌、卵巢癌 | 高血壓、蛋白尿、傷口癒合慢 |
| **Erlotinib（得舒緩）** | EGFR | 肺癌（EGFR 突變）| 皮疹、腹瀉 |
| **Osimertinib（泰格莎）** | EGFR（第三代）| 肺癌（T790M 突變）| 皮疹、腹瀉 |
| **Imatinib（基利克）** | BCR-ABL、c-KIT | CML、GIST | 水腫、噁心 |
| **Cetuximab** | EGFR | 大腸癌（RAS 野生型）、頭頸癌 | 痤瘡樣皮疹 |
| **Palbociclib / Ribociclib** | CDK4/6 | 乳癌（HR+，HER2-）| 骨髓抑制 |
| **Olaparib** | PARP | 乳癌（BRCA+）、卵巢癌 | 貧血、噁心 |

!!! tip "個管師追蹤重點"
    標靶藥物多為口服，病人在家服藥。追蹤時需主動問：皮疹狀況、血壓（Bevacizumab）、心悸（Trastuzumab）。

---

### 六、免疫治療

#### 免疫檢查點抑制劑（ICI）

癌細胞會釋放訊號「欺騙」免疫系統不攻擊它，ICI 藥物解除這種抑制，讓免疫系統恢復對癌細胞的攻擊。

| 藥物 | 靶點 | 常用癌別 |
|------|------|---------|
| **Pembrolizumab（吉舒達）** | PD-1 | 肺癌、乳癌、頭頸癌、大腸癌等多種 |
| **Nivolumab（保疾伏）** | PD-1 | 肺癌、食道癌、胃癌等 |
| **Atezolizumab** | PD-L1 | 肺癌、膀胱癌 |
| **Durvalumab** | PD-L1 | 肺癌（Stage III CCRT 後維持）|

#### 免疫相關不良反應（irAE）

!!! warning "irAE 是免疫治療最需要警覺的副作用"
    不同於化療，irAE 是**自身免疫反應**，可能發生在任何器官，出現時間不固定（數週到數個月後）。

| 器官 | 常見 irAE | 警示症狀 |
|------|----------|---------|
| **皮膚** | 皮疹、搔癢 | 嚴重起水泡、黏膜破損需緊急就醫 |
| **腸胃道** | 腹瀉、腸炎 | 一天超過 4–6 次或血便需回診 |
| **肝臟** | 肝功能上升 | 通常無症狀，需定期驗血 |
| **肺部** | 放射性肺炎 | 新的乾咳、喘氣需立即就醫 |
| **內分泌** | 甲狀腺炎、腎上腺功能不足 | 極度疲倦、低血壓、心跳加快 |
| **關節** | 關節炎 | 關節紅腫疼痛 |

### 七、荷爾蒙治療

#### 適用情況

| 癌別 | 條件 | 說明 |
|------|------|------|
| **乳癌** | ER/PR 陽性（荷爾蒙受體陽性）| 約 70–80% 的乳癌屬此類 |
| **攝護腺癌** | 幾乎所有攝護腺癌 | 睪固酮是主要驅動荷爾蒙 |

#### 常見藥物

**乳癌：**

| 藥物 | 適用 |
|------|------|
| **Tamoxifen** | 停經前後均可 |
| **Letrozole / Anastrozole / Exemestane**（芳香酶抑制劑）| 停經後 |
| **Fulvestrant** | 停經後、荷爾蒙治療失敗後 |

**攝護腺癌：**

| 藥物/方式 | 說明 |
|---------|------|
| **GnRH 類似物**（Leuprolide 等）| 化學性去勢，降低睪固酮 |
| **Bicalutamide / Enzalutamide** | 抗雄性素 |
| **Abiraterone** | 抑制雄性素合成 |

!!! tip "個管師追蹤重點"
    荷爾蒙治療療程長（乳癌通常 5–10 年），需追蹤服藥遵從性。不願繼續服藥的病人需轉介醫師討論，中斷荷爾蒙治療的乳癌病人需轉介緩和團隊。

---

### 八、治療組合方式

| 組合 | 說明 | 典型癌別 |
|------|------|---------|
| 手術 → 化療 | 術後輔助化療 | 大腸癌 II–III 期、胃癌 II–III 期 |
| 手術 → 放療 | 術後輔助放療 | 乳癌（淋巴陽性 ≥4 顆）|
| 化療 + 放療（同時）| CCRT | 子宮頸癌、頭頸癌、食道癌 |
| 化療 → 手術 | 前導性化療（先縮瘤）| 局部晚期乳癌、食道癌 |
| 化療 → 手術 → 化療 | 夾心療法 | 部分胰臟癌、胃癌方案 |
| 標靶 + 化療 | 增加療效 | HER2 陽性乳癌、肺癌 |
| 免疫 + 化療 | 增加免疫治療效果 | 肺癌、三陰性乳癌 |

### 九、個管師追蹤問題速查

**手術後（前 4 週）**
- 傷口有沒有紅腫或滲液？
- 飲食恢復狀況？體重有沒有明顯下降？
- 病理報告有沒有跟醫師討論過？
- 下一步治療計畫清楚了嗎？

**化療中**
- 有沒有發燒？（> 38°C 要立刻就醫，不能等到明天）
- 噁心嘔吐影響進食幾天？
- 手腳有沒有麻木或皮膚脫皮？
- 體力狀況如何？

**放療中**
- 吃東西有沒有困難或疼痛？（頭頸、食道）
- 皮膚有沒有脫皮或起水泡？
- 有沒有腹瀉？幾次？（腹部/骨盆照射）
- 疲倦程度？

**口服藥物（標靶 / 荷爾蒙 / 口服化療）**
- 有沒有每天按時服藥？
- 有沒有出現皮疹或皮膚變化？
- 有沒有腸胃不適？
- 有沒有不舒服但沒跟醫師說？

### 十、局部介入治療（RFA / TACE / HAIC）

> 這三種治療以肝癌為核心適應症，但也用於其他腫瘤的局部控制。由**介入放射科**或**肝膽外科**執行，個管師需了解流程、術後反應和追蹤重點。

---

#### RFA / MWA　消融治療

**原理**

| 術式 | 中文 | 原理 |
|------|------|------|
| **RFA** | 射頻消融 | 高頻電流產生熱能（70–100°C），直接燒灼腫瘤 |
| **MWA** | 微波消融 | 微波能量加熱（可達 100°C+），升溫更快、消融範圍更大 |

**適應症**

- 早期肝癌（BCLC Stage 0 / A）：腫瘤 ≤ 3 cm、數目 ≤ 3 個
- 肝轉移（大腸癌、神經內分泌瘤等）：病灶小且數目有限
- 肺癌、腎癌等局部小病灶

**執行方式**

超音波或 CT 導引下，經皮（percutaneous）將探針穿入腫瘤中心，通常不需全身麻醉，術後留觀 1–2 天。

**術後反應**

| 症狀 | 說明 |
|------|------|
| **消融後症候群（Post-ablation syndrome）** | 低燒、疲倦、右上腹不適，術後 1–5 天，多自行緩解 |
| **疼痛** | 穿刺部位或右上腹，通常輕至中度 |
| **肝功能一過性上升** | 常見，多在 1–2 週內恢復 |
| 嚴重出血、膽管損傷 | 少見，需緊急處理 |

**追蹤重點**

- 術後 **4–6 週** CT 或 MRI 評估消融是否完全（Complete Ablation）
- 完全消融 = 腫瘤周圍出現完整「安全邊界」（ablation margin）
- 不完全消融或鄰近殘存 → 評估補充消融或其他治療
- 肝癌質量指標：消融後 **3 個月內追蹤影像**（品質指標 2）

!!! tip "個管師追蹤重點"
    術後第 1 個月確認病人有無回診做影像評估。追蹤電話問：「有沒有發燒？右上腹有沒有持續疼痛加劇？」術後輕微發燒是正常，但持續超過 1 週高燒需排除感染。

---

#### TACE　經導管動脈化學栓塞

**原理**

肝癌由肝動脈供血（正常肝臟主要由門靜脈供血），利用導管將化療藥物和栓塞劑注入腫瘤的滋養動脈，達到「雙重殺傷」：局部高濃度化療 + 阻斷血流使腫瘤缺血壞死。

**兩種主要術式**

| 術式 | 說明 | 特點 |
|------|------|------|
| **cTACE（傳統型）** | Lipiodol + 化療藥（Doxorubicin 等）+ 栓塞顆粒 | 最廣泛使用 |
| **DEB-TACE（載藥微球）** | 預先載入化療藥的微球，緩慢釋放 | 局部藥物濃度更穩定，全身副作用較少 |

**適應症**

- **BCLC Stage B**（中期）肝癌：多發性病灶、不可切除
- 等待肝臟移植前的橋接治療（Bridge to transplant）
- 早期肝癌手術或消融前縮瘤

**術後反應（Post-embolization Syndrome，非常常見）**

!!! warning "栓塞後症候群——需提前告知病人"
    幾乎所有病人術後都會出現，**不代表治療失敗**，是正常的缺血壞死反應。個管師需提前衛教。

| 症狀 | 發生率 | 持續時間 |
|------|--------|---------|
| **發燒**（37.5–39°C）| 50–80% | 術後 1–7 天 |
| **右上腹疼痛**（肝區鈍痛）| 70–80% | 術後 1–5 天 |
| **噁心、嘔吐** | 40–50% | 術後 1–3 天 |
| **疲倦、食慾不振** | 多數 | 術後 1–2 週 |
| 肝功能暫時惡化 | 常見 | 2–4 週內多恢復 |

**嚴重併發症（需警覺）：**
- 肝膿瘍（Liver abscess）：術後發燒超過 1 週或體溫持續 > 38.5°C
- 肝衰竭（肝硬化基礎差的病人風險較高）
- 膽道損傷

**追蹤重點**

- 術後 **4–6 週** CT 或 MRI（肝癌用 mRECIST 評估）
- 評估腫瘤活性（動脈相強化部分 = 仍有活性）→ 決定是否再次 TACE
- 多數病人需要多次 TACE（視腫瘤反應決定間隔）
- 肝癌質量指標：TACE 後 **3 個月內追蹤影像**（品質指標 2、4）

!!! tip "個管師追蹤重點"
    術後追蹤電話（術後 3–5 天）：「有沒有發燒？發燒幾度？肚子痛有沒有越來越嚴重？」
    若發燒 > 38.5°C 持續超過 5 天，或腹痛加劇，需回診排除肝膿瘍。
    預約 4–6 週後的追蹤 CT/MRI 是個管師的核心任務。

---

#### HAIC　肝動脈灌注化療

**原理**

與 TACE 同樣利用肝動脈途徑，但 HAIC **不栓塞**，只灌注高濃度化療藥物。肝臟首過效應使腫瘤局部藥物濃度遠高於靜脈注射，全身副作用相對較低。

**常用方案**

| 方案 | 藥物 | 給藥方式 |
|------|------|---------|
| **FOLFOX-HAIC**（最常用）| Oxaliplatin + Leucovorin + 5-FU | 每 3–4 週，經導管持續灌注 2–3 天 |
| FOLFIRI-HAIC | Irinotecan + Leucovorin + 5-FU | 較少用 |

**適應症（進展期肝癌為主）**

- **伴有門靜脈腫瘤栓子（PVTT）** 的晚期肝癌（TACE 在此情況效果有限或禁忌）
- 大型多發性肝癌的降期治療（Down-staging），目標是讓不可切除變為可切除
- TACE 失敗或 TACE 療效不佳後的替代選擇
- 大腸癌等實體瘤以肝臟為主的轉移（部分情況）

**導管置入方式**

| 方式 | 說明 |
|------|------|
| **重複股動脈穿刺**（每次）| 每次治療前由介入放射科置管，不需長期導管 |
| **Port 植入**（皮下）| 手術植入皮下灌注壺（Reservoir），連接肝動脈導管，長期使用，不需每次重新穿刺 |

**副作用**

| 副作用 | 說明 |
|------|------|
| **腹痛、噁心** | 灌注中或灌注後，通常輕至中度 |
| **肝功能暫時上升** | AST/ALT 上升，多在 2–4 週內恢復 |
| **骨髓抑制** | 程度通常低於全身靜脈化療（FOLFOX-HAIC 比靜脈 FOLFOX 輕）|
| **周邊神經病變** | Oxaliplatin 相關，累積劑量後出現 |
| **導管相關感染**（Reservoir port）| 定期清洗與維護 |

**追蹤重點**

- 每次治療前確認 CBC 和肝功能（Oxaliplatin 腎毒性也需追蹤）
- 治療 **2–3 次後**（約 6–8 週）做 CT/MRI 評估腫瘤反應
- 有 Reservoir Port 的病人：未治療期間每 **4–6 週沖洗**（防止血栓阻塞）

!!! tip "個管師追蹤重點"
    HAIC 療程較長，每 3–4 週一次，每次住院 2–3 天。需協助病人做好治療行程規劃。
    若病人有植入 Reservoir Port，要提醒定期回介入放射科或護理師進行**導管沖洗**，否則導管可能阻塞。

---

#### RFA / TACE / HAIC 比較

| | RFA / MWA | TACE | HAIC |
|--|-----------|------|------|
| **主要機轉** | 熱燒灼破壞 | 局部化療 + 缺血壞死 | 局部高濃度化療 |
| **主要適應症** | 早期小腫瘤（≤3cm）| 中期多發性肝癌 | 晚期/PVTT/降期 |
| **住院天數** | 1–2 天 | 2–4 天 | 2–3 天（每療程）|
| **術後症候群** | 輕 | 明顯（發燒、疼痛）| 輕至中 |
| **追蹤影像時機** | 4–6 週後 | 4–6 週後 | 2–3 次療程後 |
| **健保給付** | 有（符合條件）| 有（符合條件）| 部分有，部分自費 |
