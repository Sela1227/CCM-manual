# C8. 胃癌照護指引

**彰濱秀傳癌症中心｜個管師指導書系列**

> 📌 本文件為胃癌專科訓練教材，供負責的個管師（A）參考。內容架構與 C1 肺癌、C2 乳癌一致。

> ⚠️ **準則引用**：NCCN Gastric Cancer 2025、台灣胃癌醫學會建議、健保署 2024/4 晚期胃癌一線免疫給付公告。**待醫師團隊確認版本與本院實作差異**。

## 一、疾病概述

### 流行病學

- 台灣每年新確診：約 **4,000 人**
- **約 30–40% 確診時已第 4 期**（症狀類似消化性潰瘍易誤判）
- 5 年存活率：**約 35.2%**（為十大癌症前 3 名最低）
- 男女比：男性高於女性
- 主要組織型：腺癌（> 95%）、其他少見類型（如 GIST、淋巴瘤、鱗狀細胞癌）

### 危險因子

| 危險因子 | 說明 |
|---------|------|
| **幽門桿菌（H. pylori）感染** | 最重要的可控因子 |
| **飲食習慣** | 醃漬、煙燻、高鹽食物 |
| **菸酒** | |
| **胃部疾病史** | 慢性萎縮性胃炎、胃部分切除術後、胃腺瘤性息肉 |
| **家族史** | 一等親有胃癌（特別 < 50 歲）|
| **遺傳症候群** | HDGC（CDH1 突變）、Lynch 症候群 |

### 常見症狀

- **早期症狀類似胃潰瘍 / 胃炎**——這是胃癌晚期診斷率高的主因
- 上腹悶脹、隱痛
- 食慾不振、體重減輕
- **黑便、解血便**（消化道出血）
- 噁心嘔吐、吞嚥困難
- 貧血、疲倦
- 晚期：腹水、腹部腫塊、淋巴結腫大（左鎖骨上淋巴結 = Virchow's node）

## 二、診斷與分期

### 診斷流程

```
症狀懷疑 / 健檢上消化道鋇劑或胃鏡發現
        ↓
    上消化道內視鏡 + 切片
        ↓
    病理確診 + HER2 / MSI / PD-L1 檢測
        ↓
    分期檢查（CT 胸/腹/骨盆、EUS、PET-CT）
        ↓
    營養與功能評估
        ↓
    MDT 討論治療計畫
```

### 必要檢查項目

| 檢查 | 目的 | 備註 |
|------|------|------|
| **上消化道內視鏡 + 切片** | 確診 | 必要 |
| **內視鏡超音波（EUS）** | T、N 分期 | 早期 / 局部進展期建議 |
| **CT 胸腹骨盆** | 分期、評估遠端轉移 | 必要 |
| **PET-CT** | 評估遠端轉移 | III 期以上建議 |
| **HER2 IHC / FISH** | **必做**——影響標靶藥物選擇 | 必要 |
| **MSI / MMR 檢測** | 評估免疫治療適合性 | 必要 |
| **PD-L1 CPS** | 影響免疫治療適合性 | 必要 |
| **CEA、CA19-9** | 腫瘤指數、追蹤 | 建議 |
| **幽門桿菌檢測** | 治療一併處理 | 必要 |
| **腹腔鏡**（局部進展期）| 排除腹膜轉移 | 部分病人 |

### 分期

**TNM 分期（AJCC 第 8 版）**

| 期別 | 大致說明 | 5 年存活率 |
|------|---------|----------|
| **I 期** | T1-2 + N0-1 | 60–80% |
| **II 期** | T1-4a + N0-3 不同組合 | 30–50% |
| **III 期** | T2-4b + N1-3b 不同組合 | 15–30% |
| **IV 期** | M1（遠端 / 腹膜轉移）| < 10% |

## 三、治療指引

> ⚠️ **依據準則**：NCCN Gastric Cancer 2025、台灣胃癌醫學會建議、健保署 2024/4 晚期胃癌一線免疫給付公告。**待醫師團隊確認版本與本院實作差異**。

#### 早期（I 期）

| 治療 | 適用情境 |
|------|---------|
| **內視鏡切除（EMR / ESD）** | T1a 限定條件 |
| **胃次全切除術 / 胃全切除術 + 淋巴結清除（D1+ / D2）** | T1b 以上局限性 |

#### 局部進展期（II-III 期）

- **新輔助化療**（FLOT 方案：5-FU + Leucovorin + Oxaliplatin + Docetaxel）+ 手術 + 術後輔助化療
- 或 手術 + 術後輔助化療（**TS-1 / Capecitabine + Oxaliplatin（XELOX）**）
- 健保給付 **TS-1（艾斯萬）**作為輔助化療

#### 晚期 / 轉移性（IV 期）

| 治療線別 | 藥物 |
|---------|------|
| **第一線（HER2+）** | **Trastuzumab + 化療**（健保給付，HER2 IHC 3+ 或 IHC 2+/FISH+）|
| **第一線（HER2-，PD-L1 CPS ≥ 5）** | **化療 + Nivolumab**（健保 2024/4 起給付）|
| **第一線（HER2-，PD-L1 CPS < 5）** | 化療為主（FOLFOX、CapeOx）|
| **第二線** | Ramucirumab ± Paclitaxel、TS-1、Irinotecan |
| **第三線** | Trifluridine/Tipiracil（TAS-102）、Nivolumab（部分病人）|
| **MSI-H** | Pembrolizumab |

## 四、本癌別特殊注意事項

| 重點 | 說明 |
|------|------|
| **HER2 / MSI / PD-L1 檢測** | 三種**全部必做**——影響整個治療策略 |
| **幽門桿菌處理** | 即使治療胃癌中，**幽門桿菌陽性者必根除**（降低殘胃癌風險）|
| **營養評估** | 胃癌術後營養問題嚴重，需早期介入 |
| **術後維生素 B12** | 胃全切除後**終身補充 B12**（胃內因子缺失）|
| **術後傾倒症候群** | 飲食調整、少量多餐 |
| **腹膜轉移評估** | 局部進展期建議腹腔鏡分期 |

## 六、個管照護重點

### 收案時

- 完成 PG-SGA、體重基準
- HER2 / MSI / PD-L1 報告確認
- 幽門桿菌狀態
- 心理困擾溫度計
- 飲食習慣評估（為術後調整鋪路）

### 治療期間

- **手術期**：
  - 術後早期下床、呼吸復健
  - 鼻胃管管路照護
  - 漸進飲食（清流質 → 流質 → 軟質）
  - 吻合口瘻管警示（術後 5-10 天）
  - 維生素 B12 衛教（胃全切除）
  - 傾倒症候群衛教
- **化療期**（FLOT、XELOX、TS-1）：
  - 周邊神經病變（Oxaliplatin 累積）
  - 手足症候群（Capecitabine、TS-1）
  - 骨髓抑制
- **標靶 / 免疫期**：HER2 標靶心毒性、Nivolumab irAE

### 追蹤期

- 術後第 1-2 年：每 3 個月（CEA + CT）
- 第 3 年：每 6 個月
- 第 4-5 年：每 6-12 個月
- 內視鏡：術後 6 個月、1 年、之後每年

## 七、常見問題與處理

### Q: 病人問「我胃都切了，以後吃什麼會飽？」

A: 胃全切除後**胃容量歸零**——食物從食道直接進小腸。建議：(1) **少量多餐**（每天 6-8 餐）(2) 細嚼慢嚼 (3) 高蛋白 + 高熱量 + 中油脂 (4) 餐後 30 分鐘內不喝湯水（減少傾倒症候群）(5) 終身補充維生素 B12。多數病人 6-12 個月適應。

### Q: 病人問「我吃完飯就頭暈想吐、心跳很快，是怎麼了？」

A: **傾倒症候群（Dumping syndrome）**——胃切除後食物太快進小腸引起。早期型（飯後 30 分鐘內）：暈眩、心跳快、出汗；晚期型（飯後 1-3 小時）：低血糖。處理：(1) 少量多餐 (2) 避免單醣（糖、果汁、甜食）(3) 餐間喝水、餐後不立刻喝湯 (4) 餐後平躺 30 分鐘。

### Q: 病人 HER2 是 IHC 2+，FISH 陽性，跟 HER2 IHC 3+ 一樣可以用標靶嗎？

A: 是的。HER2 陽性的判定有兩種方式：(1) **IHC 3+**（強表現）直接判陽性 (2) **IHC 2+**（中等表現）需做 **FISH 確認**——FISH 陽性才算 HER2+。兩者都可使用 Trastuzumab 標靶治療（健保給付）。

### Q: 病人問「我幽門桿菌陽性，吃藥會好嗎？要不要也治胃癌中的太太？」

A: 幽門桿菌可用三合一/四合一抗生素根除（療程 7-14 天，成功率 80-90%）。**胃癌病人即使治療中也要根除**——降低殘胃癌風險。**家屬無症狀不一定要篩檢**，但若有上腹痛或一等親胃癌史可考慮做。

### Q: 病人聽說「免疫治療」很好，可以用嗎？

A: 看條件：(1) **HER2-、PD-L1 CPS ≥ 5**：健保 2024/4 起給付化療 + Nivolumab（一線）(2) **MSI-H**：可使用 Pembrolizumab。HER2+ 病人優先用 Trastuzumab 標靶。是否適合需查 PD-L1 / MSI 報告由主治醫師判定。
