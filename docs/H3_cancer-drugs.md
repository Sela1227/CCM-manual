## 系統簡介

這是一套依據健保署**第九節抗癌瘤藥物**給付規定（115 年 4 月 23 日版）建立的速查工具，專為個管師日常工作設計。

### 核心功能

| 功能 | 說明 |
|------|------|
| **依癌別篩選** | 選擇癌別後，只顯示該癌別相關的藥物條件，不必在 24,000+ 字的長條文中尋找 |
| **關鍵字搜尋** | 輸入藥物名稱（中文或英文）即時篩選 |
| **藥物類型篩選** | 依化療、標靶、免疫、荷爾蒙分類篩選 |
| **條文展開** | 點擊藥物卡片展開詳細給付條件 |
| **更新提醒** | 系統偵測到健保署有新版時，自動顯示橘色警示 |

## 使用重點提示

!!! tip "查詢步驟建議"
    1. 先點選右上角**癌別篩選**，縮小到你負責的癌別
    2. 若知道藥物名稱，再用搜尋框快速定位
    3. 點擊藥物卡片展開查看詳細條件
    4. 如藥物同時適用多個癌別（跨癌藥物），切換到對應癌別會只顯示那個癌別的條件

!!! warning "使用注意"
    健保給付規定會不定期修訂。系統右上角若出現**橘色警示**，表示健保署已有更新版本尚未同步，請暫時以健保署官方版本為準，並通知癌症中心更新系統。

!!! note "9.69 免疫檢查點抑制劑"
    此類藥物（如 pembrolizumab、nivolumab、atezolizumab 等）條文最長，含大量 PD-L1 門檻表格。系統已手工整理為逐癌切片，選擇癌別後只顯示該癌別相關條件，大幅降低查閱難度。

## 藥物類型說明

個管師在協助病人了解治療方案時，可參考以下類型說明：

| 類型 | 說明 | 常見藥物 |
|------|------|---------|
| **化學治療** | 細胞毒殺性藥物，干擾癌細胞分裂 | Paclitaxel、Carboplatin、Gemcitabine |
| **標靶治療** | 針對特定基因突變或蛋白質的藥物 | Erlotinib、Trastuzumab、Imatinib |
| **免疫治療** | 活化免疫系統對抗癌細胞（含 ICI） | Pembrolizumab、Nivolumab、Atezolizumab |
| **荷爾蒙治療** | 抑制荷爾蒙依賴性癌症生長 | Tamoxifen、Letrozole、Bicalutamide |

## 系統更新說明

此系統會不定期配合健保署修訂版本更新。更新流程如下：

1. GitHub 系統每週自動偵測健保署網站是否有新版
2. 有新版時系統會自動發出通知，並準備新版 PDF
3. 個管師看到橘色警示後通知癌症中心
4. 癌症中心重建資料後推送新版

**若需手動查詢最新給付規定：**
前往健保署官方網站 → 醫療服務給付項目及支付標準 → 藥品給付規定 → 第九章 → 第九節
