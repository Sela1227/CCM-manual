# D3. 電話追蹤腳本

**彰濱秀傳癌症中心｜個管師指導書系列**

> 依情境分類的電話追蹤腳本（通用開場 / 初次收案 / 定期追蹤 / 術後 / 化療中 / 放療中 / 口服藥物 / 失聯）。

---


> 腳本是參考框架，不是逐字稿。依病人狀況靈活調整，重點是讓病人感覺被關心、有問題知道找誰。

---

### 通用開場與結尾

**開場（任何情境通用）**
```
「○○先生/小姐您好，我是彰濱秀傳癌症中心的個管師○○○，
想關心一下您最近的狀況，請問現在方便說話嗎？」
```

若不方便：
```
「沒關係，請問什麼時候比較方便？我到時候再打給您。」
→ 記錄預計回電時間
```

**通用結尾**
```
「謝謝您今天的時間。如果有任何不舒服或問題，
隨時可以打電話給我，我的分機是○○○（或手機 ○○○）。
祝您順利，我們○月○日門診見。」
```

### 初次收案電話

**時機：** 收案後 3 天內，確認已建立關係

```
「○○先生/小姐您好，我是彰濱秀傳癌症中心的個案管理師○○○，
  您的主治醫師○○醫師請我來關心您。
  請問您現在方便說話嗎？」

（確認可以說話後）

「我的工作是在您接受治療的過程中協助您——
  如果有任何關於檢查、治療或是需要哪些資源的問題，都可以聯絡我。
  
  我的直線電話是 ○○○-○○○○○，您方便的話記錄一下。
  
  想請問您，現在有沒有最困擾您的問題或擔心？」

（傾聽，依回答給予回應）

「我們會定期聯絡您。如果您有任何問題，不要客氣，直接打給我。
  祝您一切順利，再見！」
```

**記得問：**
- [ ] 確認聯絡電話是否正確
- [ ] 有無緊急聯絡人
- [ ] 目前最擔心的事（作為後續追蹤重點）

### 定期追蹤電話（治療穩定中）

**時機：** 每月或每兩週依計畫追蹤

```
「○○先生/小姐您好，我是○○○個管師，
  打電話來關心一下您最近的狀況，現在方便說話嗎？」

（確認可以說話）

「最近身體怎麼樣？」

── 依回答深入詢問：
  「有沒有哪裡不舒服？（疼痛、噁心、疲倦...）」
  「吃得好嗎？體重有沒有變化？」
  「睡得好嗎？心情怎麼樣？」
  「藥有按時吃嗎？有沒有什麼副作用？」

「下次回診是○月○日，○○醫師門診，您記得嗎？」

「如果有任何問題，隨時打給我，我的電話是○○○-○○○○○。
  保重，再見！」
```

### 術後追蹤電話（手術後 1–4 週）

**時機：** 手術後第 3–5 天、第 2 週、第 4 週各一次

```
「○○先生/小姐您好，我是○○○個管師，
  您上週剛做了手術，特別打電話來關心。現在方便說話嗎？」

【必問項目】

傷口：
  「傷口這幾天怎麼樣？有沒有紅腫、流膿或特別多的分泌物？」

疼痛：
  「疼痛程度 0 到 10 分怎麼樣？止痛藥夠用嗎？」

飲食：
  「可以吃東西了嗎？吃了什麼？體重有沒有很快下降？」

活動：
  「可以下床走動嗎？有沒有憋氣走路或腿部腫脹？」（警覺肺栓塞）

病理報告：
  「有沒有安排回診和醫師討論報告的時間？」（術後約 2 週回診）

【警示 → 立即回診】
  • 傷口紅腫熱痛加劇或流膿
  • 發燒 > 38°C
  • 突發胸痛或呼吸困難
  • 嚴重腹脹、無法排氣排便（腸阻塞）
```

### 化療中追蹤電話

**時機：** 每個 Cycle 治療後第 3–5 天（副作用高峰期）、治療後第 7–14 天（血球低點）

```
「○○先生/小姐您好，我是○○○個管師。
  您上週做了化療，特別打電話問一下您最近的狀況。現在方便嗎？」

【Cycle 後 3–5 天（副作用峰值）】

噁心嘔吐：
  「噁心嘔吐怎麼樣？有沒有吃止吐藥？一天吐幾次？吃得下東西嗎？」
  → 一天吐 3 次以上或完全無法進食超過 24 小時 → 請回診

口腔：
  「口腔裡面有沒有破？吃東西會痛嗎？」

腹瀉：
  「有沒有拉肚子？一天幾次？」
  → 一天超過 4–6 次 → 請回診

疲倦：
  「體力怎麼樣？有沒有很嚴重的疲倦影響日常活動？」

【治療後第 7–14 天（血球低點）】

發燒：
  「這幾天有沒有量體溫？有沒有發燒超過 38 度？」
  → 發燒 > 38°C → 立即就醫（不能等到明天）

感染警示：
  「有沒有喉嚨痛、傷口不舒服、或身上有什麼感染的跡象？」

出血：
  「有沒有不明原因的瘀青擴散、刷牙出血很多、或身上哪裡出血？」

!!! warning "發燒提醒"
    「特別提醒您，化療期間如果發燒超過 38 度，不管是幾點，
    都要打急診電話或直接去急診，不能等到隔天門診，
    這可能是感染，需要馬上處理。」
```

### 放療中追蹤電話

**時機：** 每週一次（整個放療療程）；放療結束後 2 週再追蹤一次

```
「○○先生/小姐您好，我是○○○個管師，
  您這週有去做放療，特別打電話關心一下。現在方便嗎？」

【依照射部位調整問題】

頭頸部放療：
  「口腔裡面有沒有破？吃東西會不會痛或吞嚥困難？」
  「皮膚有沒有紅腫、脫皮？有沒有照著皮膚護理的方法？」
  「有沒有口乾的問題？喝水量夠嗎？」
  → 吞嚥困難導致無法進食 → 安排營養師評估

胸部放療：
  「吞嚥有沒有不舒服？（食道炎）」
  「有沒有新出現的咳嗽或喘氣？」（放射性肺炎警示）
  「皮膚有沒有紅腫或脫皮？」

腹部/骨盆放療：
  「有沒有腹瀉？一天幾次？」
  「有沒有頻尿、解尿不適？」
  「有沒有噁心？」

全部都要問：
  「這幾天疲倦程度怎麼樣？影響到日常活動嗎？」

【放療完成後的追蹤】
  「放療結束後副作用可能還會持續 2–4 週，
  不要擔心，這是正常的，不代表病情惡化。
  我們下個月會安排追蹤影像，我會跟您聯繫預約時間。」
```

### 口服藥物追蹤電話（標靶 / 荷爾蒙 / 口服化療）

**時機：** 開始服藥後第 2 週、第 1 個月，之後每 1–2 個月

```
「○○先生/小姐您好，我是○○○個管師，
  您最近開始吃新的藥，打電話確認一下服藥狀況，現在方便嗎？」

服藥遵從性：
  「藥有沒有每天按時吃？有沒有漏吃過？」
  「請問您都是什麼時間吃藥？飯前還是飯後？」（確認服藥方式正確）

依藥物種類問副作用：

──標靶藥物（EGFR、ALK 等）：
  「皮膚有沒有出現紅疹或像青春痘的東西？」
  「有沒有拉肚子？一天幾次？」
  「有沒有吃東西不舒服、噁心或口腔破？」

──荷爾蒙治療（AI / Tamoxifen）：
  「有沒有關節痠痛？會影響走路嗎？」
  「有沒有熱潮紅的問題？」
  「體重有沒有明顯變化？」

──口服化療（Capecitabine / Ufur）：
  「手腳有沒有脫皮、紅腫或疼痛？（手足症候群）」
  「有沒有腹瀉？一天幾次？」

──免疫治療（PD-1/PD-L1 抑制劑）：
  「有沒有新出現的皮疹？」
  「有沒有新出現的喘氣或咳嗽？」（肺炎警示）
  「有沒有腹瀉？」
  「有沒有很嚴重的疲倦或心悸？」

「如果有任何副作用讓您擔心，或覺得影響到生活，
  一定要告訴我，我們可以跟醫師討論是否要調整藥物。」
```

### 失聯電話（首次嘗試）

**時機：** 預定追蹤未接、或預定回診未到

```
【接通時】
「○○先生/小姐您好，我是彰濱秀傳癌症中心的個管師○○○，
  我們○月○日有安排您的追蹤電話/門診，
  想確認您最近的狀況，現在方便說話嗎？」

（如果病人說有事沒辦法回診）
「沒關係，我們可以幫您重新預約，您覺得什麼時間方便？」

【未接通時 → 留言給家人】
（假設是家屬接到）
「您好，我是彰濱秀傳癌症中心的個管師○○○，
  請問這是○○先生/小姐的家人嗎？
  我想聯繫○○先生/小姐關心一下他/她的狀況，
  麻煩請告訴他/她，請他/她方便時回電給我，
  我的電話是 ○○○-○○○○○，謝謝。」

【留語音信箱】
「您好，我是彰濱秀傳癌症中心的個管師○○○，
  打電話來關心您的狀況。如果方便，請您回電給我，
  電話是 ○○○-○○○○○，謝謝您，保重。」
```

### 失聯電話（多次嘗試未果）

**時機：** 第 2–3 次嘗試仍無法聯繫

```
【第 2–3 次仍未接通】

→ 嘗試不同時段（早上 / 下午 / 傍晚）
→ 嘗試緊急聯絡人電話

【聯繫到緊急聯絡人】
「您好，我是彰濱秀傳癌症中心的個管師○○○，
  請問您是○○先生/小姐的緊急聯絡人嗎？
  我們最近一直聯繫不到他/她，想確認一下他/她是否平安，
  請問他/她最近狀況怎麼樣？
  如果方便，麻煩請他/她跟我們聯絡，電話是○○○-○○○○○。」

【三次均未果 → 準備掛號信（見 D4）】

追蹤紀錄要寫：
  • 每次嘗試的日期、時間、結果
  • 接到的人說了什麼
  • 下一步行動
```
