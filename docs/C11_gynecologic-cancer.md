# C11. 婦癌照護指引

**彰濱秀傳癌症中心｜個管師指導書系列**

> 📌 本文件為婦癌（子宮頸癌、子宮體癌、卵巢癌）專科訓練教材，供負責的個管師（A）參考。內容架構與 C1 肺癌、C2 乳癌一致。

> ⚠️ **準則引用**：NCCN Cervical / Uterine / Ovarian Cancer 2025、台灣婦癌醫學會、FIGO 分期建議。**待醫師團隊確認版本與本院實作差異**。

## 一、疾病概述

### 流行病學

**台灣三大婦癌新確診人數（依國健署 108 年資料）**：
- **子宮體癌**：約 3,124 人（首位、上升最快）
- 子宮頸癌：約 1,393 人
- 卵巢癌：約 1,677 人

**子宮頸癌**
- 平均診斷年齡：50–60 歲
- 與 HPV 感染高度相關
- HPV 疫苗 + 抹片篩檢可有效預防

**子宮體癌（含子宮內膜癌、子宮肉瘤）**
- 子宮內膜癌佔 90%、平均 56 歲（10–15% < 40 歲）
- 子宮肉瘤約 10%

**卵巢癌**
- 平均 50–60 歲
- 「沉默殺手」——早期幾乎無症狀、約 70% 確診時已晚期

### 危險因子

**子宮頸癌**

| 危險因子 | 說明 |
|---------|------|
| **HPV 感染**（特別是 16/18 型）| 必要因子 |
| 多重性伴侶 | |
| 早期性行為 | |
| 抽菸 | |
| 免疫低下（HIV、器官移植）| |

**子宮體癌**

| 危險因子 | 說明 |
|---------|------|
| **未生育 / 少生育** | 雌激素長期刺激 |
| **肥胖** | 脂肪組織轉化雌激素 |
| **多囊性卵巢症候群** | 排卵失調 |
| **糖尿病、高血壓** | 代謝症候群 |
| **泰莫西芬（Tamoxifen）使用** | 乳癌治療常見藥 |
| **Lynch 症候群** | 遺傳性 |
| **未對抗性雌激素治療** | 無黃體素拮抗 |

**卵巢癌**

| 危險因子 | 說明 |
|---------|------|
| **BRCA1/2 突變** | 遺傳性，風險 30-60% |
| **Lynch 症候群** | 遺傳性 |
| **未生育、晚停經** | 排卵次數多 |
| **家族史** | 一等親有卵巢/乳癌 |
| **子宮內膜異位症** | 部分組織型 |

### 常見症狀

**子宮頸癌**
- 早期常無症狀（靠抹片篩檢）
- **不正常出血**（性行為後、停經後）
- 異常分泌物（量多、有血絲）
- 晚期：骨盆痛、下肢水腫、血尿/血便

**子宮體癌**
- **異常陰道出血**——90% 病人有（停經後出血或經期紊亂）
- 不正常分泌物
- 骨盆痛（晚期）

> 💡 **停經後出血**：即使一滴都要當婦癌處理。

**卵巢癌（沉默殺手）**
- 早期幾乎無特異性症狀
- 腹脹、腹圍增大、消化不良
- 早飽、食慾不振、體重減輕
- 排尿習慣改變
- 晚期：腹水、骨盆腫塊、噁心嘔吐

## 二、診斷與分期

### 診斷流程

```
子宮頸癌：
抹片異常 / 症狀懷疑
        ↓
    陰道鏡 + 切片
        ↓
    病理確診
        ↓
    分期檢查（MRI、CT、PET-CT）
        ↓
    MDT

子宮體癌：
停經後出血 / 月經紊亂
        ↓
    婦科超音波（評估子宮內膜厚度）
        ↓
    子宮內膜採樣 / D&C
        ↓
    病理確診（90% 為內膜腺癌）
        ↓
    完整分期手術（子宮 + 雙側卵巢輸卵管 + 淋巴結）
        ↓
    依分期決定後續治療

卵巢癌：
症狀 / 影像 / CA-125 異常
        ↓
    婦科超音波 + CA-125
        ↓
    CT / MRI 評估
        ↓
    手術探查 + 病理（多數確診是手術中）
        ↓
    分期 + 減積手術 + 後續化療
```

### 必要檢查項目

**子宮頸癌**
| 檢查 | 目的 |
|------|------|
| 子宮頸抹片 | 篩檢、初步異常 |
| HPV DNA 檢測 | 高風險型 |
| 陰道鏡 + 切片 | 確診 |
| 骨盆 MRI | 局部分期 |
| 胸/腹/骨盆 CT 或 PET-CT | 遠端評估 |
| SCC、CEA | 腫瘤指數 |

**子宮體癌**
| 檢查 | 目的 |
|------|------|
| 婦科超音波 | 內膜厚度評估（停經後 > 4 mm 警示）|
| 子宮內膜採樣 / D&C | 病理確診 |
| 骨盆 MRI | 肌層侵犯深度 |
| 胸/腹/骨盆 CT | 遠端評估 |
| CA-125、CA19-9 | 部分輔助 |

**卵巢癌**
| 檢查 | 目的 |
|------|------|
| 婦科超音波 | 卵巢腫塊評估 |
| CA-125、HE4 | 腫瘤指數（停經前 CA-125 解讀注意）|
| CT / MRI 腹骨盆 | 評估、分期 |
| BRCA1/2 基因檢測 | 必做（影響 PARP 抑制劑使用 + 家族遺傳）|

### 分期

**子宮頸癌（FIGO 2018）**
| 期別 | 大致說明 | 5 年存活率 |
|------|---------|----------|
| I 期 | 局限子宮頸 | 80–90% |
| II 期 | 超出子宮頸但未及骨盆壁 | 60–70% |
| III 期 | 及骨盆壁 / 下 1/3 陰道 / 腎水腫 | 30–40% |
| IV 期 | 侵犯膀胱 / 直腸 / 遠端 | < 20% |

**子宮體癌（FIGO 2009/2023）**
| 期別 | 大致說明 | 5 年存活率 |
|------|---------|----------|
| I 期 | 限子宮 | 80–90%（早期手術即治癒）|
| II 期 | 侵犯子宮頸 | 70% |
| III 期 | 局部擴散（漿膜 / 卵巢 / 陰道 / 淋巴）| 50% |
| IV 期 | 侵犯膀胱 / 腸 / 遠端 | < 20% |

**卵巢癌（FIGO）**
| 期別 | 大致說明 | 5 年存活率 |
|------|---------|----------|
| I 期 | 限卵巢 | ~90% |
| II 期 | 限骨盆 | ~70% |
| III 期 | 腹膜擴散 / 後腹膜淋巴 | ~30–40% |
| IV 期 | 遠端轉移 | ~15% |

## 三、治療指引

> ⚠️ **依據準則**：NCCN Cervical / Uterine / Ovarian Cancer 2025、台灣婦癌醫學會、FIGO 分期建議。**待醫師團隊確認版本與本院實作差異**。

### 子宮頸癌

| 期別 | 治療 |
|------|------|
| **IA1 期** | 圓錐切片 / 子宮全切除（依生育意願）|
| **IA2-IB1 期** | 根除性子宮切除 + 骨盆淋巴結廓清 |
| **IB2-IVA 期** | **CCRT（同步化放療）**+ 體腔近接放療（標準）|
| **IVB 期** | 化療（Cisplatin + Paclitaxel ± Bevacizumab）+ Pembrolizumab（PD-L1+）|

### 子宮體癌

| 期別 | 治療 |
|------|------|
| **I 期低風險** | 全子宮 + 雙側卵巢輸卵管切除（手術即可治癒 80%）|
| **I 期高風險、II 期** | 手術 + 術後放療（陰道近接 / 體外）± 化療 |
| **III-IV 期** | 手術減積 + 化療（Carboplatin + Paclitaxel）+ 放療 |
| **MSI-H / dMMR** | 免疫治療（Pembrolizumab + Lenvatinib，第二線）|

### 卵巢癌

| 期別 | 治療 |
|------|------|
| **早期（I 期）** | 完整分期手術（必要時保留生育功能）+ 高風險加化療 |
| **進展期（II-IV 期）** | **減積手術 + 化療**（Carboplatin + Paclitaxel × 6 cycles）|
| **BRCA+ / HRD+** | **PARP 抑制劑維持治療**（Olaparib、Niraparib）|
| **復發** | 鉑類敏感 / 抗藥分流，依間隔期決定 |

#### 卵巢癌特殊：腹腔內化療（HIPEC）

部分中心對 III 期病人在減積手術後使用 HIPEC（腹腔熱化療），可延長存活。

## 四、本癌別特殊注意事項

| 重點 | 說明 |
|------|------|
| **HPV 疫苗推廣** | 子宮頸癌個案家屬衛教 |
| **生育意願溝通** | 早期病人手術前討論保留生育功能 |
| **CCRT 副作用**（子宮頸癌）| 直腸炎、膀胱炎、皮膚反應、骨髓抑制 |
| **CA-125 解讀**（卵巢癌）| 停經前可能假性升高（內膜異位、月經）|
| **BRCA 基因檢測**（卵巢癌）| 影響 PARP 抑制劑使用 + 家屬遺傳諮詢 |
| **泰莫西芬 + 子宮內膜癌風險** | 乳癌病人服用泰莫西芬需注意異常出血（個管師 A 同時負責乳癌+婦癌時要記住）|
| **腹水管理**（卵巢癌晚期）| 腹腔穿刺、營養支持 |
| **更年期 / 性功能** | 雙側卵巢切除後、需衛教荷爾蒙缺失症狀 |

## 六、個管照護重點

### 收案時

- 確認分期與治療計畫
- 生育意願評估（年輕病人）
- 婦癌家族史 / BRCA 狀態（卵巢癌）
- 心理困擾（女性器官切除衝擊大）
- 性功能、性生活衛教時機

### 治療期間

**子宮頸癌**
- **CCRT 期**：直腸炎、膀胱炎、皮膚反應、骨髓抑制（Cisplatin）
- **近接放療期**：婦科訪視照護
- **手術期**：尿失禁、淋巴水腫預防

**子宮體癌**
- **手術期**：腸胃功能恢復、漸進飲食
- **更年期症狀**：雙側卵巢切除後（潮紅、骨質流失）
- **泰莫西芬病人**異常出血追蹤

**卵巢癌**
- **手術期**：腸胃功能、傷口、腹水管理
- **化療期**：Carboplatin + Paclitaxel 副作用（神經病變、過敏、骨髓抑制）
- **PARP 抑制劑**：噁心、疲倦、貧血
- **腹水管理**：腹圍監測、必要時抽腹水

### 追蹤期

- 第 1-2 年：每 3 個月
- 第 3-5 年：每 6 個月
- 5 年後：每年
- 卵巢癌注意 CA-125 動態變化

## 七、常見問題與處理

### Q: 病人停經後出血擔心，問「會是癌嗎？」

A: **停經後出血一律當作婦癌處理直到證實不是**——子宮體癌 90% 病人有此症狀。即使最後是良性（萎縮性陰道炎、瘜肉），也要婦科超音波 + 內膜採樣排除。「一滴都不能輕忽」。

### Q: 卵巢癌病人問「我為什麼晚期才發現？」

A: 卵巢癌叫「**沉默殺手**」——早期幾乎無特異性症狀（腹脹、消化不良容易誤為腸胃問題），加上沒有有效篩檢方法（CA-125 + 超音波對早期敏感度低）。**不是延誤就醫的問題**，是疾病本身特性。重點放在治療上。

### Q: 年輕子宮頸癌病人問「我以後還想生小孩，可以保留子宮嗎？」

A: 看分期：(1) **IA1 期**（顯微浸潤）：圓錐切片可保留 (2) **IA2-IB1 期（< 2 cm）**：可考慮**根除性子宮頸切除術（trachelectomy）**保留子宮、術後可懷孕（高風險妊娠）(3) **IB2 以上**：通常無法保留生育。需在治療前充分討論、轉介生殖醫學。

### Q: 卵巢癌病人問「我女兒會不會也得卵巢癌？」

A: 看 BRCA 檢測：(1) **BRCA1 突變**：女兒終身卵巢癌風險 30-45%、乳癌 60-70% (2) **BRCA2 突變**：卵巢癌 10-20%、乳癌 45-55% (3) BRCA 陰性：女兒風險接近一般族群但仍略高（家族史）。**建議女兒做 BRCA 檢測 + 遺傳諮詢**——若陽性可早期密集追蹤或預防性手術。

### Q: 子宮體癌病人問「我兩邊卵巢都拿了，會變成更年期嗎？」

A: 是的——**手術性更年期**，症狀比自然更年期更突然強烈：(1) 熱潮紅、盜汗 (2) 失眠、情緒波動 (3) 性慾減退、陰道乾澀 (4) 骨質流失加速。處理：(1) 評估荷爾蒙補充（HRT）的利弊（子宮體癌病人多不適合）(2) 非荷爾蒙藥物（venlafaxine 等）改善熱潮紅 (3) 鈣 + 維生素 D 預防骨鬆 (4) 規律運動。
