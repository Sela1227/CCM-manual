# C9. 肝膽胰癌照護指引（膽道癌、胰臟癌）

## 一、疾病概述

### 流行病學

**膽道癌（含膽管癌、膽囊癌、壺腹癌）**
- 台灣每年新確診：約 2,000–2,500 人
- 平均診斷年齡：65–75 歲
- 預後差，**5 年存活率 < 20%**

**胰臟癌**
- 台灣每年新確診：約 3,000 人，**死亡率高居癌症第 7 位**
- **5 年存活率僅 8–10%**——癌中之王
- **約 80% 確診時已不適合手術**
- 平均診斷年齡：65–75 歲

### 危險因子

**膽道癌**

| 危險因子 | 說明 |
|---------|------|
| **膽結石**（膽囊癌主要因子）| 長期慢性發炎 |
| **膽道發炎**（PSC）| 原發性硬化性膽管炎 |
| **肝吸蟲感染** | 東南亞、台灣早期農村 |
| **B/C 肝炎、肝硬化** | 肝內膽管癌 |
| **化學物質暴露** | 二氯丙烷、二氯甲烷 |

**胰臟癌**

| 危險因子 | 說明 |
|---------|------|
| **吸菸** | 最重要可控因子 |
| **慢性胰臟炎** | |
| **糖尿病**（特別是新診斷）| 部分為胰臟癌前兆 |
| **家族史** | 一等親有胰臟癌 |
| **遺傳症候群** | BRCA1/2、Lynch、家族性胰臟癌 |
| **肥胖、高脂飲食** | |

### 常見症狀

**膽道癌**
- **黃疸（無痛性）**——膽管阻塞最常見
- 上腹痛（晚期）
- 體重減輕、食慾不振
- 灰白色大便、深色尿液
- 皮膚搔癢

**胰臟癌**
- **早期幾乎無症狀**——這是胰臟癌晚期診斷率高的主因
- **無痛性黃疸**（胰臟頭部腫瘤壓迫膽管）
- 上腹/背部放射性疼痛（胰臟體尾部腫瘤）
- 體重減輕、新發糖尿病
- 食慾不振、噁心嘔吐
- **Trousseau's sign**：莫名其妙的血栓（與胰臟癌關聯）

> ⚠️ 「無痛性黃疸 + 體重減輕 + 60 歲以上」永遠要懷疑胰臟癌或膽道癌。

## 二、診斷與分期

### 診斷流程

```
症狀懷疑（黃疸 / 上腹痛）
        ↓
    腹部超音波 + 血液檢查（肝功能、CA19-9）
        ↓
    動態 CT（胰臟協定）/ MRCP / EUS-FNA
        ↓
    病理確診（部分病人切片困難、手術中確診）
        ↓
    分期檢查（PET-CT、診斷性腹腔鏡）
        ↓
    MDT 討論：可切除 / 邊緣性 / 局部進展性 / 轉移性
```

### 必要檢查項目

| 檢查 | 目的 | 備註 |
|------|------|------|
| **動態 CT（胰臟協定）** | 評估腫瘤、血管侵犯 | 必要 |
| **MRCP（磁振膽胰造影）** | 評估膽管系統 | 膽道癌建議 |
| **EUS-FNA（內視鏡超音波導引切片）** | 病理確診、評估局部 | 胰臟癌建議 |
| **ERCP（內視鏡逆行性膽胰管造影）** | 同時診斷 + 治療（放支架）| 阻塞性黃疸 |
| **CA19-9** | 腫瘤指數、追蹤 | 必要（注意黃疸時假性升高）|
| **CEA** | 輔助腫瘤指數 | 建議 |
| **基因檢測**（BRCA、KRAS、MSI）| 影響標靶選擇 | 晚期建議 |
| **診斷性腹腔鏡** | 排除腹膜轉移 | 局部進展期建議 |

### 分期

**膽道癌 TNM 分期（AJCC 第 8 版）**：依部位分（膽囊、肝外膽管、肝內膽管、壺腹）有不同分期。

**胰臟癌 TNM 分期（AJCC 第 8 版）**：

| 期別 | 大致說明 | 治療策略 |
|------|---------|---------|
| **I 期** | 局限胰臟，≤ 4 cm | 可切除 |
| **II 期** | 局部淋巴結 | 可切除 / 邊緣性 |
| **III 期** | 主要動脈侵犯 | 多為局部進展期 |
| **IV 期** | 遠端轉移 | 緩和性治療 |

**胰臟癌可切除性分類**（NCCN）：
- 可切除（Resectable）
- 邊緣性可切除（Borderline resectable）
- 局部進展性（Locally advanced）
- 轉移性（Metastatic）

## 三、治療指引

> ⚠️ **依據準則**：NCCN Hepatobiliary Cancers 2025、NCCN Pancreatic Adenocarcinoma 2025、台灣肝臟醫學會、台灣胰臟醫學會建議。**待醫師團隊確認版本與本院實作差異**。

### 膽道癌

#### 可切除

- **手術切除**為唯一根治機會
- 膽囊癌：膽囊切除 ± 肝部分切除 + 淋巴結廓清
- 肝外膽管癌：依位置（Whipple、肝切除等）
- 肝內膽管癌：肝切除 ± 淋巴結廓清
- **術後輔助化療**：Capecitabine（BILCAP 試驗）

#### 不可切除 / 轉移性

- **第一線**：Gemcitabine + Cisplatin（GemCis）± **Durvalumab**（免疫，新標準）
- **第二線**：FOLFOX、FOLFIRI、IDH1 抑制劑（適合 IDH1 突變）

### 胰臟癌

#### 可切除（約 20%）

- **手術切除**（Whipple、遠端胰臟切除術等）+ **術後輔助化療**
- 輔助化療首選：**mFOLFIRINOX**（年輕體能好者）或 Gemcitabine + Capecitabine

#### 邊緣性可切除 / 局部進展性

- **新輔助化療**（FOLFIRINOX 或 Gemcitabine + nab-Paclitaxel）→ 評估手術可能性
- 部分加 CCRT

#### 轉移性

| 治療線別 | 藥物 |
|---------|------|
| **第一線（體能好）** | **mFOLFIRINOX** 或 Gemcitabine + nab-Paclitaxel |
| **第一線（體能差）** | Gemcitabine 單獨 |
| **第二線** | Nal-IRI + 5-FU/Leucovorin、其他化療 |
| **BRCA1/2 突變** | Olaparib 維持治療 |
| **MSI-H（罕見）** | Pembrolizumab |

## 四、本癌別特殊注意事項

| 重點 | 說明 |
|------|------|
| **CA19-9 解讀** | 黃疸時假性升高、Lewis 抗原陰性者不分泌（5-10% 病人）|
| **黃疸處理** | ERCP 放支架減黃，避免術前膽汁鬱積 |
| **胰島素管理** | Whipple 後可能糖尿病、血糖監測 |
| **胰酶補充** | 胰臟切除後消化酶不足、需補充胰酶（如 Creon）|
| **疼痛管理** | 胰臟癌後背痛劇烈、考慮**腹腔神經叢阻斷術** |
| **栓塞警示** | 胰臟癌深部靜脈栓塞風險高（Trousseau's sign）|
| **預後溝通** | 預後差、需早期討論安寧 / DNR |

## 六、個管照護重點

### 收案時

- 黃疸狀態、減黃計畫（ERCP 預約）
- CA19-9 基準值（注意黃疸假性升高）
- 體重 / PG-SGA、營養介入
- 預期餘命溝通、DNR 討論時機評估
- 心理支持（胰臟癌衝擊極大）

### 治療期間

- **手術期**（Whipple）：
  - 術後 7-14 天住院
  - 胰瘻管警示（術後 5-10 天，最常見併發症）
  - 胃排空延遲
  - 漸進飲食、胰酶補充衛教
  - 血糖監測
- **化療期**（FOLFIRINOX）：
  - 嚴重副作用（噁心、腹瀉、神經病變、骨髓抑制）
  - 體能要求高（不適合 ECOG ≥ 2）
  - 個管師密切追蹤
- **CCRT 期**（局部進展期）：副作用管理
- **緩和期**：
  - 疼痛控制（強烈鴉片類藥物常需要）
  - 腹腔神經叢阻斷術評估
  - 安寧轉介

### 追蹤期

- 術後第 1-2 年：每 3 個月（CT + CA19-9）
- 第 3 年：每 6 個月
- 第 4-5 年：每 6-12 個月
- 5 年後：每年
- 復發者轉緩和路徑

## 七、常見問題與處理

### Q: 病人問「我皮膚為什麼變黃？」

A: 黃疸代表膽汁排不出去——多為**腫瘤壓迫膽管**或膽管本身阻塞。處理：(1) 抽血看肝功能、總膽紅素 (2) 影像評估阻塞位置 (3) **ERCP 放支架**減黃（通常 1-2 週改善）(4) 嚴重者考慮經皮穿肝引流（PTCD）。減黃是後續治療的前置步驟。

### Q: 胰臟癌病人問「為什麼後背這麼痛？」

A: 胰臟癌的**後背痛**是腫瘤侵犯**腹腔神經叢**的典型症狀。處理：(1) 階梯止痛（從 NSAIDs 到強鴉片類）(2) **腹腔神經叢阻斷術**（CT 或 EUS 引導）效果好、可大幅減少嗎啡用量 (3) 放療止痛 (4) 安寧團隊介入。**不要忍痛**——疼痛控制好對生活品質很大幫助。

### Q: 病人 CA19-9 數值很高，是不是病情變壞？

A: 不一定。CA19-9 解讀注意：(1) **黃疸時假性升高**——減黃後再驗才準 (2) **Lewis 抗原陰性者**（5-10% 人口）即使有癌也不分泌 CA19-9 (3) **動態變化比絕對值重要**——治療中急速上升警示。需配合影像看才完整。

### Q: 胰臟癌病人問「我吃飯後一直拉肚子有油花」

A: **胰臟外分泌不足**——胰酶分泌不夠造成脂肪消化困難（脂肪痢）。處理：(1) **補充胰酶**（如 Creon），餐前 / 餐中服用 (2) 飲食調整：少油、避免過量脂肪 (3) 補充脂溶性維生素（A、D、E、K）。改善後體重可回升。

### Q: 病人問「手術後變糖尿病了，可以治嗎？」

A: Whipple 或胰臟全切後**胰島功能受損**——部分病人會新發糖尿病或原本糖尿病惡化。處理：(1) 監測血糖、HbA1c (2) 多需胰島素治療（口服藥效果有限）(3) 飲食、運動配合 (4) 內分泌科介入。這是手術後遺症、不是治療失敗。
