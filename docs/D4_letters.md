### 失聯掛號信

**用途：** 電話、簡訊、緊急聯絡人均無法聯繫，依院內規定寄送掛號信

```
受文者：○○先生/小姐 敬啟

您好：

本院癌症中心個案管理師於 ○年○月○日 至 ○年○月○日 期間，
嘗試以電話（○○○-○○○○○）聯繫您共 3 次，均未能聯絡到您。

您是本中心的關懷個案，我們關心您目前的健康狀況與治療情形。
如您方便，懇請撥打以下電話與我們聯繫：

癌症中心個管師：○○○
電話：○○○-○○○○○
服務時間：週一至週五 上午 8:00 – 下午 5:00

若您已另就醫或有其他考量，敬請告知，我們將依您的意願調整關懷方式。

敬祝 康復平安

彰濱秀傳紀念醫院 癌症中心個管師
○○○ 敬上
○○○○ 年 ○ 月 ○ 日
```

### 轉介通知（通用格式）

**用途：** 轉介心理師 / 社工師 / 營養師 / 安寧團隊時，隨轉介單附上的說明

```
受文者：○○○ ○師 您好

轉介個案摘要：

姓名：○○○（病歷號：○○○○○）
癌別/分期：（填寫）
主治醫師：（填寫）
目前治療狀況：（簡述目前接受的治療）

轉介原因：
  □ PG-SGA ≥ 4（轉介營養師）
  □ Distress Thermometer ≥ 4（轉介心理師）
  □ 經濟困難 / 社會支持不足（轉介社工師）
  □ 病情進展，考慮安寧銜接（轉介安寧團隊）
  □ 其他：（說明）

個案目前主要困難：
  （簡述病人/家屬反映的主要問題）

聯絡方式：個管師 ○○○，分機 ○○○

感謝您的協助，如需進一步資訊請與我聯繫。

個管師 ○○○
○○○○ 年 ○ 月 ○ 日
```

### 回診提醒（簡訊格式）

**用途：** 回診前 2–3 天發送簡訊提醒

日期：○月○日（○）
