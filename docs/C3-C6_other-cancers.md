## C3. 大腸直腸癌照護指引

> 📌 本文件為大腸直腸癌專科訓練教材。內容架構與 C1 肺癌、C2 乳癌一致。
> ⚠️ **準則引用**：本檔治療策略依據 NCCN Colon Cancer 2025 v1、台灣大腸直腸外科醫學會建議。**待醫師團隊確認版本**。

---

### 一、疾病概述

#### 流行病學

- 台灣癌症發生率：**第 2 位**（依國健署 108 年資料，標準化發生率 42.9/10 萬人）
- 台灣癌症死亡率：**第 3 位**
- 平均診斷年齡：**65–70 歲**
- 性別差異：男性略高於女性（直腸癌男性比女性多約 30%）
- 本院每年新案數：約（待補充）例

#### 危險因子

| 危險因子 | 說明 |
|---------|------|
| **年齡** | ≥ 50 歲為高風險族群 |
| **大腸瘜肉** | 腺瘤性息肉是癌前病變 |
| **家族史** | 一等親有大腸癌史風險增加 |
| **遺傳症候群** | FAP（家族性腺瘤瘜肉症）、Lynch 症候群 |
| **發炎性腸病** | 潰瘍性結腸炎、克隆氏症長期發炎 |
| **生活習慣** | 紅肉/加工肉品、低纖維、肥胖、缺乏運動、抽菸、飲酒 |

#### 常見症狀

| 部位 | 常見症狀 |
|------|---------|
| **遠端結腸 / 直腸**（約 60%）| 直腸出血、排便習慣改變、新發生便秘或腹瀉、大便變細、腹痛 |
| **近端右側結腸**（約 40%）| 慢性貧血、疲倦、腹部腫塊、體重減輕 |
| **直腸**（特殊症狀）| 排便急迫感、裡急後重感、排便帶血或黏液、排便不完全感 |
| **晚期/轉移** | 黃疸（肝轉移）、腹脹（腹水）、咳嗽肋膜積液（肺轉移）|

> ⚠️ 早期通常無症狀，**糞便潛血篩檢（FIT）**是早期診斷關鍵。

---

### 二、診斷與分期

#### 診斷流程

```
糞便潛血陽性 / 症狀懷疑
        ↓
    大腸鏡檢查 + 切片
        ↓
    病理確診
        ↓
    分期檢查（CT 胸/腹/骨盆、MRI 直腸癌）
        ↓
    基因檢測（MSI/MMR、RAS、BRAF）
        ↓
    MDT 討論治療計畫
```

#### 必要檢查項目

| 檢查 | 目的 | 備註 |
|------|------|------|
| **大腸鏡 + 切片** | 確診、評估腫瘤位置 | 必要 |
| **CT 胸腹骨盆** | 分期、評估遠端轉移 | 必要 |
| **直腸 MRI** | 直腸癌局部分期（T 期、N 期、CRM）| 直腸癌必要 |
| **CEA** | 腫瘤指數，治療反應與復發追蹤 | 必要 |
| **MSI/MMR 檢測** | 評估是否適合免疫治療、判斷 Lynch 症候群 | 全部需做 |
| **RAS / BRAF 基因** | 決定標靶藥物選擇（轉移性必做）| 轉移性必做 |

#### TNM 分期（AJCC 第 8 版）

| 期別 | 大致說明 | 5 年存活率（參考）|
|------|---------|----------------|
| **0 期**（Tis）| 原位癌 | > 95% |
| **I 期** | T1-2, N0 | ~90% |
| **II 期** | T3-4, N0 | ~70-85% |
| **III 期** | 任何 T, N1-2 | ~50-70% |
| **IV 期** | M1（遠端轉移）| ~15% |

---

### 三、治療指引

> ⚠️ **依據準則**：NCCN Colon/Rectal Cancer 2025、台灣大腸直腸外科醫學會建議。**待醫師團隊確認版本與本院實作差異**。

#### 早期（I-II 期）

- **手術切除**為主（部分結腸切除 + 局部淋巴結切除，N ≥ 12 顆）
- **II 期高風險者**（T4、淋巴結取樣 < 12、低分化、淋巴/血管/神經侵犯、術前阻塞或穿孔）→ 考慮輔助化療（FOLFOX 或 CapeOx）

#### III 期（淋巴結轉移、無遠端轉移）

- **手術 + 術後輔助化療**（FOLFOX 或 CapeOx，6 個月）
- 直腸癌 III 期：**術前 CCRT 或全程新輔助療法（TNT）**+ 手術 + 術後輔助化療

#### IV 期（轉移性 mCRC）

| 治療線別 | 治療策略 |
|---------|---------|
| **第一線** | 化療 ± 標靶（依 RAS/BRAF/MSI 分流）|
| **RAS wild-type** | 加 EGFR 抑制劑（Cetuximab / Panitumumab）|
| **RAS 突變** | 加 VEGF 抑制劑（Bevacizumab）|
| **BRAF V600E 突變** | Encorafenib + Cetuximab |
| **MSI-H / dMMR** | **免疫治療**（Pembrolizumab，健保 2025/6 起給付第一線）|
| **可切除肝/肺轉移** | 手術切除 + 圍術期化療（部分病人可治癒）|

#### 直腸癌特殊：術前放化療

- **T3-4 或 N+** 病人需術前 CCRT 或 TNT（**全程新輔助療法**）
- 目的：縮小腫瘤、降低局部復發率、提高保肛機會

### 四、本癌別特殊注意事項

| 重點 | 說明 |
|------|------|
| **基因檢測** | MSI/MMR（全部）、RAS（轉移必做）、BRAF（轉移必做）|
| **造口照護** | 直腸癌部分需永久或暫時性造口（迴腸造口/結腸造口），需造口護理師協助 |
| **CEA 追蹤** | 治療前基準值、術後每 3-6 個月、急遽上升需查復發 |
| **直腸癌術前放化療** | T3/T4 或 N+ 必做，5-FU 或 Capecitabine 為基底 |
| **遺傳諮詢** | < 50 歲確診、家族史強烈、Lynch 症候群可能 → 轉介遺傳諮詢 |

### 六、個管照護重點

#### 收案時

- 完成 CEA 基準值記錄
- 確認基因檢測（MSI/MMR）已開單
- 評估營養狀態（PG-SGA）—— 大腸癌常見體重減輕
- 評估 ECOG / Karnofsky—— 影響治療強度決定
- 心理困擾溫度計（DT）—— 確診衝擊大、需介入

#### 治療期間

- **手術期**：傷口照護、造口照護衛教（如適用）、飲食重新適應
- **化療期**：FOLFOX 注意周邊神經病變（Oxaliplatin）；Capecitabine 注意手足症候群
- **CCRT 期**（直腸癌）：注意腹瀉、皮膚反應、頻尿
- **轉移性個案**：副作用管理 + 心理支持 + 安寧轉介時機評估

#### 追蹤期

- 術後第 1-2 年：每 3 個月 CEA + 每 6 個月 CT
- 第 3-5 年：每 6 個月 CEA + 每年 CT
- 大腸鏡：術後 1 年、3 年、5 年（依風險調整）

### 七、常見問題與處理

#### Q: 病人問「我為什麼要做造口？是不是一輩子？」

A: 直腸癌靠近肛門時可能需要造口。**永久性造口**（如低位直腸癌切除後）約佔少數；**暫時性造口**多為保護吻合處、術後 3-6 個月可關回。實際情況請主治醫師說明。

#### Q: 病人問「我做完手術，為什麼還要化療？」

A: 病理顯示淋巴結有轉移（III 期）或 II 期高風險，**輔助化療可降低 30-40% 復發率**。這是「保險」，不是治療失敗。

#### Q: 病人是 RAS 突變/BRAF 突變，治療選擇有差嗎？

A: 是的。**RAS 突變**不能用 EGFR 抑制劑（Cetuximab/Panitumumab），改用 VEGF 抑制劑（Bevacizumab）；**BRAF V600E 突變**有專屬組合療法（Encorafenib + Cetuximab）。基因型決定藥物選擇。

#### Q: 病人問「我是 MSI-H，聽說有免疫治療？」

A: 是的。**MSI-H / dMMR 的轉移性大腸癌可用 Pembrolizumab**（健保 2025/6 起第一線給付）。MSI-H 約佔轉移性大腸癌 5-7%，反應率比一般化療高。

## C4. 肝癌照護指引

> 📌 本文件為肝癌專科訓練教材。
> ⚠️ **準則引用**：BCLC 2022 update、台灣肝臟醫學會 HCC 治療準則、台灣肝癌醫學會。**待醫師團隊確認版本**。

---

### 一、疾病概述

#### 流行病學

- 台灣癌症死亡率：長年位居前 2 位
- 主要病因：**B 型肝炎（約 70%）**、C 型肝炎（約 20%）、酒精性肝病、脂肪肝
- 平均診斷年齡：60–70 歲
- 男女比：約 3:1（男性高）
- 本院每年新案數：約（待補充）例

#### 危險因子

| 危險因子 | 說明 |
|---------|------|
| **B / C 型肝炎** | 最主要——每位肝癌病人都要查 HBV / HCV 狀態 |
| **肝硬化** | 任何原因（病毒、酒精、NAFLD）導致的肝硬化 |
| **黃麴毒素** | 發霉花生、玉米 |
| **酒精性肝病** | 長期飲酒 |
| **脂肪肝（NAFLD/NASH）** | 近年比例上升 |
| **家族史** | 一等親肝癌史 |

#### 常見症狀

- **早期常無症狀**（這是肝癌可怕之處——靠定期篩檢才能早期發現）
- 右上腹脹痛
- 食慾不振、體重減輕、疲倦
- 黃疸、腹水（晚期 / 肝功能差時）
- 食道靜脈瘤出血（門脈高壓）

> ⚠️ 高風險族群（B/C 肝、肝硬化）應每 3-6 個月做 **AFP + 腹部超音波**篩檢。

---

### 二、診斷與分期

#### 診斷流程

```
高風險族群追蹤（AFP + 超音波）/ 症狀懷疑
        ↓
    影像檢查（動態 CT / MRI 含肝臟標靶顯影劑）
        ↓
    符合典型影像特徵 → 直接確診（不一定需切片）
    不典型 → 切片確認
        ↓
    肝功能評估（Child-Pugh、ALBI）
        ↓
    BCLC 分期 + ECOG 評估
        ↓
    MDT 討論治療計畫
```

> 💡 肝癌特殊：**典型動態影像特徵**（動脈期顯影、靜脈期/延遲期 wash-out）可不切片直接診斷，避免切片風險。

#### 必要檢查項目

| 檢查 | 目的 | 備註 |
|------|------|------|
| **動態 CT 或 MRI** | 確診、分期 | MRI 含肝臟標靶顯影劑敏感度更高 |
| **AFP** | 腫瘤指數、追蹤反應 | 約 60-70% 病人 AFP 上升 |
| **HBV / HCV** | 病因評估、抗病毒治療指徵 | 必查 |
| **肝功能（ALT/AST/Bilirubin/Albumin/INR/PT）** | Child-Pugh / ALBI 分級 | 影響治療選擇 |
| **食道胃鏡** | 評估靜脈瘤、門脈高壓 | 肝硬化必要 |

#### BCLC 分期（巴塞隆納臨床肝癌分期）

> 肝癌**主要使用 BCLC 分期**而非 TNM——因為 BCLC 同時考慮腫瘤、肝功能、體能狀態，更能反映治療決策。

| BCLC 期 | 條件 | 治療策略 |
|---------|------|---------|
| **0 期 極早期** | 單一 ≤ 2 cm + Child-Pugh A + ECOG 0 | 切除 / RFA |
| **A 期 早期** | 單一或 ≤ 3 顆且每顆 ≤ 3 cm + Child-Pugh A-B + ECOG 0 | 切除 / RFA / 肝臟移植 |
| **B 期 中期** | 多發性腫瘤（不符 Milan 準則）+ Child-Pugh A-B + ECOG 0 | TACE（肝動脈化學栓塞）|
| **C 期 晚期** | 血管侵犯 / 肝外轉移 + Child-Pugh A-B + ECOG 1-2 | 全身性治療（標靶 / 免疫）|
| **D 期 終末期** | Child-Pugh C 或 ECOG 3-4 | 支持性療法（best supportive care）|

#### Child-Pugh 分級

詳見 [D1 評估量表彙整 → Child-Pugh Score](D1_assessment-tools.md)

#### Milan 準則（肝臟移植條件）

- 單一腫瘤 ≤ 5 cm，**或**
- ≤ 3 顆腫瘤且每顆 ≤ 3 cm
- **無**大血管侵犯
- **無**肝外轉移

---

### 三、治療指引

> ⚠️ **依據準則**：BCLC 2022 update、台灣肝臟醫學會 HCC 治療準則。**待醫師團隊確認版本與本院實作差異**。

#### 0 / A 期（極早期 / 早期）

| 治療 | 適用情境 |
|------|---------|
| **手術切除** | 單一腫瘤、肝功能好（Child-Pugh A）、無門脈高壓 |
| **射頻消融（RFA）** | 腫瘤 ≤ 3 cm、不適合手術或位置適合 |
| **肝臟移植** | 符合 Milan 準則、合併嚴重肝硬化首選 |

#### B 期（中期）

- **TACE 經肝動脈化學栓塞**：阻斷腫瘤血流、注入化療藥
- 治療次數依反應而定，每 2-3 個月評估
- 可重複進行；無效時升級到 C 期治療

#### C 期（晚期 / 全身性治療）

| 治療線別 | 藥物 |
|---------|------|
| **第一線（首選）** | **Atezolizumab + Bevacizumab**（免疫 + 標靶，IMbrave150 試驗）|
| **第一線替代** | Lenvatinib（口服標靶）、Sorafenib（口服標靶）|
| **第二線** | Regorafenib、Cabozantinib、Ramucirumab |
| **特殊**（食道靜脈瘤等出血風險）| 不適合 Bevacizumab → 改 Lenvatinib |

#### D 期（終末期）

- 安寧緩和療護
- 症狀控制：腹水、肝性腦病、疼痛
- [E 進階：安寧轉介](E_advanced.md)

---

### 四、本癌別特殊注意事項

| 重點 | 說明 |
|------|------|
| **肝功能評估** | Child-Pugh **必做**，影響每個治療決策 |
| **病毒性肝炎** | B/C 肝必查 + 抗病毒治療（即使治療肝癌中也要持續抗病毒）|
| **AFP 追蹤** | 治療前基準、治療中每 1-3 個月、急遽上升警示 |
| **多種治療選項** | 手術、RFA、TACE、標靶、免疫——個管師要熟悉每種 |
| **戒酒衛教** | 有飲酒習慣者必衛教、轉介戒酒資源 |
| **食道靜脈瘤監測** | 肝硬化者治療前胃鏡，避免出血意外 |

### 五、品質指標

| 指標 | 目標 |
|------|------|
| AFP 檢測率 | ≥ 95% |
| 肝功能評估率 | ≥ 95% |
| B/C 肝病毒檢測率 | ≥ 95% |
| MDT 討論率 | ≥ 70% |
| 抗病毒治療率（B/C 肝+） | ≥ 80% |
| 治療後影像追蹤率 | ≥ 85% |
| 戒酒衛教率 | ≥ 90% |

### 六、個管照護重點

#### 收案時

- 確認 AFP、HBV/HCV、肝功能基準值
- Child-Pugh 評估
- 評估有無食道靜脈瘤（胃鏡報告）
- 飲酒史、戒酒衛教時機
- 心理困擾溫度計（DT）

#### 治療期間

- **手術期**：傷口照護、肝功能監測（術後 2-4 週可能短暫惡化）
- **RFA / TACE**：發燒（栓塞後症候群）、腹痛、肝功能短期波動屬正常，需區分嚴重併發症
- **標靶治療期**（Lenvatinib / Sorafenib）：高血壓、手足症候群、腹瀉、疲倦
- **免疫 + 標靶（Atezo + Bev）**：irAE 警訊（喘、皮疹、腹瀉）+ Bev 出血風險

#### 追蹤期

- 治療後每 3 個月：AFP + 動態 CT/MRI
- 第 2 年起：每 6 個月
- B/C 肝抗病毒治療終身追蹤

---

### 七、常見問題與處理

#### Q: 病人問「為什麼我不能開刀？」

A: 肝癌能否手術不只看腫瘤——還要看：(1) 肝功能（Child-Pugh）能不能承受切除 (2) 殘餘肝體積夠不夠 (3) 有沒有門脈高壓 (4) 多發性或血管侵犯。許多病人是因肝硬化嚴重而不能開刀，不是腫瘤太大。

#### Q: 病人是 B 肝帶原，治療肝癌中要不要繼續吃抗病毒藥？

A: **絕對要**。化療、TACE、免疫治療都可能讓 B 肝病毒復發、引發猛爆性肝炎。抗病毒治療要持續、不能中斷。

#### Q: 病人問「TACE 做完發燒、肚子痛，是不是失敗了？」

A: 這是**栓塞後症候群**（post-embolization syndrome）——發燒、腹痛、噁心、ALT/AST 短期上升，多數 1 週內緩解，是正常反應、不是治療失敗。但若持續惡化要回診。

#### Q: 病人問「我可以喝酒嗎？」

A: **不可以**。任何階段（治療前、治療中、追蹤期）都應戒酒，避免肝功能進一步惡化。

#### Q: 病人問「我可以做肝臟移植嗎？」

A: 看是否符合**米蘭準則**（單一 ≤ 5 cm 或 ≤ 3 顆且每顆 ≤ 3 cm，無血管侵犯，無肝外轉移）。台灣以活體肝移植為主，等候時間相對短，但需有合適捐贈者。

## C5. 頭頸癌照護指引

> 📌 本文件為頭頸癌專科訓練教材，含口腔癌、口咽癌、下咽癌、喉癌、鼻咽癌等。
> ⚠️ **準則引用**：NCCN Head and Neck Cancers 2025、台灣頭頸部腫瘤醫學會。**待醫師團隊確認版本**。

---

### 一、疾病概述

#### 流行病學

- 台灣男性癌症發生率：**第 4 位**（口腔癌為主）
- 25-44 歲男性癌症發生率：**第 1-2 位**（口腔癌）
- 男女比：男性遠高於女性（與檳榔/菸/酒高度相關）
- 主要部位：口腔（最多）、口咽、下咽、喉、鼻咽

#### 危險因子

| 部位 | 主要危險因子 |
|------|------------|
| **口腔癌 / 口咽癌 / 下咽癌 / 喉癌** | 檳榔、抽菸、飲酒（**檳菸酒三聯**最高風險）|
| **鼻咽癌** | EBV 病毒感染、高鹽醃漬食品、家族史、廣東/福建族群 |
| **HPV 相關口咽癌** | HPV 16 型感染（年輕、非吸菸者比例上升）|

#### 常見症狀

| 部位 | 症狀 |
|------|------|
| **口腔** | 潰瘍 > 2 週不癒、白斑/紅斑、咀嚼疼痛、牙齒鬆動、頸部腫塊 |
| **鼻咽** | 鼻塞、流鼻血、聽力下降（單側中耳積液）、頸部腫塊（最常見的就診原因）|
| **下咽 / 喉** | 聲音沙啞 > 2 週、吞嚥困難、異物感、咳血 |

> ⚠️ **頸部不明腫塊 + 上呼吸消化道症狀**永遠要懷疑頭頸癌。

---

### 二、診斷與分期

#### 診斷流程

```
症狀懷疑 / 健檢發現
        ↓
    耳鼻喉科內視鏡 + 切片
        ↓
    病理確診（多為鱗狀細胞癌）
        ↓
    分期檢查（CT/MRI、PET-CT）
    + 鼻咽癌：EBV DNA 定量
        ↓
    HPV 檢測（口咽癌）
        ↓
    多學科 MDT（含牙科、營養師、語言治療師）
```

#### 必要檢查項目

| 檢查 | 目的 | 備註 |
|------|------|------|
| **內視鏡 + 切片** | 確診 | 必要 |
| **頸部 CT / MRI** | T、N 分期 | 必要 |
| **PET-CT** | 評估遠端轉移、第二原發癌 | III-IV 期建議 |
| **鼻咽癌 EBV DNA** | 確診、追蹤、預後因子 | 鼻咽癌必查 |
| **HPV / p16 檢測** | 口咽癌預後分組 | 口咽癌必查 |
| **牙科評估** | 放療前牙齒處理（避免放射性骨壞死）| 放療前必做 |
| **吞嚥/語言評估** | 治療前基線、治療中追蹤 | 必要 |
| **營養評估（PG-SGA）** | 治療前 50% 已有營養不良 | 必要 |

#### TNM 分期（AJCC 第 8 版）

> 頭頸癌各部位有不同的 TNM 規則，以下為通則。**口咽癌依 HPV/p16 狀態分為兩套分期系統**（HPV+ 預後較好）。

| 期別 | 大致說明 | 5 年存活率 |
|------|---------|----------|
| **I 期** | T1, N0 | 80–90% |
| **II 期** | T2, N0 | 60–80% |
| **III 期** | T3 或 N1 | ~50% |
| **IV 期** | T4 / N2-3 / M1 | ~30% |

### 三、治療指引

> ⚠️ **依據準則**：NCCN Head and Neck Cancers 2025、台灣頭頸部腫瘤醫學會。**待醫師團隊確認版本與本院實作差異**。

#### 早期（I-II 期）

- **單一治療**：手術 **或** 放療
- 部位選擇：口腔癌偏手術、喉癌可選放療保留發聲

#### 局部晚期（III-IVA-B 期）

| 部位 | 治療策略 |
|------|---------|
| **口腔癌** | 手術 + 術後放療（高風險再加化療 → 同步放化療 CCRT）|
| **口咽 / 下咽 / 喉癌** | **CCRT（同步放化療）** 為標準（器官保留療法） |
| **鼻咽癌** | **CCRT 為主治療**（鼻咽癌不適合手術，對放療反應好）|
| **N+ 病人** | 加頸部淋巴結廓清術或頸部放療 |

#### 復發 / 轉移性

- 第一線：**Pembrolizumab**（PD-L1+，免疫治療）± 化療
- 第二線：Cetuximab + 化療、其他免疫
- 鼻咽癌特殊：對化療反應較好（PF: Cisplatin + 5-FU）

#### 鼻咽癌特殊提醒

- **不適合手術**（位置鄰近顱底、血管），主治療為**放療**
- 早期單獨放療、中晚期 CCRT
- 治療中 EBV DNA 定量追蹤

### 四、本癌別特殊注意事項

| 重點 | 說明 |
|------|------|
| **放療前牙科評估** | 放射線可能導致放射性骨壞死，治療前要拔不良牙齒、處理蛀牙 |
| **吞嚥功能** | 治療前評估、治療中監測、治療後語言治療師復健 |
| **說話功能** | 喉癌切除後可能用人工發聲、食道發聲 |
| **外觀改變** | 手術切除大範圍時影響顏面，需心理支持 + 整形重建 |
| **戒菸戒檳** | 治療開始前、治療中、治療後**都要持續衛教** |
| **營養支持** | 治療中體重會下降 5-10%，常需鼻胃管 / 胃造口 |
| **口腔照護** | 放療中口腔黏膜炎、口乾——專屬口腔護理衛教 |
| **頸部放療皮膚照護** | 放療區皮膚會變色、脫皮——保持乾燥、避免日曬與摩擦 |

### 六、個管照護重點

#### 收案時

- 完成 PG-SGA、體重基準
- 牙科評估狀態（放療前重要）
- 戒菸戒檳衛教（持續、反覆）
- 心理支持——頭頸癌外觀+功能影響大、心理衝擊強
- 評估社會支持（家屬照顧能力）

#### 治療期間

- **手術期**：傷口照護、氣管造口照護、語言治療師早期介入
- **CCRT 期**：
  - 口腔黏膜炎管理（最常見、最痛苦）
  - 體重下降監測（每週量、降 10% 警示）
  - 鼻胃管 / 胃造口時機評估
  - 放療皮膚照護
  - Cisplatin 注意腎毒性、聽力
- **放療結束後**：口乾、味覺改變、頸部僵硬可能持續半年至一年

#### 追蹤期

- 第 1 年：每 1-3 個月
- 第 2 年：每 2-4 個月
- 第 3-5 年：每 4-6 個月
- 5 年後：每年
- 鼻咽癌追蹤 EBV DNA、影像

### 七、常見問題與處理

#### Q: 病人說「我以後不能說話了嗎？」

A: 看部位與治療方式。**喉癌全切除**會失去自然發聲，但有電子喉、食道發聲、人工發聲鈕等替代方案。**保留性治療**（CCRT）多數病人能保留聲音。語言治療師會在治療前後介入訓練。

#### Q: 病人說「我已經戒檳榔/菸/酒了，為什麼還會復發風險高？」

A: 已戒是好事，**繼續戒**才能降低**第二原發癌**風險（口腔癌病人有較高機率得到食道癌、肺癌）。完全戒除後 5-10 年風險才接近一般族群。

#### Q: 病人放療中嘴巴痛到吃不下，怎麼辦？

A: **口腔黏膜炎**是 CCRT 最常見的副作用。處理：(1) 鹽水/小蘇打水漱口（不用酒精性漱口水）(2) 局部止痛凝膠 (3) 軟質、室溫食物 (4) 嚴重時改鼻胃管 / 胃造口。**不要中斷治療**——中斷會降低治癒率。

#### Q: 病人問「為什麼我要先看牙科？」

A: 放療範圍含口腔時，**放射線會影響牙齒血流、唾液腺**，未來拔牙容易引起放射性骨壞死。治療前先處理蛀牙、拔不良牙齒，避免治療後悔不及。

#### Q: 鼻咽癌病人問「為什麼別人都開刀我不用？」

A: 鼻咽位置鄰近顱底、大血管、神經，**手術完全切除幾乎不可能**。但鼻咽癌對放射線反應非常好，所以**放療為主治療**，治癒率不亞於手術。

## C6. 攝護腺癌照護指引

> 📌 本文件為攝護腺癌專科訓練教材。
> ⚠️ **準則引用**：NCCN Prostate Cancer 2025、台灣泌尿科醫學會。**待醫師團隊確認版本**。

---

### 一、疾病概述

#### 流行病學

- 台灣男性癌症發生率：**第 5 位**（依國健署資料）
- 男性癌症死亡率：**第 7 位**
- 平均診斷年齡：**70 歲以上**為主
- 約 75% 為早期/中期，25% 為晚期
- 本院每年新案數：約（待補充）例

#### 危險因子

| 危險因子 | 說明 |
|---------|------|
| **年齡** | ≥ 50 歲風險顯著上升、≥ 65 歲為主要族群 |
| **家族史** | 一等親有攝護腺癌（尤其 < 65 歲）|
| **種族** | 非裔最高、亞裔較低（但台灣近年上升）|
| **基因** | BRCA1/2 突變（與遺傳性乳癌/卵巢癌相關）|
| **生活習慣** | 高脂飲食、肥胖（爭議中）|

#### 常見症狀

- **早期常無症狀**（靠 PSA 篩檢或健檢發現）
- 排尿症狀：頻尿、夜尿、尿急、解尿不順、尿流變細、餘尿感
- 血尿、血精
- **進階期**：骨痛（最常見的轉移部位是骨）、貧血、體重減輕

> ⚠️ 攝護腺癌排尿症狀跟**良性攝護腺肥大（BPH）很像**，需鑑別診斷。

---

### 二、診斷與分期

#### 診斷流程

```
PSA 異常 / 肛門指診（DRE）異常 / 症狀
        ↓
    多參數攝護腺 MRI（mpMRI）
        ↓
    經直腸超音波導引切片（TRUS biopsy）/ MRI 融合切片
        ↓
    病理確診 + Gleason 分數
        ↓
    分期檢查（影像、骨掃描）
        ↓
    風險分層（NCCN 五分類）
        ↓
    MDT / 醫病共享決策
```

#### 必要檢查項目

| 檢查 | 目的 | 備註 |
|------|------|------|
| **PSA** | 篩檢、追蹤、復發指標 | **基本指標** |
| **肛門指診（DRE）** | 觸診評估 | 必要 |
| **多參數攝護腺 MRI** | 切片前定位、分期 | 中高風險建議 |
| **經直腸超音波導引切片** | 病理確診 | 必要（≥ 12 點切片）|
| **Gleason 分數** | 病理惡性度（2-10 分）| 必要 |
| **骨掃描** | 評估骨轉移 | 中高風險建議 |
| **CT / PET-PSMA** | 評估其他轉移 | 高風險或復發 |

#### TNM 分期（AJCC 第 8 版）+ 風險分層

**Gleason 分數**：腫瘤兩個最大區域分級（1-5）相加（2-10 分）。

| Gleason | ISUP Grade | 意義 |
|---------|------------|------|
| ≤ 6 | 1 | 低度 |
| 3+4=7 | 2 | 中度（較好）|
| 4+3=7 | 3 | 中度（較差）|
| 8 | 4 | 高度 |
| 9-10 | 5 | 高度 |

**NCCN 風險分層**（局限性攝護腺癌，影響治療選擇）：

| 風險群 | 條件 | 治療傾向 |
|--------|------|---------|
| **極低風險** | T1c + Gleason ≤ 6 + PSA < 10 + 切片陽性 < 3 點 | **積極監控**首選 |
| **低風險** | T1-2a + Gleason ≤ 6 + PSA < 10 | 積極監控 / 根除性治療 |
| **中度風險** | T2b-c 或 Gleason 7 或 PSA 10-20 | 根除性治療（手術 / 放療 ± ADT 短期）|
| **高風險** | T3a 或 Gleason 8-10 或 PSA > 20 | 根除性治療 + ADT 2-3 年 |
| **極高風險** | T3b-4 或 N1 | 多模式治療（放療 + 長期 ADT）|

---

### 三、治療指引

> ⚠️ **依據準則**：NCCN Prostate Cancer 2025、中山醫大附設醫院 2017 版。**待醫師團隊確認版本與本院實作差異**。

#### 局限性攝護腺癌（依風險分層）

**極低 / 低風險**

- **積極監控（Active Surveillance）**：定期 PSA + DRE + 重複切片，視變化轉積極治療
- 適用：年紀大、預期餘命 < 10 年、不希望治療副作用
- 根除性治療（手術 / 放療）也可選

**中度風險**

- **根除性攝護腺切除**（含骨盆腔淋巴結廓清）
- **或放射線治療**（外放射或 + 組織插種近接治療）
- ± 4-6 個月短期 ADT

**高 / 極高風險**

- **放療 + 2-3 年長期 ADT**（首選）
- 或根除性手術（選擇性病人）
- 高風險特別注意：可能需要術後放療 / ADT

#### 轉移性攝護腺癌

| 治療線別 | 藥物 |
|---------|------|
| **去勢敏感性 mHSPC** | ADT + (Docetaxel 或 Abiraterone 或 Enzalutamide / Apalutamide) |
| **去勢抗性 mCRPC（一線）** | Abiraterone、Enzalutamide、Docetaxel |
| **mCRPC（二線後）** | Cabazitaxel、Radium-223（骨轉移）、Olaparib（BRCA+）、Lutetium-177-PSMA |

#### ADT（Androgen Deprivation Therapy 雄性素去除療法）

- 機制：阻斷睪固酮對癌細胞的刺激
- 形式：
  - 注射 LHRH agonist / antagonist（每 1/3/6 個月一次）
  - 雙側睪丸切除（永久性，少用）
- 副作用：熱潮紅、性慾減退、骨質疏鬆、代謝症候群、心血管風險

### 六、個管照護重點

#### 收案時

- 完成 PSA 基準、Gleason 分數記錄
- 風險分層確認
- 性功能、排尿功能基線評估
- 預期餘命 / 共病評估（CCI）—— 影響治療選擇
- 心理困擾溫度計

#### 治療期間

- **積極監控期**：定期追蹤完成度、心理焦慮支持
- **手術期**：尿失禁照護（凱格爾運動）、勃起功能恢復
- **放療期**：頻尿、解尿疼痛、放射性直腸炎、皮膚反應
- **ADT 治療期**：
  - 熱潮紅管理
  - 骨質保護（鈣 + 維生素 D + 規律運動 + 必要時雙磷酸鹽）
  - 心血管風險監測（血壓、血糖、血脂）
  - 性功能/性生活衛教
  - 情緒（憂鬱發生率較高）

#### 追蹤期

- 治療後第 1-2 年：每 3 個月 PSA
- 第 3-5 年：每 6 個月
- 5 年後：每年
- PSA 上升 → 影像評估復發位置

### 七、常見問題與處理

#### Q: 病人問「我可以選不治療嗎？」

A: 低風險、預期餘命短的病人，**積極監控**確實是合理選項——很多攝護腺癌進展緩慢，治療副作用反而比疾病本身嚴重。但「不治療」不是放任不管，而是**密切監控**（PSA、定期切片）、必要時轉積極治療。

#### Q: 病人問「我以後還能有性生活嗎？」

A: 看治療方式：(1) **積極監控**：完全不影響 (2) **保留神經手術**：約 50-70% 能恢復 (3) **放療**：多數初期不影響、治療後 1-2 年漸退 (4) **ADT**：性慾顯著下降、勃起困難。**個管師要主動衛教**——病人很多不敢問。

#### Q: 病人問「打了荷爾蒙針，為什麼覺得熱？」

A: **熱潮紅**是 ADT 最常見副作用——睪固酮下降影響體溫調節。處理：(1) 保持涼爽環境 (2) 避免咖啡、酒精、辛辣 (3) 嚴重時藥物治療（venlafaxine、gabapentin）。

#### Q: 病人問「PSA 比之前高了，是不是又長了？」

A: 不一定。PSA 受良性肥大、感染、運動、射精影響。**治療後 PSA 上升的判讀**：
- 手術後：PSA 應 < 0.2，連續 2 次 ≥ 0.2 = 生化復發
- 放療後：PSA 最低值 + 2.0 = 生化復發（Phoenix 標準）
- ADT 後：PSA 連續 3 次上升或 ≥ 2.0 = 去勢抗性

需醫師評估是否做影像檢查。

#### Q: 高齡病人問「我都 80 歲了還要治療嗎？」

A: 看 (1) Gleason 分數（高度才需積極治療）(2) 預期餘命 (3) 共病情況 (4) 病人意願。**多數低-中度風險的高齡病人，積極監控是合理選擇**。重點是**不要漏掉高度惡性者**（Gleason 8-10）。
