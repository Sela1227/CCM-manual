## 1. 系統概述

### 什麼是 HIS？

HIS（Hospital Information System）是醫院的核心資訊系統。本院的 HIS 整合了多個子系統，用同一個帳號登入就可以切換使用。

### 子系統總覽

| 子系統 | 功能說明 | 使用頻率 |
|--------|---------|---------|
| 門診系統 | 查門診病歷、看診紀錄 | ⭐⭐⭐ 每天 |
| 住院系統 | 查住院病歷、住院狀況 | ⭐⭐⭐ 每天 |
| 會診系統 | 開立會診、查詢會診結果 | ⭐⭐ 常用 |
| 檢查系統 | 排檢查、查檢查排程 | ⭐⭐ 常用 |
| PACS 系統 | 看影像（CT、MRI、X光等） | ⭐⭐ 常用 |
| 檢驗系統 | 查抽血報告、腫瘤指數 | ⭐⭐⭐ 每天 |
| 癌症總匯系統 | 查癌症病人整合資訊 | ⭐⭐⭐ 每天 |
| 化療藥物系統 | 查化療處方、療程 | ⭐⭐ 常用 |
| 放射線治療系統 | 查放療排程、進度 | ⭐ 需要時 |
| 期別系統 | 查詢/登錄癌症分期 | ⭐⭐ 收案時 |
| 癌計書系統 | 癌症治療計畫書 | ⭐⭐ 常用 |
| 多專科會議系統 | MDT 會議排程、紀錄 | ⭐ 每月 |

## 9. 癌症總匯系統

### 用途

這是個管師最常用的系統，整合了癌症病人的所有重要資訊。

### 可以看到的資訊

- 病人基本資料
- 癌症診斷
- 分期資訊
- 治療紀錄（手術、化療、放療等）
- 追蹤狀況
- 個管紀錄

### 進入方式

（待補充）

### 主要功能

**1. 個案查詢**

輸入病歷號，可以看到這個病人所有癌症相關的資訊。

**2. 我的個案清單**

可以看到你負責的所有個案，支援篩選：
- 依癌別
- 依狀態（治療中、追蹤中、結案）

- 依下次追蹤日期

**3. 追蹤提醒**

系統會提醒：
- 該追蹤的個案

- 追蹤日期和方式
- 病人狀況
- 處理事項

## 15. 常見問題排解

### Q: 系統登不進去？

A: 
1. 確認帳號密碼是否正確
2. 確認 Caps Lock 沒有開啟
3. 嘗試重新開機
4. 還是不行 → 聯絡資訊室

### Q: 找不到病人資料？

A:
1. 確認病歷號是否正確
2. 用身分證字號查（注意英文要大寫）
3. 確認病人是否在本院有就醫紀錄

### Q: 報告還沒出來？

A:
- 不同檢查出報告時間不同（參考各系統說明）
- 可以先跟病人說大概時間
- 急件可以聯絡相關單位

### Q: 系統很慢或當掉？

A:
1. 等待一下再試
2. 關掉不需要的視窗
3. 重新登入
4. 還是不行 → 聯絡資訊室

### Q: 操作上有問題不會用？

A:
1. 先問學姊
2. 查這份手冊
3. 還是不會 → 問資訊室
