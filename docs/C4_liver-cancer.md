## 一、疾病概述

### 流行病學

- 台灣癌症死亡率：長年位居前 2 位
- 主要病因：**B 型肝炎（約 70%）**、C 型肝炎（約 20%）、酒精性肝病、脂肪肝
- 平均診斷年齡：60–70 歲
- 男女比：約 3:1（男性高）
- 本院每年新案數：約（待補充）例

### 危險因子

| 危險因子 | 說明 |
|---------|------|
| **B / C 型肝炎** | 最主要——每位肝癌病人都要查 HBV / HCV 狀態 |
| **肝硬化** | 任何原因（病毒、酒精、NAFLD）導致的肝硬化 |
| **黃麴毒素** | 發霉花生、玉米 |
| **酒精性肝病** | 長期飲酒 |
| **脂肪肝（NAFLD/NASH）** | 近年比例上升 |
| **家族史** | 一等親肝癌史 |

### 常見症狀

- **早期常無症狀**（這是肝癌可怕之處——靠定期篩檢才能早期發現）
- 右上腹脹痛
- 食慾不振、體重減輕、疲倦
- 黃疸、腹水（晚期 / 肝功能差時）
- 食道靜脈瘤出血（門脈高壓）

> ⚠️ 高風險族群（B/C 肝、肝硬化）應每 3-6 個月做 **AFP + 腹部超音波**篩檢。

## 二、診斷與分期

### 診斷流程

```
高風險族群追蹤（AFP + 超音波）/ 症狀懷疑
        ↓
    影像檢查（動態 CT / MRI 含肝臟標靶顯影劑）
        ↓
    符合典型影像特徵 → 直接確診（不一定需切片）
    不典型 → 切片確認
        ↓
    肝功能評估（Child-Pugh、ALBI）
        ↓
    BCLC 分期 + ECOG 評估
        ↓
    MDT 討論治療計畫
```

> 💡 肝癌特殊：**典型動態影像特徵**（動脈期顯影、靜脈期/延遲期 wash-out）可不切片直接診斷，避免切片風險。

### 必要檢查項目

| 檢查 | 目的 | 備註 |
|------|------|------|
| **動態 CT 或 MRI** | 確診、分期 | MRI 含肝臟標靶顯影劑敏感度更高 |
| **AFP** | 腫瘤指數、追蹤反應 | 約 60-70% 病人 AFP 上升 |
| **HBV / HCV** | 病因評估、抗病毒治療指徵 | 必查 |
| **肝功能（ALT/AST/Bilirubin/Albumin/INR/PT）** | Child-Pugh / ALBI 分級 | 影響治療選擇 |
| **食道胃鏡** | 評估靜脈瘤、門脈高壓 | 肝硬化必要 |

### BCLC 分期（巴塞隆納臨床肝癌分期）

> 肝癌**主要使用 BCLC 分期**而非 TNM——因為 BCLC 同時考慮腫瘤、肝功能、體能狀態，更能反映治療決策。

| BCLC 期 | 條件 | 治療策略 |
|---------|------|---------|
| **0 期 極早期** | 單一 ≤ 2 cm + Child-Pugh A + ECOG 0 | 切除 / RFA |
| **A 期 早期** | 單一或 ≤ 3 顆且每顆 ≤ 3 cm + Child-Pugh A-B + ECOG 0 | 切除 / RFA / 肝臟移植 |
| **B 期 中期** | 多發性腫瘤（不符 Milan 準則）+ Child-Pugh A-B + ECOG 0 | TACE（肝動脈化學栓塞）|
| **C 期 晚期** | 血管侵犯 / 肝外轉移 + Child-Pugh A-B + ECOG 1-2 | 全身性治療（標靶 / 免疫）|
| **D 期 終末期** | Child-Pugh C 或 ECOG 3-4 | 支持性療法（best supportive care）|

### Child-Pugh 分級

詳見 [D1 評估量表彙整 → Child-Pugh Score](D1_assessment-tools.md)

### Milan 準則（肝臟移植條件）

- 單一腫瘤 ≤ 5 cm，**或**
- ≤ 3 顆腫瘤且每顆 ≤ 3 cm
- **無**大血管侵犯
- **無**肝外轉移

## 三、治療指引

> ⚠️ **依據準則**：BCLC 2022 update、台灣肝臟醫學會 HCC 治療準則。**待醫師團隊確認版本與本院實作差異**。

### 0 / A 期（極早期 / 早期）

| 治療 | 適用情境 |
|------|---------|
| **手術切除** | 單一腫瘤、肝功能好（Child-Pugh A）、無門脈高壓 |
| **射頻消融（RFA）** | 腫瘤 ≤ 3 cm、不適合手術或位置適合 |
| **肝臟移植** | 符合 Milan 準則、合併嚴重肝硬化首選 |

### B 期（中期）

- **TACE 經肝動脈化學栓塞**：阻斷腫瘤血流、注入化療藥
- 治療次數依反應而定，每 2-3 個月評估
- 可重複進行；無效時升級到 C 期治療

### C 期（晚期 / 全身性治療）

| 治療線別 | 藥物 |
|---------|------|
| **第一線（首選）** | **Atezolizumab + Bevacizumab**（免疫 + 標靶，IMbrave150 試驗）|
| **第一線替代** | Lenvatinib（口服標靶）、Sorafenib（口服標靶）|
| **第二線** | Regorafenib、Cabozantinib、Ramucirumab |
| **特殊**（食道靜脈瘤等出血風險）| 不適合 Bevacizumab → 改 Lenvatinib |

### D 期（終末期）

- 安寧緩和療護
- 症狀控制：腹水、肝性腦病、疼痛
- [E 進階：安寧轉介](E_advanced.md)

## 四、本癌別特殊注意事項

| 重點 | 說明 |
|------|------|
| **肝功能評估** | Child-Pugh **必做**，影響每個治療決策 |
| **病毒性肝炎** | B/C 肝必查 + 抗病毒治療（即使治療肝癌中也要持續抗病毒）|
| **AFP 追蹤** | 治療前基準、治療中每 1-3 個月、急遽上升警示 |
| **多種治療選項** | 手術、RFA、TACE、標靶、免疫——個管師要熟悉每種 |
| **戒酒衛教** | 有飲酒習慣者必衛教、轉介戒酒資源 |
| **食道靜脈瘤監測** | 肝硬化者治療前胃鏡，避免出血意外 |

## 六、個管照護重點

### 收案時

- 確認 AFP、HBV/HCV、肝功能基準值
- Child-Pugh 評估
- 評估有無食道靜脈瘤（胃鏡報告）
- 飲酒史、戒酒衛教時機
- 心理困擾溫度計（DT）

### 治療期間

- **手術期**：傷口照護、肝功能監測（術後 2-4 週可能短暫惡化）
- **RFA / TACE**：發燒（栓塞後症候群）、腹痛、肝功能短期波動屬正常，需區分嚴重併發症
- **標靶治療期**（Lenvatinib / Sorafenib）：高血壓、手足症候群、腹瀉、疲倦
- **免疫 + 標靶（Atezo + Bev）**：irAE 警訊（喘、皮疹、腹瀉）+ Bev 出血風險

### 追蹤期

- 治療後每 3 個月：AFP + 動態 CT/MRI
- 第 2 年起：每 6 個月
- B/C 肝抗病毒治療終身追蹤

## 七、常見問題與處理

### Q: 病人問「為什麼我不能開刀？」

A: 肝癌能否手術不只看腫瘤——還要看：(1) 肝功能（Child-Pugh）能不能承受切除 (2) 殘餘肝體積夠不夠 (3) 有沒有門脈高壓 (4) 多發性或血管侵犯。許多病人是因肝硬化嚴重而不能開刀，不是腫瘤太大。

### Q: 病人是 B 肝帶原，治療肝癌中要不要繼續吃抗病毒藥？

A: **絕對要**。化療、TACE、免疫治療都可能讓 B 肝病毒復發、引發猛爆性肝炎。抗病毒治療要持續、不能中斷。

### Q: 病人問「TACE 做完發燒、肚子痛，是不是失敗了？」

A: 這是**栓塞後症候群**（post-embolization syndrome）——發燒、腹痛、噁心、ALT/AST 短期上升，多數 1 週內緩解，是正常反應、不是治療失敗。但若持續惡化要回診。

### Q: 病人問「我可以喝酒嗎？」

A: **不可以**。任何階段（治療前、治療中、追蹤期）都應戒酒，避免肝功能進一步惡化。

### Q: 病人問「我可以做肝臟移植嗎？」

A: 看是否符合**米蘭準則**（單一 ≤ 5 cm 或 ≤ 3 顆且每顆 ≤ 3 cm，無血管侵犯，無肝外轉移）。台灣以活體肝移植為主，等候時間相對短，但需有合適捐贈者。
