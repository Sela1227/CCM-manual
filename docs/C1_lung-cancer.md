### 流行病學

- 台灣癌症發生率排名：（待補充）
- 本院每年新案數：約（待補充）例
- 好發年齡：（待補充）

### 分類

| 類型 | 英文 | 比例 | 特性 |
|------|------|------|------|
| 非小細胞肺癌 | NSCLC | ~85% | 又分腺癌、鱗狀細胞癌、大細胞癌 |
| 小細胞肺癌 | SCLC | ~15% | 惡性度高、易轉移 |

### 非小細胞肺癌亞型

| 亞型 | 比例 | 特性 |
|------|------|------|
| 腺癌 | ~60% | 最常見，常見於非吸菸者，EGFR 突變較多 |
| 鱗狀細胞癌 | ~25% | 與吸菸高度相關 |
| 大細胞癌 | ~10% | 較少見 |

### 危險因子

- 吸菸（最主要）
- 二手菸暴露
- 空氣污染
- 職業暴露（石棉、氡氣等）
- 家族史
- 肺部疾病史

### 常見症狀

- 早期：常無症狀
- 咳嗽（持續或改變）
- 咳血
- 呼吸困難
- 胸痛
- 體重減輕
- 聲音沙啞

## 二、診斷與分期

### 診斷流程

```
懷疑肺癌（症狀/影像異常）
        ↓
    影像檢查（CT）
        ↓
    組織診斷（切片）
        ↓
    分期檢查
        ↓
    基因檢測（NSCLC）
        ↓
    MDT 討論治療計畫
```

### 必要檢查項目

| 檢查 | 目的 | 備註 |
|------|------|------|
| 胸部 CT | 評估腫瘤大小、位置 | 含顯影劑 |
| 病理切片 | 確定診斷和分型 | 必要 |
| PET-CT 或 骨掃描 | 評估遠端轉移 | 分期用 |
| 腦部 MRI 或 CT | 評估腦轉移 | 尤其晚期 |
| 肺功能檢查 | 評估手術耐受度 | 考慮手術時 |

### 基因檢測

> ⚠️ 非小細胞肺癌務必做基因檢測，會影響治療選擇！

| 基因 | 檢測率目標 | 陽性率 | 對應藥物 |
|------|-----------|-------|---------|
| EGFR | ≥ 90% | ~55%（台灣） | 標靶藥物（TKI） |
| ALK | ≥ 80% | ~5% | 標靶藥物 |
| ROS1 | - | ~1-2% | 標靶藥物 |
| PD-L1 | ≥ 70% | 不一定 | 免疫治療參考 |
| BRAF | - | ~2% | 標靶藥物 |

### TNM 分期（第 8 版）

**T 分期（腫瘤）**

| 分期 | 定義 |
|------|------|
| T1 | ≤ 3 cm |
| T2 | > 3-5 cm 或侵犯主支氣管/臟層肋膜 |
| T3 | > 5-7 cm 或侵犯胸壁/心包膜等 |
| T4 | > 7 cm 或侵犯縱膈/心臟/大血管等 |

**N 分期（淋巴結）**

| 分期 | 定義 |
|------|------|
| N0 | 無淋巴結轉移 |
| N1 | 同側支氣管周圍或肺門淋巴結 |
| N2 | 同側縱膈或隆突下淋巴結 |
| N3 | 對側縱膈/對側肺門/同側或對側斜角肌或鎖骨上淋巴結 |

**M 分期（轉移）**

| 分期 | 定義 |
|------|------|
| M0 | 無遠端轉移 |
| M1a | 對側肺轉移/惡性肋膜積液 |
| M1b | 單一器官單一轉移 |
| M1c | 單一或多器官多處轉移 |

**綜合分期**

| 期別 | TNM | 大致說明 |
|------|-----|---------|
| I 期 | T1-2, N0, M0 | 早期，可手術 |
| II 期 | T1-2, N1 或 T3, N0 | 局部侵犯，可考慮手術 |
| III 期 | T3-4, N1-2 或任何 T, N2-3 | 局部晚期 |
| IV 期 | 任何 T, 任何 N, M1 | 晚期，有遠端轉移 |

## 三、治療指引

### 治療原則

| 期別 | 主要治療 | 說明 |
|------|---------|------|
| I-II 期 | 手術 ± 輔助治療 | 早期以手術為主 |
| III 期 | 多模式治療 | 依情況組合手術、化療、放療 |
| IV 期 | 全身性治療 | 標靶、免疫、化療 |

### 早期肺癌（I-II 期）

**首選：手術切除**

- 肺葉切除 + 淋巴結廓清
- 術後依病理結果考慮輔助治療

**術後輔助治療**

| 情況 | 建議 |
|------|------|
| II 期以上 | 考慮輔助化療 |
| EGFR 突變 | 考慮輔助標靶 |
| 高風險 IB 期 | 個案討論 |

### 局部晚期（III 期）

- 可切除：術前或術後化療 + 放療
- 不可切除：同步化放療 → 免疫治療維持

### 晚期肺癌（IV 期）

> 💡 治療選擇高度依賴基因檢測結果！

**有驅動基因突變（EGFR/ALK/ROS1 等）**

→ 標靶治療優先

| 基因 | 第一線藥物（舉例） |
|------|------------------|
| EGFR | Osimertinib, Afatinib, Gefitinib |
| ALK | Alectinib, Crizotinib |
| ROS1 | Crizotinib |

**無驅動基因突變**

→ 依 PD-L1 表現決定

| PD-L1 | 建議治療 |
|-------|---------|
| ≥ 50% | 免疫治療單藥或併用化療 |
| 1-49% | 免疫治療 + 化療 |
| < 1% | 化療 ± 免疫治療 |

### 小細胞肺癌

| 期別 | 治療 |
|------|------|
| 侷限期 | 同步化放療 |
| 擴散期 | 化療 + 免疫治療 |

## 四、品質指標

### 診斷相關指標

| 指標 | 定義 | 目標 | 你可以做的事 |
|------|------|------|-------------|
| 組織診斷率 | 有病理或細胞學診斷的比例 | ≥ 90% | 確認病人有做切片 |
| 分期完整率 | 有完整 TNM 分期的比例 | ≥ 95% | 確認分期檢查完成 |
| EGFR 檢測率 | NSCLC 做 EGFR 檢測的比例 | ≥ 90% | 提醒醫師開檢測、追蹤報告 |
| ALK 檢測率 | NSCLC 做 ALK 檢測的比例 | ≥ 80% | 同上 |
| PD-L1 檢測率 | 晚期 NSCLC 做 PD-L1 的比例 | ≥ 70% | 同上 |

### 治療相關指標

| 指標 | 定義 | 目標 | 你可以做的事 |
|------|------|------|-------------|
| MDT 討論率 | 經多專科討論的比例 | ≥ 70% | 確認個案有排入 MDT |
| 治療等待時間 | 診斷到開始治療的天數 | ≤ 30 天 | 協調加速流程 |
| 早期手術率 | I-II 期接受手術的比例 | ≥ 80% | 追蹤治療決定 |
| 術後輔助化療率 | II-IIIA 期術後化療的比例 | ≥ 60% | 追蹤術後治療 |
| 標靶治療率 | EGFR+ 晚期使用標靶的比例 | ≥ 85% | 確認基因結果對應治療 |

### 照護相關指標

| 指標 | 定義 | 目標 | 你可以做的事 |
|------|------|------|-------------|
| 個管收案率 | 新診斷個案收案比例 | ≥ 90% | 主動收案、不漏接 |
| 營養篩檢率 | 完成營養篩檢的比例 | ≥ 90% | 確實做評估 |
| 心理篩檢率 | 完成 DT 篩檢的比例 | ≥ 85% | 確實做評估 |
| 追蹤完成率 | 持續追蹤的比例 | ≥ 80% | 定期追蹤、減少失聯 |

## 五、個管照護重點

### 收案時

**評估重點：**
- 吸菸史（現在/過去/從未）
- 呼吸功能狀況
- 體重變化
- 心理調適（確診衝擊）

**確認事項：**
- [ ] 診斷和分期是否確定
- [ ] 基因檢測是否送出
- [ ] 是否已排 MDT 討論
- [ ] 治療計畫書是否完成

### 治療期間

**手術病人：**
- 術前：確認檢查完成、手術同意書
- 術後：傷口照護、呼吸復健、追蹤回診

**化療病人：**
- 注意副作用：噁心、疲倦、血球下降
- 衛教：發燒要立即就醫

**標靶治療病人：**
- EGFR TKI：注意皮疹、腹瀉、甲溝炎
- ALK 抑制劑：注意視力問題、水腫

**放療病人：**
- 注意：皮膚反應、食道炎、疲倦
- 可能影響吞嚥，注意營養

**免疫治療病人：**
- 注意免疫相關副作用（皮膚、腸道、肝臟、肺臟、內分泌）
- 衛教：有任何不適要回報

### 追蹤期

**追蹤頻率建議：**

| 時期 | 頻率 |
|------|------|
| 治療中 | 每次治療或每 2-4 週 |
| 治療結束後 1-2 年 | 每 3-6 個月 |
| 2 年以上 | 每 6-12 個月 |

**追蹤重點：**
- 症狀變化（咳嗽、呼吸、疼痛）
- 體重變化
- 影像追蹤結果
- 心理適應

## 六、常見問題與處理

### Q: 病人問「我還可以活多久？」

A: 這是很難回答的問題。你可以說：
- 「每個人的狀況不同，醫師會根據您的情況給您最適合的治療。」
- 「我們一起努力，有任何問題隨時可以問。」
- 如果病人很焦慮，考慮轉介心理師。

### Q: 家屬要求不要告訴病人病情

A: 
- 了解家屬的擔心
- 說明病人有知的權利
- 建議召開家庭會議，由醫師協助溝通
- 必要時社工師介入

### Q: 病人猶豫要不要做基因檢測（自費部分）

A:
- 解釋基因檢測的重要性（會影響藥物選擇）
- 說明有健保給付和自費的項目
- 若經濟有困難，轉介社工評估補助資源

### Q: 病人抱怨標靶藥物副作用很不舒服想停藥

A:
- 先了解是什麼副作用、多嚴重
- 說明副作用處理方式（很多可以改善）
- 強調不要自行停藥，要跟醫師討論
- 安排提早回診

### Q: 病人治療後影像顯示腫瘤變大

A:
- 這是醫師要判斷和說明的
- 你的角色是支持病人的情緒
- 協助安排後續檢查或治療調整
- 評估是否需要心理支持
