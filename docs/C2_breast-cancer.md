## 一、疾病概述

### 流行病學

- 台灣女性癌症發生率：第一位
- 本院每年新案數：約（待補充）例
- 好發年齡：45-69 歲

### 危險因子

- 年齡增加
- 家族史（尤其一等親）
- BRCA1/BRCA2 基因突變
- 初經早、停經晚
- 未生育或晚生育
- 荷爾蒙補充療法
- 肥胖
- 飲酒

### 分子分型

> 💡 乳癌的分子分型非常重要，會決定治療策略！

| 分型 | ER/PR | HER2 | Ki-67 | 特性 |
|------|-------|------|-------|------|
| Luminal A | + | - | 低 | 預後最好，荷爾蒙治療為主 |
| Luminal B | + | -/+ | 高 | 可能需加化療 |
| HER2+ | -/+ | + | - | 需標靶治療 |
| 三陰性 | - | - | - | 較難治療，以化療為主 |

## 二、診斷與分期

### 診斷流程

```
發現乳房異常（自我檢查/篩檢）
        ↓
    乳房攝影/超音波
        ↓
    粗針切片（Core biopsy）
        ↓
    病理報告 + 分子檢測（ER/PR/HER2/Ki-67）
        ↓
    分期檢查
        ↓
    MDT 討論治療計畫
```

### 必要檢查項目

| 檢查 | 目的 |
|------|------|
| 乳房攝影 | 偵測腫塊、鈣化 |
| 乳房超音波 | 評估腫塊性質 |
| 粗針切片 | 確定診斷 |
| 病理：ER/PR | 荷爾蒙受體狀態 |
| 病理：HER2 | HER2 表現狀態 |
| 病理：Ki-67 | 細胞增殖指數 |
| 胸部 X 光/CT | 分期檢查 |
| 肝臟超音波/CT | 分期檢查 |
| 骨掃描 | 骨轉移評估（視期別） |

### TNM 分期

**簡化說明：**

| 期別 | 大致情況 |
|------|---------|
| 0 期（原位癌） | 未侵犯，預後極佳 |
| I 期 | ≤ 2cm，無淋巴轉移 |
| II 期 | 2-5cm 或有腋下淋巴轉移 |
| III 期 | 局部晚期，較大或多處淋巴轉移 |
| IV 期 | 有遠端轉移 |

## 三、治療指引

### 治療原則

| 期別 | 主要治療 |
|------|---------|
| 早期（I-II） | 手術 → 輔助治療（依分子分型） |
| 局部晚期（III） | 術前治療 → 手術 → 輔助治療 |
| 轉移性（IV） | 全身性治療為主 |

### 手術方式

| 方式 | 適應症 |
|------|-------|
| 乳房保留手術 | 腫瘤較小、位置適合 |
| 全乳切除 | 腫瘤較大、多發、病人選擇 |
| 前哨淋巴結切片 | 臨床 N0 |
| 腋下淋巴結廓清 | 淋巴結轉移確認 |

### 輔助治療

| 分型 | 輔助治療建議 |
|------|-------------|
| Luminal A | 荷爾蒙治療 5-10 年 |
| Luminal B | 荷爾蒙治療 ± 化療 |
| HER2+ | 標靶（Trastuzumab）+ 化療 + 荷爾蒙（若 ER+） |
| 三陰性 | 化療（± 免疫治療） |

### 放射治療

- 乳房保留手術後：必須放療
- 全乳切除後：高風險者需放療

## 五、個管照護重點

### 收案時

**評估重點：**
- 確認分子分型（ER/PR/HER2/Ki-67）
- 了解病人對外觀改變的擔心
- 評估家庭支持系統

**特別注意：**
- 乳癌對女性身體意象影響大
- 初診斷時情緒支持很重要

### 治療期間

**手術病人：**
- 術前：解釋手術方式、可能的外觀改變
- 術後：傷口照護、引流管照護
- 復健：肩關節活動、預防淋巴水腫

**化療病人：**
- 掉髮：事先衛教、協助假髮資源
- 噁心、疲倦：衛教處理方式

**荷爾蒙治療病人：**
- 長期服藥（5-10 年）
- 注意副作用：熱潮紅、關節痛、子宮內膜問題
- 追蹤服藥遵從性

**標靶治療病人（HER2+）：**
- 注意心臟毒性
- 定期追蹤心臟功能

### 特殊議題

**淋巴水腫：**
- 高風險：腋下淋巴結廓清後
- 衛教預防方式
- 有症狀要轉介復健科

**生育議題：**
- 年輕病人可能關心
- 治療前可討論生育保存
- 轉介生殖醫學科

**乳房重建：**
- 有需求可轉介整形外科
- 說明時機和選項

## 六、常見問題與處理

### Q: 病人問「我可以保留乳房嗎？」

A: 這要由醫師評估。你可以說明影響因素包括腫瘤大小、位置、病人意願等。強調保留與否不影響存活率，重要的是選擇適合自己的方式。

### Q: 病人很擔心掉髮

A:
- 同理她的擔心（這很正常）
- 說明大約何時會掉、何時會長回來
- 介紹假髮資源（癌症資源中心可借用）
- 有些化療方案掉髮較少，可跟醫師討論

### Q: 病人荷爾蒙治療吃了很不舒服想停藥

A:
- 了解是什麼副作用
- 說明荷爾蒙治療的重要性（降低復發）
- 很多副作用可以處理或換藥
- 安排回診與醫師討論，不要自行停藥

### Q: 病人問可不可以懷孕

A:
- 這是複雜的問題，需要醫師評估
- 一般建議治療完成後一段時間再考慮
- 若有生育計畫，治療前要提出
- 可轉介生殖醫學科諮詢
