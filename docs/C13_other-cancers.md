# C13. 其他癌照護指引

**彰濱秀傳癌症中心｜個管師指導書系列**

> 📌 本文件為「**其他癌**」（非 1-9 號核心 MDT 團隊涵蓋癌別）的照護指引總覽，由個管師 C 負責。
> ⚠️ **本檔特殊性**：「其他癌」是異質性極高的類別——涵蓋甲狀腺癌、皮膚癌、軟組織惡性肉瘤、腦瘤、鼻咽癌以外的耳鼻喉癌、不明原發癌（CUP）、神經內分泌瘤等多種癌別。**本檔提供通用照護原則 + 各癌別概要**，深入內容仍待醫師團隊提供。

---

## 一、本檔涵蓋癌別範圍

「其他癌」是 10 號 MDT 團隊涵蓋的範疇，包含但不限於：

| 癌別 | 大致定位 |
|------|---------|
| **甲狀腺癌** | 預後通常良好（乳突癌、濾泡癌）；少數惡性高（未分化、髓質）|
| **皮膚癌** | 黑色素瘤、鱗狀細胞癌、基底細胞癌 |
| **軟組織惡性肉瘤** | 異質性高，依組織型分類 |
| **骨肉瘤** | 兒童青少年好發 |
| **腦瘤 / 中樞神經系統腫瘤** | 膠質母細胞瘤、星狀細胞瘤、腦膜瘤等 |
| **不明原發癌（CUP）** | 已轉移但找不到原發處 |
| **神經內分泌瘤（NET）** | 胰臟、腸道、肺等部位 |
| **生殖細胞瘤** | 睪丸癌等 |

---

## 二、通用診斷與照護原則

### 個管師接觸「其他癌」病人時的關鍵問題

1. **這個癌的治癒率如何？** 甲狀腺癌乳突型 > 95%；惡性黑色素瘤晚期 < 30%；膠質母細胞瘤 < 10%——**差異極大**
2. **主治科別是哪一科？** 甲狀腺癌通常一般外科 / 內分泌科；皮膚癌皮膚科；腦瘤神經外科；肉瘤骨科；NET 內分泌或腫瘤內科
3. **有沒有可參考的指引？** 多數可參考 NCCN 各癌別準則
4. **病人需要什麼支持？** 通用個管照護不變（評估、衛教、轉介、追蹤）

### 通用診斷流程

```
症狀 / 影像異常 / 健檢發現
        ↓
    依疑似癌別到對應科別
        ↓
    切片或手術確診
        ↓
    分期檢查
        ↓
    MDT（10 號團隊）討論
        ↓
    依癌別準則治療
```

## 三、各癌別簡要重點

### 甲狀腺癌

| 重點 | 說明 |
|------|------|
| **常見型別** | 乳突癌（80%）、濾泡癌、髓質癌、未分化癌 |
| **治療** | 手術切除 + **放射性碘（I-131）**治療 + 甲狀腺荷爾蒙抑制治療 |
| **追蹤關鍵** | 甲狀腺球蛋白（Tg）、Anti-Tg 抗體、頸部超音波 |
| **預後** | 乳突癌、濾泡癌極好（5 年 > 95%）；未分化癌差 |

### 皮膚癌

| 重點 | 說明 |
|------|------|
| **黑色素瘤（最惡性）** | 警示 ABCDE：不對稱 / 邊緣不規則 / 顏色多變 / 直徑 > 6 mm / 變化中 |
| **基底細胞癌** | 最常見、轉移罕見、手術切除為主 |
| **鱗狀細胞癌** | 局部侵犯、可能轉移 |
| **治療** | 手術切除 ± 標靶（BRAF/MEK 抑制劑，黑色素瘤）/ 免疫治療 |
| **預防** | 防曬、避免人工日光浴 |

### 軟組織惡性肉瘤

| 重點 | 說明 |
|------|------|
| **異質性高** | 50+ 亞型，依組織學決定治療 |
| **常見部位** | 四肢、後腹膜、軀幹 |
| **治療** | 手術 + 術後放療（保肢治療）± 化療 |
| **重要指標** | 手術邊緣（margin）—— R0 切除是治癒關鍵 |

### 腦瘤 / 中樞神經系統腫瘤

| 重點 | 說明 |
|------|------|
| **膠質母細胞瘤（GBM）** | 預後最差（中位存活 12-18 個月）|
| **治療** | 手術 + CCRT（Temozolomide + 放療）+ 維持化療（**Stupp 方案**）|
| **腦水腫** | 類固醇（Dexamethasone）控制 |
| **癲癇** | 預防性抗癲癇藥（部分病人）|

### 不明原發癌（CUP）

| 重點 | 說明 |
|------|------|
| **特性** | 已遠端轉移、找不到原發處 |
| **檢查** | 詳盡 IHC、PET-CT、必要時基因檢測 |
| **治療** | 依推測原發或廣譜化療、近年免疫治療有突破 |

### 神經內分泌瘤（NET）

| 重點 | 說明 |
|------|------|
| **常見部位** | 胰臟、小腸、肺 |
| **特殊** | 部分有荷爾蒙分泌症狀（類癌症候群、胰島素瘤等）|
| **治療** | 手術、體抑素類似物（Octreotide、Lanreotide）、PRRT 標靶放射治療、標靶治療 |

## 四、共通注意事項

| 重點 | 說明 |
|------|------|
| **預後差異大** | 個管師需先了解該癌別的預後，才能適當溝通 |
| **MDT 跨科** | 「其他癌」病人多需多科別會診（如腦瘤需神外+腫瘤+放腫） |
| **症狀管理** | 不同癌別的特殊症狀（甲狀腺荷爾蒙、皮膚癌局部、腦瘤神經學）|
| **罕見癌挑戰** | 少見癌別資源較少、可能需轉介至有經驗的醫學中心 |
| **病友支持** | 部分罕見癌有專屬病友會（如黑色素瘤、腦瘤、肉瘤）|

## 六、個管照護重點

### 收案時

- **首要任務**：確認癌別、分期、預後預期
- 主治科別與 MDT 討論狀態
- 評估標準量表（ECOG、PG-SGA、DT）
- 心理支持（罕見癌的孤立感）
- 預後溝通評估（部分癌別預後差、需早期討論）

### 治療期間

- 依各癌別治療方式照護（參考 [F4 癌症治療](F4_treatment.md) 通論）
- 副作用監測
- 跨科協調（罕見癌常需多科會診）

### 追蹤期

- 依各癌別準則決定追蹤頻率
- 注意該癌別特殊指標（甲狀腺 Tg、黑色素瘤皮膚科檢查、腦瘤 MRI 等）

## 七、常見問題與處理

### Q: 病人說「我的癌很罕見，網路上資料很少」

A: 罕見癌挑戰：(1) 治療指引可能不如常見癌完整 (2) 醫療資源集中在大醫學中心 (3) 病友會少。應對：(1) 引導至 NCCN 或對應學會（國際 + 國內）(2) 評估是否轉介經驗豐富的醫學中心 (3) 連結對應病友團體（黑色素瘤協會、腦瘤協會、肉瘤協會等）。

### Q: 甲狀腺癌病人問「我做完手術，為什麼還要喝放射性碘？」

A: **放射性碘（I-131）治療**目的：(1) **消除殘留甲狀腺組織**（甲狀腺全切後仍可能有殘餘）(2) **消除微小轉移**（提高長期存活）(3) **後續以 Tg 監測復發**（殘餘組織會分泌 Tg、影響追蹤）。治療前需低碘飲食 1-2 週、停甲狀腺荷爾蒙以提高 TSH。

### Q: 黑色素瘤病人問「BRAF 是什麼？我為什麼要檢測？」

A: **BRAF V600E** 是黑色素瘤約 50% 病人有的基因突變。**陽性者**可使用 **BRAF + MEK 抑制劑**（如 Dabrafenib + Trametinib）口服標靶——反應率高、副作用相對可控。**陰性者**通常用免疫治療（Pembrolizumab、Nivolumab + Ipilimumab）。檢測決定治療選擇。

### Q: 腦瘤病人問「我為什麼一直頭痛 / 噁心？」

A: 多為**腦水腫**（腫瘤周圍水腫壓迫腦組織）。處理：(1) **類固醇（Dexamethasone）**降腦水腫——多在數天內改善 (2) 嚴重時甘露醇靜脈注射 (3) 急性惡化需 CT 排除腦疝 (4) 預防性抗癲癇藥（部分病人）。**頭痛突然加劇 / 嘔吐 / 意識改變**要立即就醫。

### Q: 不明原發癌（CUP）病人問「為什麼找不到原發？是不是檢查沒做仔細？」

A: 即使做了完整 PET-CT、內視鏡、IHC、基因檢測，仍有約 3-5% 癌症找不到原發處——這是**疾病特性**，不是檢查不夠。應對：(1) 依轉移部位 + IHC 推測可能原發 (2) 廣譜化療或依推測原發治療 (3) 近年**腫瘤基因檢測**（NGS）+ **免疫治療**對部分 CUP 有突破。
