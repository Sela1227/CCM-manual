# C12. 血液及淋巴癌照護指引

**彰濱秀傳癌症中心｜個管師指導書系列**

> 📌 本文件為血液及淋巴癌專科訓練教材，供負責的個管師（A）參考。內容架構與 C1 肺癌、C2 乳癌一致。

> ⚠️ **準則引用**：NCCN Hematologic Cancers（B-cell / T-cell Lymphoma、AML、ALL、CML、MM）2025、台灣血液病學會、台灣骨髓移植醫學會建議。**待醫師團隊確認版本與本院實作差異**。

## 一、疾病概述

### 流行病學

**白血病（Leukemia）**
- 急性淋巴性白血病（ALL）：兒童最常見的癌
- 急性骨髓性白血病（AML）：成人為主、預後差
- 慢性淋巴性白血病（CLL）：老年人、惰性
- 慢性骨髓性白血病（CML）：費城染色體陽性、靶向治療顯著改善預後

**淋巴瘤**
- **何杰金氏淋巴瘤（HL）**：年輕人多、預後好（5 年存活 > 85%）
- **非何杰金氏淋巴瘤（NHL）**：老年居多、亞型多元
  - DLBCL（最常見、約 30%）
  - 濾泡性淋巴瘤、套細胞淋巴瘤、Burkitt 淋巴瘤等

**多發性骨髓瘤（MM）**
- 平均診斷年齡：60–70 歲
- 男性略高
- 5 年存活率：約 50%（新藥顯著改善）

### 危險因子

| 危險因子 | 說明 |
|---------|------|
| **化學物質暴露** | 苯、農藥（白血病風險）|
| **放射線暴露** | 醫療放射、原子彈倖存者 |
| **病毒感染** | EBV（伯基特淋巴瘤、何杰金氏）、HTLV-1（成人 T 細胞白血病）、HIV |
| **免疫低下** | 器官移植後、HIV、自體免疫疾病用免疫抑制劑 |
| **化療史** | 治療相關 AML、MDS |
| **遺傳症候群** | 唐氏症（ALL）、范可尼貧血等 |
| **年齡** | AML、MM、CLL 多老年；ALL、HL 年輕族群多 |

### 常見症狀

**白血病**
- **三低一多**：貧血、白血球低（感染）、血小板低（出血）、白血球異常增多
- 疲倦、發燒、夜間盜汗
- 容易瘀青、流鼻血、牙齦出血
- 骨頭疼痛、淋巴結腫大（部分）
- 中樞神經侵犯（ALL 較常見）

**淋巴瘤**
- **無痛性淋巴結腫大**（頸部、腋下、腹股溝）
- **B symptoms（B 症狀）**：
  - 不明原因發燒 > 38°C
  - 夜間盜汗
  - 6 個月內體重減 > 10%
- 肝脾腫大
- 局部壓迫症狀（縱膈、腹腔）

**多發性骨髓瘤**
- **CRAB 症狀**：
  - **C**alcium（高血鈣）
  - **R**enal failure（腎功能異常）
  - **A**nemia（貧血）
  - **B**one lesions（骨頭病變、壓迫性骨折）
- 反覆感染（免疫球蛋白異常）
- 神經學症狀（脊椎壓迫、漿細胞瘤）

## 二、診斷與分期

### 診斷流程

```
血液異常 / 淋巴結腫大 / 骨痛
        ↓
    完整血液檢查（CBC + 分類 + 生化 + 骨髓功能）
        ↓
    淋巴結 / 骨髓切片 / 抽吸
        ↓
    病理確診 + 流式細胞儀 + 免疫組織化學
        ↓
    細胞遺傳學 / FISH / 分子檢測
        ↓
    分期檢查（PET-CT、骨髓檢查）
        ↓
    MDT 討論治療策略
```

### 必要檢查項目

| 檢查 | 目的 | 適用 |
|------|------|------|
| **CBC + 分類** | 篩檢血球異常 | 全部 |
| **骨髓抽吸 + 切片** | 確診白血病、骨髓侵犯 | 白血病、MM 必做 |
| **淋巴結切片** | 淋巴瘤確診 | 淋巴瘤必做 |
| **流式細胞儀** | 細胞表面標記、亞型 | 全部血液癌 |
| **免疫組織化學（IHC）** | 病理分類 | 淋巴瘤必做 |
| **染色體分析 / FISH** | 預後分組、靶向治療 | 全部 |
| **PET-CT** | 淋巴瘤分期、追蹤 | 淋巴瘤必做 |
| **EBV / HIV 檢測** | 病因評估 | 部分 |
| **腰椎穿刺** | 中樞神經侵犯評估 | ALL、部分 NHL |
| **β2 微球蛋白、白蛋白** | MM 分期（ISS）| MM 必做 |
| **免疫球蛋白電泳、輕鏈** | MM 確診、追蹤 | MM 必做 |

### 分期

**白血病**：依細胞起源 + 急/慢性 + 細胞遺傳學分類，無傳統 TNM 分期。

**何杰金氏淋巴瘤（Ann Arbor / Lugano）**

| 期別 | 大致說明 |
|------|---------|
| I 期 | 單一淋巴結區或單一淋巴外器官 |
| II 期 | ≥ 2 個淋巴結區（同側橫膈）|
| III 期 | 淋巴結區跨越橫膈 |
| IV 期 | 廣泛侵犯淋巴外器官（肝、骨髓等）|

加註：A（無 B 症狀）/ B（有 B 症狀）

**多發性骨髓瘤（ISS / R-ISS）**

| 分期 | 條件 |
|------|------|
| I 期 | β2 微球蛋白 < 3.5 + 白蛋白 ≥ 3.5 |
| II 期 | 介於 I 與 III |
| III 期 | β2 微球蛋白 ≥ 5.5 |

加 LDH、染色體：R-ISS（修訂版）

## 三、治療指引

> ⚠️ **依據準則**：NCCN Hematologic Cancers（B-cell / T-cell Lymphoma、AML、ALL、CML、MM）2025、台灣血液病學會、台灣骨髓移植醫學會建議。**待醫師團隊確認版本與本院實作差異**。

### 急性白血病（AML / ALL）

| 階段 | 治療 |
|------|------|
| **誘導治療** | 化療（AML：7+3 ara-C + 蒽環類；ALL：多藥組合）|
| **鞏固治療** | 高劑量化療 / 造血幹細胞移植 |
| **維持治療** | ALL 需要 |
| **中樞神經預防** | ALL 必做（鞘內化療 / 顱內放療）|
| **靶向藥物** | AML：FLT3、IDH1/2 抑制劑；ALL：BCR-ABL+ 用 TKI |
| **CAR-T 細胞療法** | 復發/難治 ALL（健保給付）|

### 慢性白血病

**CML**
- **TKI（口服標靶）**為主治療：Imatinib、Dasatinib、Nilotinib
- **完治標準**：BCR-ABL IS < 1%（治療滿半年後）

**CLL**
- 觀察 + 等待（早期無症狀）
- 治療：BTK 抑制劑（Ibrutinib、Acalabrutinib）、Venetoclax + 抗 CD20、化免疫治療

### 淋巴瘤

**何杰金氏淋巴瘤**
- 早期：ABVD 化療 + 局部放療
- 進展期：ABVD × 6 cycles 或 BV+AVD（Brentuximab vedotin 加強）
- 復發/難治：BV、Pembrolizumab、Nivolumab、自體幹細胞移植

**非何杰金氏淋巴瘤**
- DLBCL：**R-CHOP**（Rituximab + 化療）為標準
- 濾泡性：R-CVP、R-CHOP、Bendamustine + R
- 套細胞：R-CHOP 加強、BTK 抑制劑、CAR-T
- 復發：CAR-T 細胞療法（健保部分給付）

### 多發性骨髓瘤

| 階段 | 治療 |
|------|------|
| **誘導治療** | **VRD**（Bortezomib + Lenalidomide + Dex）或 **VTD** |
| **自體幹細胞移植** | 適合年輕、體能好者 |
| **維持治療** | Lenalidomide |
| **復發** | 第二代蛋白酶體抑制劑、Daratumumab、Pomalidomide、CAR-T（部分）|
| **完治標準** | VTD 8 次；或 VTD 4 次 + 骨髓移植；或 VRD 6 個月 |

### 造血幹細胞移植

- **自體**：MM、淋巴瘤、部分白血病
- **異體**：高風險 AML、復發 ALL、CML 末期等
- 移植前清髓化療、移植後抗排斥藥（GVHD 處理）

## 四、本癌別特殊注意事項

| 重點 | 說明 |
|------|------|
| **發燒嗜中性球低下（FN）** | 化療後常見、**38°C + 嗜中性球 < 0.5** = 緊急住院、IV 抗生素 |
| **腫瘤溶解症候群（TLS）** | 大量腫瘤細胞溶解（高鉀、高磷、高尿酸、低鈣）——化療前預防（allopurinol、水化）|
| **B 症狀追蹤** | 淋巴瘤治療反應指標 |
| **造血幹細胞移植照護** | 移植後 100 天高度風險期 / 抗排斥藥終身（部分）|
| **GVHD（移植物抗宿主病）** | 急性 / 慢性、皮膚 / 腸 / 肝表現 |
| **EBV / B 肝再激活** | 免疫治療 / 化療中可能再活化 |
| **疫苗** | 移植後需重新接種、感染預防 |
| **生育保存** | 年輕病人治療前討論精子/卵子凍存 |
| **生命末期溝通** | 部分血液癌惡化快、需早期討論 |

## 六、個管照護重點

### 收案時

- 完成 CBC、骨髓報告、染色體 / FISH 確認
- B 症狀基準（淋巴瘤）
- 感染風險評估（嗜中性球）
- 中央靜脈導管（Port-A / PICC）置放規劃
- 生育意願評估（年輕病人）
- 心理困擾溫度計

### 治療期間

- **化療誘導期**：
  - **嗜中性球低下發燒（FN）警示**——38°C + ANC < 0.5 = 立即就醫
  - 腫瘤溶解症候群預防（allopurinol、水化）
  - 中央靜脈導管照護
  - 口腔黏膜炎、腹瀉
- **造血幹細胞移植期**：
  - 隔離防護
  - GVHD 監測（皮膚紅疹、腹瀉、肝功能）
  - 移植後 100 天密集追蹤
- **靶向治療期**：
  - TKI（CML）：每月 BCR-ABL 監測
  - BTK 抑制劑（CLL）：心房顫動、出血風險
  - 蛋白酶體抑制劑（MM）：周邊神經病變
- **CAR-T 治療期**：
  - 細胞激素釋放症候群（CRS）：發燒、低血壓
  - 神經毒性（ICANS）

### 追蹤期

- 急性白血病治療後密集追蹤
- 淋巴瘤：每 3-6 個月 PET-CT、CBC
- MM：每月 IgG / 輕鏈、骨頭追蹤
- CML：每 3 個月 BCR-ABL

## 七、常見問題與處理

### Q: 病人問「我為什麼會得血癌？是不是被別人傳染的？」

A: **血液癌不會傳染給別人**。多數原因不明、跟基因突變累積、化學物質暴露、輻射、部分病毒（EBV、HTLV-1）等有關。**家屬不需擔心被傳染**——可以正常接觸、共餐、擁抱。

### Q: 化療中發燒到 38°C，要不要去急診？

A: **要立刻去急診！** 化療後白血球低下時發燒（**FN：發燒嗜中性球低下**）是**致命急症**——可能 1-2 小時就敗血症。即使看起來只是小感冒、半夜也要去。**不要先吃退燒藥觀察**。

### Q: 病人問「我可以拒絕骨髓移植嗎？」

A: 移植不是必要的——多數淋巴瘤、CML 不需要移植。**急性白血病高風險者、淋巴瘤復發、MM 部分病人**移植效果好但有風險（GVHD、感染、死亡率）。是否做需衡量：(1) 病情風險 (2) 年齡與體能 (3) 是否有合適捐贈者 (4) 病人意願。可以拒絕、但要充分溝通替代方案。

### Q: 淋巴瘤病人問「我做完化療為什麼還要做 PET-CT？」

A: 看治療反應 + 是否復發。淋巴瘤的 **PET-CT 是評估腫瘤活性**的金標準，治療中（中期評估）+ 治療後（治療反應）+ 追蹤期都會做。**完全反應（PET 陰性）是治癒的關鍵指標**。

### Q: MM 病人問「我吃了藥還在痛、骨頭還是會斷嗎？」

A: MM 骨痛來自**骨溶解性病變**（osteolytic lesions），即使治療有效骨頭結構不會立即修復。處理：(1) **雙磷酸鹽**（Zometa、Aredia）每月一次 IV，預防骨折 (2) 適度止痛 (3) 嚴重骨折考慮**椎體成形術**（vertebroplasty）(4) 避免劇烈運動或負重。預期 6-12 個月後骨頭重塑改善。
